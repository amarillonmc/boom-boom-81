# 小兵的角色 - 库·丘林（FGO）

## 1F

- Author: 小兵
- Posted at: 十一月 19, 2025, 11:25 上午

**角色填写人：**小兵
(https://media.fgo.wiki/d/d5/%E6%84%9A%E4%BA%BA%E8%8A%82_%E5%8D%A1%E9%9D%A2_FFJ_017.png)

**基础信息：**

**角色名称：**库·丘林（FGO）

**角色背景：**
出自游戏《Fate/Grand Order》， 《Fate/stay night》
职阶Lancer。乳名是瑟坦特。库·丘林拥有「库林的猛犬」这层含义。这是因为瑟坦特年幼时，在错手杀死富商库林的看门狗后宣誓，「若这条狗有孩子，那自己将负责抚养它，将其培养成同样强大而忠诚的看门狗。在达成这个目标前，我将负责保护你」。

性格习惯
不喜欢多余的装饰，喜欢像野兽般战斗的枪兵。即使对方是好友，若是对方在敌阵的话就会一边笑叹运气不好一边拼个你死我活、就算对方是父母的仇敌，只要是友方就会在誓约的限度内守护到底，遵守着像这样的凯尔特的战士特有的想法生存着。在工作中时异常的冷酷，不讨厌从背后进行突然袭击，表现出只要是命令的话连普通人也会杀掉"封口"的无情。发现目击了战斗的卫宫士郎，就想要当场杀掉，展现出于战斗以外也不留祸根的果断的行动方式。
而另一方面，又有着爽快奔放的性格，是懂得照顾他人的大哥。即使分属敌我双方的立场，对中意的人也会积极地搭话，就算被无视也绝不气馁。
性格豪迈，曾言："哪怕是仇人，只要志趣相投就一起痛饮到天亮才是人之常情。"为人可靠。粗野粗蛮又和蔼可亲，本性正直而笃于忠义。

库·丘林成年后，加入了职责为保护阿尔斯特王的「赤枝骑士团」。为了与某位女性结合，作为条件，他决定前往被称为影之国的魔境。库·丘林克服了诸多难关，抵达了影之国。他得到了国主·魔女斯卡哈的喜爱，在她的身边刻苦锻炼，学会了各种魔术与体术。

相貌衣着
留着一头蓝色短发，几缕碎刘海随意的飘着，脑后梳着长长的马尾辫，留至腰间。双耳戴着一对青色的耳坠。赤色的双瞳，蓝色的眉毛。身穿一身蓝色的紧身衣，紧身衣延伸至手腕处形成碗甲，双肩披着银色肩甲，手上拿着赤色的刺穿死棘之枪。

日常生活
擅长钓鱼、潜水、登山。喜欢强气的女子跟乱来的约定，讨厌绕圈子的方案跟背叛。非常喜欢向女孩子搭讪但是到目前为止还没成功过。

**战斗能力：**

**核心能力·装备：**
四枝之浅滩 A
一个不多见的耐久技能。不易轻易战死，只要不是受到必死的致命伤，就能存活下去的能力。即使身负濒死重伤也能行动和战斗。
在传说里，濒死的库·丘林将自己的身躯绑在石柱上，然后才断气。这样的他，纵然是绝望的战斗也发挥出惊人的坚韧顽强。

重整旗鼓 C
脱离战斗的能力。
由于持有这两个技能，实际上相对于单挑，
库·丘林更适合在大型合战中发挥其真正价值。
然而由于本人极为喜好单对单的决斗，
结果几乎无人知晓这一事实。

卢恩 B
拥有北欧的魔术刻印·卢恩。
影之国的战士必须精通百般武艺，
而不能只会挥舞长枪——―
也许是因为师父斯卡哈有着上述教育理念，
总之库·丘林同样精通卢恩魔术。
虽然作为魔术师，他的实力也已接近一流，
然而本人却因嫌麻烦而几乎不使用魔术。

避矢之加护 B
对飞行道具的防御。库·丘林天生持有的能力。被视为只要用眼睛确认了攻击对象，无论怎样的远距离攻击都可以闪避。
另外，纵使处于无法眼观察对象的状况，也能应付大部分的飞行道具。

『穿刺死棘之枪』
等级：B → B+ 种类：对人宝具
攻击距离：2~4 最大捕捉：1人
把本为投掷使用的对军宝具Gáe Bolg，经库·丘林自我加工后作为对人来使用的宝具，是他的必杀技和王牌。
是将意味着事物顺序的"因果"逆转的招式，在放出此的瞬间，"对手的心脏被贯穿"这个"果"会先被造出。"由于心脏被贯穿了所以枪是命中了的"这个"因"则从后发生。若没有相当的幸运，可说是一招确定为"若放出即死"的招式。魔力消耗较低，按"几乎确实地破坏敌人心脏"这种强烈的效果说来，并不怎样的需要魔力，成为了非常好用的宝具。
因为放出去的时候已经有"贯穿对手的心脏"的结果，所以就算抢先一步杀死库·丘林也无法改变这个结果，因此无法取消这把枪的运行轨迹。

**被动特性·常驻效果：**
对魔力 C
只能无效化二节以下的咏唱所引起的魔术，在其以上的大魔术和仪礼咒法实在是无法防御。

神性 B
神灵适性的有无。等级越高代表越是物质性地混有神灵的血。
库·丘林拥有高神性，因为其父亲为具备了知识、招式与魔术等一切技能的光之神鲁格。半神半人的他，生来就发挥着优秀的才能。从年幼时起就以异于常人的怪力著称。

**关键弱点/明确限制：**
对抗『穿刺死棘之枪』的手段：不算是方法的方法，尽力远离或战术撤退，不让库·丘林有机会使用。

宝具不能连续发动。

本身库·丘林作为英灵在体力耗尽后会消散。（视为淘汰）

幸运E（娱乐，非战斗本专用）：库·丘林最差的属性，在娱乐，非战斗本里似乎扩大了导致很容易让他发生倒霉事。

**行为逻辑：**
**核心动机：**库·丘林，并不是一个单纯的鲁莽战士。

本身是追求最高的敏捷性和优秀的格斗战能力的英灵，喜欢配合强效宝具进行简单粗暴却行之有效的高速战斗。

来到这里如果遇到强者他一定会主动申请挑战的。
**角色定位：**正面战斗员　侦查/潜行者
**行动倾向：**
主动进攻型

库·丘林被淘汰时如果身边有队友则会：
队友："Lancer死了！"
其他人："简直不是人！"
