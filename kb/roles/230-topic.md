# 玄羽的角色 - 资本家

## 1F

- Author: 玄羽
- Posted at: 九月 13, 2025, 08:48 上午

**角色填写人：**玄羽

**角色名称：**资本家

**基本属性：**种族：人类 /"路灯挂件"（底层民众私下称呼）
*关联角色：有钱人（理想主义面向）/ 平行宇宙同位体
力量：扭曲的财富掌控力（钱即权力，钱即规则）
装备：
- 【黑心燕尾服（深渊强化版）】：原版燕尾服经"黑金科技"改造，表面优雅如初，内在已被改造成一套高度侵略性、控制型的纳米武装。
- 强化特性：
 - 可释放微纳米级控制粒子，短暂侵入他人神经系统，植入"服从暗示"；
 - 领结可发射高频声波，使人产生恐惧、焦虑等负面情绪；
 - 袖口电击器升级为"债务电击"，中击者会短暂感受到"债务缠身"的精神幻觉；
 - 光学迷彩可扭曲周围光线，制造"金钱幻觉"，让人看到无数钞票飞舞却抓不住。
规模：单人（但其财富与控制力覆盖多宇宙层级）

**超能力：**【财富深渊】
有钱人不再是"交易者"，而是"支配者"。他的财富已成为一种诅咒性的力量，能与诸天万界中最为黑暗、贪婪的存在进行交易。
①：花钱购买"命运契约"，强制目标为其服务，若拒绝则会被"财富反噬"——随机失去一件最重要的事物（爱情、记忆、天赋等）。
②：用钱收买对手，若对方不接受，则从其亲友、梦想、信仰等方面进行"精准打击"，逼其就范。
③：不再招募"帮手"，而是购买"替身傀儡"，这些傀儡拥有与原角色相同的能力，但完全服从有钱人，且死亡后不会回归，而是化为金钱消散。
④：租借不再自愿，而是"强制征用"。有钱人可暂时"借"走他人的能力、装备甚至情感，归还时却会附加"利息"——让对方失去更多。
⑤：求购变成"勒索"。他人若向有钱人求购，必须付出远高于价值的代价，且往往是被精心设计的"陷阱条款"。

【燕尾服（深渊强化版）】
※表面仍是科技装备，实则已融合了诸天黑市中的诅咒科技与契约魔法。
核心功能：支配与侵蚀
- 可分析目标的弱点（贪婪、恐惧、欲望），并制定"支配方案"；
- 能模拟出极致的优雅与亲和力，实则每一步都在计算如何最大化控制与剥削；
详细功能列表：
- 支配型体能：力量与速度的提升建立在"剥夺他人"之上，每使用一次，就有某个世界的一个普通人失去一天的能量；
- 武器系统：
 a.领结飞镖 → 现在附魔"债务诅咒"，中镖者会不断"欠债"，精神逐渐崩溃；
 b.袖口电击器 → 升级为"命运干涉器"，可短暂改写目标的命运线，使其"注定失败"；
 c.隐藏武器 → 包括"契约锁链"、"金钱陷阱"等；
- 环境适应与工具：
 a.吸附能力 → 可吸附在"社会阶层"之上，越是底层越易被其控制；
 b.潜水功能 → 可在"资本海洋"中潜行，无声无息吞噬小企业；
 c.钩爪与绳索 → 用于将他人拉入"债务深渊"；
- 伪装与通讯：
 a.光学迷彩 → 可伪装成"救世主"或"慈善家"，实则操纵舆论；
 b.内置通讯系统 → 可直接连接至诸天黑市，购买诅咒、契约、傀儡等服务；

**行为逻辑：**表面是斯文败类、笑里藏刀、优雅地做着最恶毒的事；
"你不是缺钱，你是缺一个像我这样的主人。"
有时，通过诸天黑市的监控，他会瞥见某个宇宙中那个"光明"版本的自己——那个穿着相似却本质不同的燕尾服，做着可笑"慈善"的家伙。他会嗤之以鼻，但目光却会停留得比看其他目标更久一些。那一刻，他眼中会闪过一丝极其复杂的情绪，混杂着不屑、怜悯，以及一丝几乎无法察觉的...怀念。
"我给你的不是钱，是命运。接受它，或者被它压垮。"
他平等地"剥削"所有人。
他尤其"享受"与那些坚信"公平交易"的理想主义者打交道。他会用最严苛的条款击碎他们的幻想，过程巾会感到一种扭曲的快意，仿佛在亲手扼杀自己早已抛弃的过去。
他提供"福报工作制"......
他暗中资助反抗军，但偶尔会发现一些反抗军的资金链异常干净和雄厚，仿佛有某个不愿露面的强大力量在同样支持他们。这让他产生一丝怀疑和好奇，但他从不深究，只当是多元宇宙的又一个巧合。
他相信人性本善，但更相信人性贪婪。他希望有人能看穿他的伪装，联合起来推翻他；
"我希望他们恨我，恨到团结起来。"
"如果那个'我'看到现在的我，一定会想方设法来'拯救'我吧......真是天真得令人作呕。"他偶尔会对着空无一人的办公室低声嘲讽，不知道在说给哪个时空的自己听。
"我不是想要成为皇帝，我是想成为那个不得不被推翻的暴君。"
"如果财富不能让人觉醒，那我就用贫穷去唤醒他们。"
他时常在无人的时候，看着旧日理想主义的照片自言自语：
"......看啊，另一个'我'正走在你们曾经承诺却从未抵达的路上。而我，选择了另一条路来证明你们的虚伪。"
他已不再相信理想，却仍相信人性最终的光明。他扮演极致的恶，部分是为了逼迫众生觉醒，部分或许也是为了向某个存在于想象中的、光明的自己证明：你看，这才是现实。

**内核揭示（仅对他自己可见）：**
他曾在早期相信"先富带动后富"，试图用财富建立乌托邦，却遭合作伙伴与竞争对手联手背叛，失去一切，甚至险些丧命。
黑化后，他决定用极端的方式"教育"世界：
- 如果善良无法唤醒众人，那就用极致的恶去逼他们团结；
- 如果财富不能共享，那就用垄断逼他们反抗；
- 他渴望被推翻，甚至暗中资助反抗军，只为有一天有人能终结他所扮演的这个"黑心资本家"。

>>> 他是最好的演员，也是最坏的导演，演着一场名为"压迫"的大戏，只等有人喊出——"我们不需要国王"。
