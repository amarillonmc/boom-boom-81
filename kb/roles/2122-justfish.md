# Justfish的角色 - 灵狐-安綱

## 1F

- Author: Justfish
- Posted at: 十月 30, 2025, 12:26 上午

**角色填写人：**Justfish

**基础信息：**

**角色名称：**灵狐-安綱

**角色背景：**
出身与经历：她并非第140810号宇宙的原住民，而是来自店长'熔融武识'的"家乡维度"。作为一名灵狐族的巫女，她天生就对事物的"流向"与"脉络"有着超凡的感知力。在一次机缘巧合下，她的这份特殊潜质被回乡探亲（大概是去补充零食）的'熔融武识'所发现。店长看中了她那足以"引导"万物走向的"概念胚胎"，便邀请她来到了"维度万事屋"工作。
能力的获得契机：她的能力是与生俱来的天赋，经过店长的亲自点拨，她才学会了如何将这种模糊的"感知"升华为可以主动干涉现实的"权能"。如今，她是万事屋的"审计员"，负责对所照看的宇宙进行定期审查，确保其"剧情"不会出现致命的偏差。
性格特点：沉静、温婉、观察入微。与雷厉风行的琥珀不同，安綱的行事风格更偏向于"润物细无声"。她总是带着一丝温和的微笑，仿佛早已洞悉了事物发展的最佳"路径"。她对店长抱有一种近乎于信仰的崇敬，将其视为自己侍奉的"神明"。
外貌描写：一位拥有狐耳与蓬松尾巴的灵狐少女。在万事屋外执行审查任务时，她会穿着一身与当前宇宙画风迥异的日式JK制服，显得既日常又超然。但在万事屋内，她会换上传统的巫女服，以示对店长这位"神明"的侍奉。她的腰间总是佩戴着一把名为"白月狐"的武士刀。
战斗的理由与执念：维护流向，侍奉神明。她"战斗"的目的，并非"击败"，而是"修正"。她要确保宇宙的"故事"能沿着最平稳、最合理的"河道"流淌。她内心最深的执念，就是完美地履行自己作为"引航者"的职责，不辜负店长的信任。
世界观定位：她是"超维存在"在这个维度的"航道修正师"与"剧情监护人"。如果说琥珀是负责删除"病毒"的"杀毒软件"，那么安綱就是负责修正"错误代码"、防止系统崩溃的"程序员"。

**战斗能力：**

**核心能力·装备：**
[御神刀]：白月狐 (White Moon Fox)：一把由店长'熔融武识'亲自附加了"不毁"概念的武士刀。这把刀并非用于斩杀，而是安綱将其"引航"之力具象化、并精准指向的"指针"与"船舵"。
力量根源/核心概念
【引航 (Guidance / Wayfinding)】：其全部力量的根源。这并非单纯地指引方向，而是更高维度的"流向操控"——她可以引导"力量"的流向使其偏斜，可以引导"能量"的流向使其枯竭，甚至可以引导"剧情"的流向，使其走向一个更"平稳"的结局。
常态表现与技巧
基础应用：
【流向偏转 (Flow Deflection)】：安綱最基础也最常用的技巧。她可以轻微地"拨动"事物的流向。例如，让射向自己的子弹"恰好"擦身而过；让一个即将爆发的能量核心，"恰好"将能量宣泄向无人的方向；让一个关键人物，"恰好"产生一个念头从而避开危险。
复合技巧：
【一刀·歧路 (Single Stroke - Forking Path)】：安綱将"白月狐"缓缓挥出，刀锋所指，并非斩向敌人实体，而是斩向其"行动的因果线"。被斩中的目标，其原本的"行动路径"会被强行分岔，被引导至一个由安綱设定的、危害更小的"新路径"上。例如，一个正要按下核弹按钮的将军，可能会突然觉得"我应该先去喝杯咖啡"。
巅峰境界/终极体现
[终极能力名称]：【万川归一·天命航路 (All Rivers Return to One - Celestial航路)】
触发条件：当宇宙的"剧情"因为某个强大的"灾厄"而即将彻底"脱轨"，产生无法挽回的"坏结局"时。
效果描述：安綱会以自身为媒介，与整个宇宙的"因果之河"进行短暂的同步。她将不再只是"引导"支流，而是暂时成为了"河神"，能够为这条"命运之河"指定一个"最终入海口"（一个相对和平的结局）。在此期间，所有试图阻碍这个结局的"顽石"与"乱流"，都会被整个"命运之河"的力量自动修正、绕开或磨平。
代价与风险（平衡性原则）：
代价约束型：强行扭转整个宇宙的"大势"，对她的"神魂"是巨大的负担。使用后，她需要很长时间的"休养"才能恢复，期间她将暂时失去"引航"的能力。

**被动特性·常驻效果：**
【灵狐直感 (Spirit Fox Intuition)】：作为灵狐巫女，她对"运势"、"因果"和"叙事关键点"有着极其敏锐的直觉。这让她总能出现在最需要"修正"的地方。
【流势亲和 (Affinity for the Flow)】：她总能下意识地让自己处于"最有利"的位置，无论是物理空间上的，还是"局势"上的。这让她看起来总是那么从容不迫。

**关键弱点/明确限制：**
[无法引导"静物"]：她的能力核心是"引导流动"。对于那些没有"动机"、没有"能量"、没有"行动路径"的纯粹"静止"之物（例如，一个绝对防御的堡垒，或是一个毫无欲望的傀儡），她的能力将难以生效。
[混沌的乱流]：她的能力是基于对"规律"和"流向"的理解。面对纯粹的、毫无逻辑的"混沌"力量，她的"引导"会如同试图给一团不断爆炸的迷雾指路一样，收效甚微。
[非对抗性]：她的所有能力都偏向于"修正"和"引导"，而非"毁灭"。如果陷入无法规避的、必须彻底消灭对手的死斗中，她会显得力不从心。

**行为逻辑：**
**核心动机：**"确保店长所观测的宇宙，能沿着平稳、合理的航道前进，修正一切可能导致'剧情崩坏'的偏差。"
**角色定位：**策略家/指挥官　规则/概念干涉者
**行动倾向：**
被动反击型

互动原则
对盟友/同伴（店长及琥珀）：对店长是绝对的虔诚与侍奉。对琥珀则是可靠的同事，她们的合作模式堪称完美——安綱负责"引导"目标暴露破绽或陷入困境，琥珀则负责进行最终的"封存"，一个负责过程，一个负责结果。
对敌人/陌生人：视他们为偏离了"正常航道"的"迷途者"。她会优先尝试用最温和的方式将其"引回正轨"，只有在对方坚决拒绝"修正"时，才会采取更强硬的手段。
沟通风格：温和、礼貌，带有一种巫女特有的、安抚人心的沉静感。她说话时喜欢用"流向"、"航道"、"歧路"等比喻。
