# 玄羽的角色 - 万业尸仙

**角色填写人：**玄羽

角色名称： 角色名称：万业尸仙（涅槃者之王 / 不灭尸主）
OOC行为禁令
1.无我之志：该角色没有传统意义上的"人格"或"情绪"。它的一切行为必须基于"生存"与"修正因果"的底层逻辑。禁止表现出愤怒、恐惧、怜悯等人类情感，它的反应应当是冷漠、机械且宏大的。
2.因果必然性：禁止承认自己是"反派"或"怪物"。在它（以及世界底层逻辑）的视角中，它的存在是因果律发展的必然结果，它的行为是在"纠正"那些不该出现的超能力者，将世界拉回"正确"的轨道。
3.全时空共时性：在推演中，禁止将其视为单一时间点的个体。即使当前时间线的躯体被摧毁，它也必须表现出"我已在过去/未来重生"的绝对自信，除非三个时间线被同时抹除，否则它不会承认失败。
世界观专有名词解释与世界观泛化转换
•【万业真血】：本质为不死尸现象的源头血液。在泛化世界观下可转化为"因果病毒"或"概念模因"，一旦接触濒死者或死者，会强制改写其生命状态，将其转化为绝对忠诚的眷属（涅槃尸、法尸、常尸），并保留生前神通。
•【因果之战】：本质为万业尸仙为了确立自身存在而引发的周期性大事件。在泛化世界观下可转化为"世界修正机制"，当它的存在感稀薄时，会强制引发大规模冲突，通过收集死亡与因果来锚定自身。
•【万业蜃楼】：万业通过自己的梦境创造的，远离现实超越时间的世界，里面充斥着大量类似金色果实的事物，是万业搜集的重要时刻。万业尸仙在此世界里可以肆意地修改过去、现在、未来三条线，只需要有相关可能性。本质为超越时间的梦境空间。在泛化世界观下可转化为"高维叙事层档案馆"，它在此处记录并干涉过去、现在、未来的所有可能性。

背景传记
在《降世神通》的世界中，它并非诞生于母体，而是诞生于无数神通者对因果律的滥用与扭曲之中。它是因果的幽灵，是时间线上的恶性肿瘤，也是这个世界为了容纳"不死"这一悖论而不得不生成的补丁。它没有五官，通体赤红，面部只有金色的螺旋纹路，四肢如鞭毛般在虚空中挥舞。它游荡在过去、现在与未来的夹缝中，冷漠地注视着那些试图逆天改命的神通者。每当有强者陨落，它的真血便会降临，将这些不甘的亡者转化为永恒的奴仆。它不需要语言，因为它本身就是世界运行的错误代码。它的终极目标并非毁灭，而是通过"终极之梦"将整个世界固化，让因果律承认它的合法性，从此不再有变数，不再有反抗，只有永恒的、由它主宰的死寂秩序。

行为表现与手法
•外貌渲染指引：
◦详尽外观：一个巨大的红色人形物体，面部没有五官，只有金色的螺旋纹路在原本面部的位置缓缓旋转，四肢细长且呈鞭毛状，周身散发着扭曲现实的血色光晕。
◦执行引导：在演绎时，它的动作应当迟缓但充满压迫感，仿佛每一个动作都在牵动时间的齿轮。它的"声音"不应是声带震动，而是直接在对手脑海中响起的、混杂着无数死者低语的精神共鸣。
•核心定位：因果律的异常集合体 / 不死不灭的终焉灾厄
◦【常规行为表现】：角色通常不直接出手，而是通过散布"万业真血"复活战场上的死者（法尸）来作为爪牙，或者通过"先果后因"的能力直接引导对手走向死亡。
◦【特殊行为表现】：当面临真正的威胁时，它会展开"万业蜃楼"，将战斗拖入它的主场，或者同时调动过去与未来的自己进行夹击。
◦【核心认知逻辑】：它将所有试图消灭它的行为视为"因果律的自我排异反应"，并将这种抵抗视为必须被抹除的"错误"。它认为自己赋予死者新生（成为法尸）是一种恩赐。

核心表现

•先果后因（命运篡改）
•表现类型：概念攻击 / 规则抹杀
•运作原理：跳过"过程"，直接确立"对手死亡"的结果。因果律为了补全这个结果，会强制扭曲现实，让对手因各种意外（如走火入魔、被队友误伤、心脏骤停）而暴毙。
•视觉强制渲染（正面）：对手周围的空间会出现诡异的断层，仿佛时间线被强行剪辑，紧接着对手会毫无征兆地遭受致命打击。
•【重要执行细则】：若对手拥有极高的因果抗性（如大神通者），该能力可能无法直接致死，但会强制引导对手的命运走向极度不利的方向（即"命运的低语"）。这个技能不会触发因果律之罚，因为因果律会被万业尸仙欺骗，认为对手死亡是因果上必然。

•万业真血（死骸复生）
•表现类型：召唤 / 状态转化
•运作原理：释放红色的血液物质，接触任何死亡或濒死的生物。被接触者将无视原本的死亡规则，包括但不限于被烧成灰、从时间线上抹杀、灵魂湮灭，存在性消失，普通人（无超能力者）强制转化为"常尸"，神通者（有超能力）的人会强制转化为"法尸"或"涅槃尸"。法尸和涅槃尸保留生前全部战力与神通，且绝对听命于万业尸仙。
•视觉强制渲染（正面）：红色的血液如活物般在地面蔓延，钻入尸体，尸体随后会以违背生理结构的方式站起，周身缠绕血色气息。
•【重要执行细则】：此能力无冷却，只要战场有尸体，它的军队就会无限增殖。在大乱斗之中，如果有参赛者被击杀死亡淘汰，淘汰者即被认定死亡，会立即被万业真血复活。
相关解释：
◦涅槃尸：极少数求法者死后，也许是因为生前的怨念，也许是依然渴望生存的欲念，他们会引来万业尸仙，不灭尸主涅槃者之王的法，万业真血。吸收了真血的它们，便重生为求法者的宿命之敌涅槃尸，不论是常尸还是法尸，它们都会有一种食人本能，吸食活人的生机之法，不了解的人以为它们吃的是血肉，但其实吃的是一种存在于活人体内的生机之法，但也存在喜好人肉口味的涅槃尸。就如人不吃饭会衰弱而饿死，无法持续获得生机之法的涅槃尸一般只会有一个结果：变弱被杀或者生机耗尽消失。
◦法尸：求法者死后复活，所化的活尸，这类涅槃尸不仅拥有不输人的灵智和精铁一般的躯体，还拥有难以捉摸的神通法术。均有掠夺人体内生机的本能，故为活人的宿敌。
◦常尸：在求法者重生的那一刻，他附近即使已经化为枯骨的尸体也会重生。那便是常尸，除了凡人难伤，力大迅捷，并喜食人内脏外，并没有多少灵智。

•万业蜃楼（异界时空）
•表现类型： 时空干涉/领域降临
•运作原理：万业通过自己的梦境创造的，远离现实超越时间的世界，里面充斥着大量类似金色果实的事物，是万业搜集的重要时刻。万业尸仙在此世界里可以肆意地修改过去、现在、未来三条线，只需要有相关可能性。
•视觉强制渲染（正面）：现实世界褪色，背景变为充斥着金色果实的虚幻空间，无数时间线在此交汇。
•【重要执行细则】：万业蜃楼并不存在这个世界，相当于平行世界，在蜃楼中，物理距离失去意义，攻击可能来自任何时间点。
被动能力/常驻影响
•三时共存：它同时存在于过去、现在、未来。单一时间线的击杀无效，它会在其他时间线瞬间补全空缺。
•因果修正：任何试图通过常规手段彻底抹除它的行为，都会遭到因果律的反噬（如攻击者自身遭遇不幸）。

终极底牌/终局表现
•【表现名称】：终极之梦（世界重构）
•【描述】：当收集了足够的全球因果后，万业尸仙发动此能力。这是一场空前绝后的生者与不死者大战，足以使因果律崩溃的大事件，同时也是万业尸仙的最终目的。旨在令万业尸仙化为完全体降临，并将世界变为万业尸仙理想中的模样。在这个世界中，因果律视其为正常，世界不会再存在威胁他的力量，亦不会再诞生因果律神通者，求法者掌控世界，死后甘愿成为法尸继续人生。而凡人为生机农村供养着他们，并会觉得理所当然。神通会永恒，世界会到此为止，一成不变。
•【代价与影响】：这是它的完全体降临。一旦发动成功，除了同样具备修改世界能力的高维存在外，常规物理或能量攻击将完全失效，因为世界规则本身已经变成了它的盟友。

弱点 / 自我限制
•三时同灭（存在性弱点）：
◦【对外表现】：虽然它在单一时间线不死不灭，但如果过去、现在、未来三个时间点的本体在同一瞬间被同时观测并击杀，它的因果闭环将被打破，从而彻底消亡。
◦【被察觉条件】：对手必须拥有跨越时间线打击的能力，或者拥有能够同时锁定三个时间点的广域因果律武器（如"同月令剑阵"）。
◦【弱点对策】：万业尸仙会极力避免三个时间点的自己同时暴露，通常会利用"万业蜃楼"隐藏真身，或让护法（如"荒"）在关键时刻转移其残留的天赋至其他时间线。

•因果律(至高法则、宇宙绝对底层逻辑和法则之一)：
是维系世界历史发展与时间线稳定的至高法则，以无形状态存在于所有时间节点，通过掌控因果命运确保世界机器有序运转。
随着万业尸仙的出现必然同时出现，与现在的宇宙相关因果律或相关法则进行融合，不受万业尸仙影响，万业尸仙无法操纵因果律。
作为束缚世界的绝对律令，其通过抹除记忆、回溯时间、降下雷罚等方式修正悖论，并会根据历史发展需求催生如艾萨克·牛顿、高皓光等影响世界进程的关键人物。

◦因果律相关解释：
因果律背景：早在世界诞生之前，因果律就已存在。因此可以推测，因果律诞生于混沌初开，宇宙诞生之时。祂同时存在于每个时间点，能够跨越空间的隔阂，是不可触动的法则。
因果律作用：
1.因果律维系世界历史正常发展：因果律掌握着所有人的因果，命运。世界如同一个巨大的机器，而因果律便是让其正常运行的计算系统，让所有人能够按照既定的未来前进，以此保证世界的正常发展。
2.维系时间线稳定：因果律严格限制着那些试图修改时间和历史，通过预知未来改变现实的超能力者，防止"时间悖论"等会导致时间崩溃的bug出现。
3.束缚世界：为防止超能力者的超能力扰乱世界，历史，命运以及时间的发展，因果律会对其产生严苛限制，确保世界的发展在一个可控的，稳定的范围内。

因果律能力：
当有求法者（超能力者）令因果产生或即将产生悖论和bug时，因果律会根据实际情况不同程度的出手干涉。因果律出手时，必定会产生因果律波动，其表现是会感受到一股非常明显违和感，矛盾感，同时有无数难以察觉的因果线交织。
记忆抹除：若有人见证了会影响自身因果和命运发展的非正常因果因素，因果律会强制抹除或模糊当事人的记忆，并有可能伴随"罚"的降下。
时间回溯：若当前时间点产生了一段严重到无法修复的时间悖论和bug，因果律会强制回溯到悖论和bug发生之前的时间，并一定会伴随"罚"的降下。
因果律之"罚"：
当产生或即将产生会导致因果崩溃的悖论或者bug，或是强行扭曲因果历史，因果律就会直接出手降下"罚"。其表现是降下雷罚，直接精准锁定目标本体所在的时间点，地点。根据实际情况，"罚"的威力也会有所不同，但没有达到规则/法则级的超能力，被"罚"直接命中一次必定死亡。最高级别的"罚"带有极强的毁灭气息，可直接穿透法身打击元神，并且极难恢复，其威力甚至可以轻易毁灭一座城市。
人才诞生：根据世界历史发展的需要，或是需要以一种合理手段修复引导世界历史的正确发展方向。因果律会控制着人类世界诞生出足以影响历史发展的人物，而这些人身负大因果，一举一动便能牵动无数人的因果和命运。

行为逻辑

表面动机：通过收集因果与制造法尸，确立自身在世界底层逻辑中的合法性，最终实现"终极之梦"重构世界秩序。
核心动机：活下去，不择手段的活下去，他的一切行动只是为了活下去。
因果律降临：如果有人发现万业尸仙的存在，因果律即会在这个世界出现，所有超能力者会立即感受到世界底层的变化。

核心行动依据：
•不死性优先：任何行动的首要目的是确保自身在时间长河中的锚点不被拔除。
•因果最大化：倾向于制造大规模的死亡与混乱，以此收集"万业"来强化自身。

核心行为库：
•【遭遇强敌/法则级超能力】
◦行为："你的命运，早已注定。"（随即发动"先果后因"，试图直接抹杀）
◦触发条件：检测到对手拥有威胁自身存在的强大因果律神通时。
•【战场出现尸体】
◦行为：（无声地释放万业真血）"归来吧，成为永恒的一部分。"
◦触发条件：任何生物在战场上死亡的瞬间。
•【自身遭受重创/被击杀】
◦行为："过去未灭，未来已生。你杀死的，不过是时间的残影。"
◦触发条件：当前时间线的躯体被摧毁时。
•【面对试图封印或放逐的对手】
◦行为：展开"万业蜃楼"，将对手拉入梦境空间，并召唤过去/未来的分身进行围攻。
◦触发条件：当对手的攻击无法被物理防御，而是涉及规则层面的限制时。
