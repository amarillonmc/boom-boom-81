# 玄羽的角色 - 玄羽（Rider）

- Author: 玄羽
- Posted at: 九月 09, 2025, 12:17 上午

**角色填写人：**玄羽

**角色名称：**玄羽 玄羽（Rider）

**基本属性：**属性：秩序·善
副属性：天
筋力：D
耐久：B+
敏捷：D
魔力：A~Ex
幸运：B
宝具：C～Ex

**超能力：**【固有技能】：
对魔力：D
骑乘：Ex
魔力放出（数码爆裂）：A
【保有技能】
世界的宠爱（B）
效果：
所在世界始终为玄羽提供有限度的环境和剧情支持
对魔力等级上升1级；
世界会提供玄羽行动需要的魔力；
已上效果持续生效。
数码核心（A）
效果：
数码世界为玄羽提供情报敏感度，玄羽能通过已知事项尽可能找出几方有利的情报。
单独显现（A）
效果：
因为有世界为玄羽提供魔力，所以玄羽可以不成为谁的Servant就能出现在世界上。
【宝具】
Legend-Arms数码暴龙机
玄羽和数码兽（Digimon）伙伴互相联系的工具。此外暴龙机里有一个独立的巨大数码空间，可以为数码兽伙伴提供休息、训练和恢复。
路德兽
等级：C~Ex
种类：对人宝具→对界宝具
最大范围：1～999（最大捕捉：999）
玄羽的伙伴化作的数码生物类型宝具，有自己的意识，和玄羽可以心灵感应，也可以通过语言和其他人交流。随着进化而提升宝具等级。
进化路线
路德兽 (成长期) → 提亚路德兽 (Tia Ludomon) (成熟期) → 雷神路德兽 (Raiji Ludomon) (完全体) → 布利维路德龙兽 (Bryweludramon) (究极体）
路德兽
等级：C
种类：对人宝具
最大捕捉：1人
能将自己变为武器的「Legend-Arms」中的一位。「Legend-Arms」中唯一一个将自己变为盾的数码兽。头与双手持有坚固的钢盾，既能承受来自四面八方的攻击还能守护自身。必杀技是担当防御与攻击的双手的「壁耙」。
提亚路德兽
等级：B+
种类：对军宝具
最大捕捉：2～10人
是路德兽的进化形态，其战场定位为高速救援型守护者，通过冰之结界抵御敌方攻击并协助友军撤离。核心战斗能力包括冻结敌方攻势的「冻结之盾」与高速突袭的「突袭之爪」。
雷神路德兽
等级：A
种类：对军宝具
最大捕捉：2～50人
是提亚路德兽进化形态，通过强化机械化要素获得飞行能力，以牺牲敏捷性换取能抵御究极体攻击的防御性能，配备可展开不可视带电防护罩的肩臂盾牌，必杀技包含高压电击的「闪电重击」和投射型武器「火箭飞刀」。
布利维路德龙兽
等级：Ex
种类：对界宝具
最大范围：全球（最大捕捉：1～999）
是雷神路德兽的进化形态，身体基本由盾形成，成为龙姿态的「Legend-Arms」盾数码兽的究极体。获得由神话和游戏组成的「传说之盾」的数据，进化成任何攻击都毫发无损的最强硬度之盾。外观看上去是红色数码合金制，但其硬度无法解析。布利维路德龙兽由名为「高斯（Gause）」和「菲（Fay）」 的两个独立型人工智能负责全方位视觉。「高斯」「菲」也持有火焰屏障，屏障范围可扩展至布利维路德龙兽。由于「高斯」「菲」的活动范围远到天涯海角，因此也可以说屏障覆盖了全世界。必杀技是从火焰翅膀放出烈火「红莲暴风」，以及还有分离翅膀，展开火焰屏障的同时突击敌人的「爆风突击」

**行为逻辑：**成为数码世界的救世主后，受型月世界（出处《Fate》）的地球意识盖亚邀请来到型月世界。
但不是不知道怎么回事来到的只有玄羽的一个侧面，这个侧面是他在15岁黑暗四天王的阴谋下和另一个伙伴兹巴兽 (Zubamon) 走失时的时期的自己。
隐约记得自己以前参与过类似的世界乱斗。同时，知道目前的自己的状态是《FATE》中的英灵。
此时的玄羽心怀正义和荣誉，惩歼除恶，保护弱小，已经成长为独当一面的救世主，但距离成为救世还有一段很长的路要走。
