# 格拉摩根的角色 - 莉娅斯·乌洛玻尔恩

## 1F

- Author: 格拉摩根
- Posted at: 十二月 31, 2025, 03:07 上午

**角色填写人：**格拉摩根

**基础信息：**

**角色名称：**莉娅斯·乌洛玻尔恩

**角色背景：**
星海竞技场的聚光灯下，莉娅斯·乌洛玻尔恩的身影总能带来双重惊喜——时而身着缀满绒毛装饰的森系裙摆，唱响治愈人心的歌谣；时而换上蕾丝与缎带交织的暗黑礼裙，演绎神秘冷冽的乐章。这两种风格迥异的姿态，是她作为诗姬的专属印记，更是她践行"一直在一起"契约的核心方式。

诗姬状态下的莉娅斯，所有行动都围绕着"守护契约"与"平衡力量"展开。化身灵动娇憨的模样时，她会卸下龙神的所有威严，做一个与生灵共鸣的温柔歌者。她的歌声里藏着森林的私语，能唤来身披星光的灵兽相伴左右，抬手就能织出柔软的能量屏障，为身边的人分担伤害、抚平伤痛。她会蹲下身和林间的小兽低语，用鳞片化作的糖果安抚受惊的小家伙，也会在演唱会后与粉丝围坐，倾听他们的心愿。这份对所有生命的共情，让她成了诗姬阵营里最受喜爱的守护者，也让"一直在一起"的约定，延伸到了每一个生灵身上。

当她换上冷艳神秘的装扮，便会展现出截然不同的一面。此时她的歌声不再是纯粹的治愈，而是蕴含着轮回法则的镇魂曲，带着精准的掌控力，守护着舞台与世界的秩序。她能以旋律凝聚暗影锁链，温柔却坚定地束缚住扰乱秩序的威胁；唱响的乐章会形成无形的结界，将负面能量隔绝在外，既不伤及对方性命，又能有效阻止危机蔓延。她在舞台上优雅转身时，披风展开如蝠翼，暗紫色的光晕流转间，尽显沉稳与可靠——这份克制的力量，让她成为守护竞技场安宁的重要防线，也让她的契约守护多了一层坚定的底色。

这两种姿态从未让莉娅斯感到割裂，反而让她的守护更加全面。灵动的治愈歌声是羁绊的基础，冷艳的暗影掌控是守护的底线。她常常在演唱会上自由切换状态：先用温柔的旋律凝聚全场热情，再以神秘的歌声构筑坚实屏障，两种力量相互配合，既温暖又果决。当身边的伙伴遇到危险时，她会瞬间切换到冷艳形态，用暗影力量化解危机；而当表演氛围过于冰冷时，她又会融入灵动的气质，让舞台多一份温度。对她而言，这两种姿态不是相互独立的标签，而是践行"一直在一起"誓言的两种方式——无论是温柔相伴，还是坚定守护，都是契约的核心意义。

但在这双重诗姬姿态之下，始终潜藏着辰之十二神皇乌洛玻罗斯的原始形态与力量。那是一尊缠绕星辰的巨型衔尾龙，鳞甲如暗紫水晶般折射出慑人的光芒，翼展遮天蔽日时，整片天地都会陷入永恒的暮光。这尊"永劫死龙"掌控着最纯粹的轮回法则：能吞噬威胁的力量本源，将其打回初始虚无；能以蛇口衔尾的轮回之环重置时空，让所有破坏归于平静；其邪黑炎波足以撕裂次元壁垒，焚烧一切来犯之敌。

这股毁天灭地的力量，被莉娅斯用两种姿态的羁绊牢牢封印。灵动形态下的生灵共鸣，是她对生命的承诺——上古时期，她曾见轮回之力吞噬太多无辜，如今便以与万物相伴的温柔，软化毁灭的锋芒；冷艳形态下的精准克制，是她对力量的约束——用自身的意志，将原始的轮回之力转化为可控的守护手段，而非肆意破坏。只有当伙伴或粉丝面临生死绝境时，封印才会短暂松动：曾有逆転大陸的残党突袭演唱会，莉娅斯背后同时浮现出巨型龙影与灵兽群，灵动姿态的生命能量为龙影注入韧性，冷艳姿态的暗影法则为龙息划定界限，既冻结了敌人的攻势，又护住了在场所有人，完成了一次完美的协同守护。

如今的莉娅斯，在两种姿态的交替演绎中，诉说着轮回与羁绊的真谛。她既是林间生灵的伙伴，也是暗影中的守护者，两种模样让她的契约誓言更加厚重。而那潜藏的永劫死龙之力，从来都不是威胁，而是她践行"一直在一起"誓言的最终底气——无论以何种姿态站在舞台上，她都会用最适合的方式，守护着珍视之人，让轮回成为羁绊的永恒载体。

**战斗能力：**

**核心能力·装备：**
- 诗姬形态（莉娅斯·乌洛玻尔恩）：以歌声为媒，凝聚羁绊，兼具治愈守护与暗影约束之力。
​
- 神皇形态（辰之十二神皇 乌洛玻利亚斯）：掌控轮回法则，拥有不灭再生与吞噬威胁的太古伟力，维系时空秩序。

**被动特性·常驻效果：**
诗姬形态（莉娅斯·乌洛玻尔恩）

1. 契约共鸣：与伙伴/生灵的羁绊越深，自身治愈、守护的基础效果越强，是"一直在一起"誓言的被动体现。
​
2. 歌声传导：力量天然以歌声为载体，无需额外引导即可随旋律扩散，覆盖范围内的伙伴或目标。
​
3. 生灵亲和：自动吸引周边友善生灵靠近，形成协同，无需主动召唤或沟通。
​
4. 形态反噬：双形态切换后会进入短暂虚弱期，防御与恢复效率小幅下降，是封印力量的天然代价。

神皇形态（辰之十二神皇 乌洛玻利亚斯）

1. 轮回不灭：濒死时自动触发轮回法则，重置自身状态至全盛，同时清除所有负面影响，是永恒存在的核心被动。
​
2. 能量虹吸：持续吸收周边敌方逸散的能量本源，缓慢提升自身攻防，无需主动摄取。
​
3. 秩序压制：对混沌、破坏类力量天生具备压制力，使其效果自然削弱，是维护时空平衡的本能。
​
4. 全域感知：无需专注即可感知大范围时空与能量变动，提前察觉威胁，无感知盲区。

**关键弱点/明确限制：**
1. 力量断崖式削弱：从十二神皇级原始龙神转为诗姬，以双形态羁绊强行封印永劫死龙的太古伟力，日常仅能调用轮回法则转化的低阶可控力量，对抗同级神明/高阶威胁时战力严重不足；维持封印需持续消耗精力，无法长时间高强度作战。
​
2. 诗姬皮囊的双重束缚：一是媒介依赖，力量必须通过歌声施展，被沉默/干扰发声则能力近乎瘫痪；二是承载上限低，皮囊无法承受原始龙神之力，解封绝境形态会造成反噬损伤，形态切换后有防御与移速短暂下降的窗口期，易被针对；双态共鸣结束后还会陷入强制虚弱期。
​
3. 契约守护的致命软肋：所有行动以羁绊契约为最高准则，过度执着于守护伙伴与"一直在一起"的誓言，易被敌人通过挟持羁绊对象或制造守护陷阱拿捏；为避免伤及无辜，战斗中会刻意克制力量输出，甚至放弃最优进攻时机，给对手留足反制空间。

**行为逻辑：**
**核心动机：**莉娅斯·乌洛玻尔恩作为诗姬的行为逻辑，核心围绕「契约守护」与「力量克制」展开，以双形态适配不同场景，始终践行"一直在一起"的羁绊誓言：

1. 契约为根：所有行动的核心是守护与伙伴、粉丝缔结的羁绊，以"不辜负约定"为最高准则，无论是治愈陪伴还是直面威胁，都以维系这份联结为出发点。
​
2. 双态适配：灵动形态主打"温柔守护"，用歌声凝聚生灵共鸣、治愈伤痛，以共情化解冲突；冷艳形态侧重"秩序维稳"，以暗影力量约束威胁、隔绝负面，避免伤害扩大，两种姿态按需切换，实现全面守护。
​
3. 力量克制：刻意封印原始龙神的毁灭之力，将轮回法则转化为可控的守护手段——不追求歼灭敌人，而是以"约束+庇护"平衡战局，既不让威胁蔓延，也不伤及无辜，呼应对生命的敬畏。
​
4. 歌声为媒：以演唱为核心行动方式，将羁绊、治愈、约束的力量融入旋律，既通过舞台表演凝聚伙伴信念，也在危机中以歌声快速施展能力，让守护更具温度与感染力。
