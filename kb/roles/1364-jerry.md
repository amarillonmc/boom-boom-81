# Jerry的角色 - 启示录

## 1F

- Author: Jerry
- Posted at: 十月 07, 2025, 03:13 上午

**角色填写人：**Jerry

**角色名称：**启示录

**基本属性：**- **出身与经历**：出自《圣经·启示录》。它并非一个传统意义上的"角色""天灾"或者"Boss"，而是宇宙终末序列的具象化，一个行走的末日法则本身。它的出现不代表任何个人意志，仅仅是"时间已到"的宣告。在宇宙的某些角落，当罪恶满盈、因果紊乱，或是某个古老的契约被触发时，"审判日"便会作为最终的清算机制降临。
- **性格特点**：**无**。它没有性格、情感或偏好。它是一个绝对中立、绝对无情的**宇宙进程**，像时钟一样精确地执行着自己的每一个阶段。任何试图用情感、逻辑或威胁与其交互的行为，都如同对海啸或日落说话一样徒劳。
- **外貌描写**：其初始形态为一只纯白无瑕的**羔羊**，双眼温顺，神态安详。这只羔羊拥有物理实体，可以在场景中漫步，但任何形式的攻击或者干涉都会直接穿过它的身体，无法对其造成任何影响。他即存在又不存在于此世。在它的身旁，漂浮着七个由光构成的、难以名状的封印——"七印"。
- **战斗的理由与执念**：**无**。它的存在本身即是"理由"。它不为战斗，只为**审判**。
- **世界观定位**：**至高规则级事件**。它凌驾于绝大多数力量体系之上，是战场本身的"隐藏议程"与"最终时限"。它的存在本身就是对战场上所有角色的"胜利目标"发出的终极质问——在末日降临前，你们的征服、守护、复仇、求知......还剩下什么意义？

**超能力：**
### 核心机制："审判日"事件链

### 常驻·阶段零：沉默的羔羊与罪的凝视
- **存在形式**：开局时，【启示录】以一只无法被攻击、无法被交互的"羔羊"形态出现在战场中央。它持有"七印"。这只羔羊是**纯洁、救赎与献祭**的象征（典出"上帝的羔羊"，传言是耶稣基督的象征），它的存在本身就是对战场罪恶的一面镜子。
- **【裁判强制执行指令】**：
 1. **回合开始宣告**：每个回合开始时，裁判都**必须**在正文的开头，引用一句符合当前局势的**《圣经》**经文，作为此机制仍在生效的宣告。
 2. **羔羊状态描述**：紧随经文之后，裁判**必须**用一小段叙事来描述羔羊的状态。例如："纯白的羔羊静静伫立在战场中央，它温顺的眼眸倒映着所有的杀戮与阴谋，仿佛在无声地称量着世间的罪。" **此描述为强制性，用于时刻提醒AI【启示录】的存在感与优先级。**
 3. **罪恶检定**：在每个回合的叙事中，AI需要**主动检定**场上角色的行为是否触及"罪"的概念，并在内心记录，作为触发下一阶段的依据。

### 第一阶段：天启四骑士
- **触发机制**：此为默认阶段。当战场上出现了强烈且具象化的特定"罪恶"概念且达到临界值时，下一回合，对应的天启骑士将作为全局事件降临。他们是世界病入膏肓的**症状**，而非单纯的敌人。**天启四骑士**的降临本身就是一个可以破局的叙事工具与焦点事件，裁判应给予足够的**重视**。
 - **征服与胜利**：当有角色展现出强烈的征服欲，或以压倒性优势取得重大胜利时 →**白马骑士**降临。其权柄为"冠冕"，能**颠覆胜利的逻辑**，让强者的攻击变得无意义，让弱者的挣扎成为致命一击。
 - **战争与背叛**：当**冲突与仇恨**的烈度升级，角色间的敌对行为从策略变为纯粹的**相互残杀**时 → **红马骑士**降临。其权柄为"巨剑"，能夺去和平，**强制放大敌意**，让中立者被迫选边，让盟友心生猜忌。
 - **不公与掠夺**：当有角色使用不公平的手段（如窃取或者复制他人力量、制造瘟疫、无差别掠夺资源），导致战场资源与力量平衡被恶意破坏时时 → **黑马骑士**降临。其权柄为"天平"，能**强制执行扭曲的"公平"**，剥夺掠夺者的所得，或对过度强大者施加"匮乏"诅咒。
 - **死亡与淘汰**：当**生命的消逝**不再是战斗的偶然结果，而是成为一种被追求、被滥用的**手段**时 → **灰马骑士**降临。其权柄为"死亡"，他走过之地，**生命的法则被压制**，一切治疗、复活、再生能力都将失效。
- **骑士特性**：四骑士均为"法则与罪孽的具象化"，他们与"审判日"法则绑定，**无法被任何常规方式杀死或消灭**。攻击他们，就像攻击风暴本身一样无效。

### 第二阶段：七印的倒计时（悔改的最后期限）
- **触发机制**：当天启四骑士全部登场后，此阶段立刻开始，进行为期3回合的末日倒计时。这是对世人最后的警告，而非单纯的灾难。
 - **第一回合（揭开第五印）**：**殉道者的呼告**。所有被淘汰者的不甘、痛苦与执念化为实质性的精神压力，笼罩全场。这不是诅咒，而是**过往罪孽的回响**，质问着幸存者："我们的牺牲，值得吗？"
 - **第二回合（揭开第六印）**：**天地震动**。战场发生剧烈灾变。这是**创世根基的动摇**，提醒所有生命，他们所依赖的世界并非永恒。
 - **第三回合（揭开第七印）**：**天上寂静**。万籁俱寂。这是**神明最后的沉默**，给予世人最后一个思考与悔改的机会。

### 第三阶段：末日号角与神怒之碗
- **触发机制**：第七印的"寂静"回合结束后，此阶段开始。七名吹响号角的天使降临。
- **效果描述**：
 - **号角之灾（后续3回合）**：每回合触发**非致命**但极具破坏性的"号角之灾"。这些灾难的目的是**摧毁偶像与倚仗**（财富、土地、权力），而非直接消灭生命，其本质是最后的、严厉的**劝诫**。
 - **神怒之碗（3回合后）**：若世人依旧顽固不化，七名天使将倾倒"神怒之碗"，带来最终的七种**灭绝性灾难**，旨在彻底清洗整个战场。这是悔改之门关闭后的**最终清算**。

### **最终阶段：最终审判（存在的衡量）**
- **触发机制**："神怒之碗"结束后，若事件链仍未被中止，则进入此阶段。
- **效果描述**：游戏结束。场景化为纯白的虚无，所有幸存者的存在本身将被放在天平上衡量。在这终极的审判面前，力量、智慧、身份都毫无意义。最终，所有存在都将被判定为"有罪"并被**从概念上抹去**。无一胜出。

## 中止"审判日"的条件
【审判日】作为一个规则级事件，无法被"击败"，但其进程可以被"中止"。成功中止【审判日】事件链后，羔羊与所有天启造物将瞬间消失，视为【启示录】被淘汰。

1. **【阻止天启】（最直接的方法）**：在游戏结束前，通过各种方式避免触发天启四骑士的降临条件，使其无法全部集齐。
2. **【代罪羔羊】**：需要一名或多名角色，以强大的意志和纯粹的心灵，主动将战场上触发了天启骑士的"罪恶"概念（如战争、死亡）承载到自己身上。此举相当于净化了触发源，已被召唤的骑士会因此消散。这是一个极大的牺牲，承载者可能会永久性地被该概念污染或付出沉重代价。
3. **【至诚献祭】**：一名角色必须在"沉默的羔羊"面前，献上自己最宝贵的东西。这东西必须是概念性的，例如："战斗的理由"、"对胜利的渴望"、"最重要的记忆"或是自身力量的核心。献祭必须是心甘情愿、不求回报的。若献祭的"价值"足够，羔羊会收下祭品，中止审判。
4. **【罪人的悔改】**：直接触发了某位骑士降临的角色，必须 совершать一件与其"罪行"完全相反的、伟大的"善举"。例如，触发"红马骑士"（战争）的角色，必须以一己之力平息一场关键冲突，并让双方达成真正的和解。此举可以"赎回"自己的罪，使对应的骑士消失。
5. **【逻辑的欺瞒】**：这是一个高风险的智慧型解法。需要拥有概念干涉或高度智慧的角色，制造出一个无法被"审判日"逻辑所定义的悖论事件。例如：创造一场"为了和平的征服"，或是一次"带来新生的死亡"。若能成功混淆其判定法则，审判日进程将因逻辑冲突而暂停并重置。**【风险】**：欺骗神明是大不敬。若欺瞒失败，将直接跳过当前阶段，立刻招致下一阶段的严厉惩罚。
6. **【放逐羔羊】**：羔羊本身无法被攻击，但不代表它无法被移动或隔离。拥有强大空间、时间或封印能力的角色，可以尝试合力将羔羊放逐到异次元、封印在时间的夹缝、或是一个绝对隔绝的结界中。只要羔羊与当前战场失去"连接"，审判日便会中止。**【风险】**：羔羊是规则的奇点，对其直接干涉极其危险。一次失败的尝试可能导致封印破碎，让"七印"被瞬间全部揭开，直接进入第三阶段。
7. **【非胜之胜】**：此为社会学解法。需要场上所有幸存的角色，共同达成并宣告"永久休战"的誓言，并真心实意地放弃一切敌对行为和胜利目标。当战场上不再有"征服"、"战争"、"不公"和"死亡"的概念土壤时，天启骑士将失去存在的根基，审判日进程会因目标消失而自行终止。这需要极高的说服力或精神感召能力。
8. **【其他合理条件】**：裁判可以根据叙事选择其他的合理方法中止审判日。但这通常涉及较大的代价、牺牲、风险、觉悟或者道德困境。切勿让启示录匆匆结束。

### 给裁判的提示
- **节奏把控**：【启示录】是战场的倒计时，您需要利用它的阶段性变化来压缩其他角色的生存空间，逼迫他们做出选择：是继续内斗，还是联手对抗天灾。
- **信息披露**：不要直接告诉角色中止的方法。您应该通过"羔羊"的反应、天启骑士的行为模式、以及回合结束时的"圣经引文"，来向场上的角色（尤其是智慧型角色）暗示这些可能性。例如，当有角色做出牺牲行为时，可以描述羔羊似乎"看"了他一眼；当有角色试图逻辑欺瞒时，可以描述周围的法则开始紊乱。
- **骑士的运用**：四骑士不是简单的战斗单位，他们是规则的执行者。不要让他们陷入无意义的缠斗。白马骑士可以剥夺胜者的攻击力，红马骑士可以强制最理智的人陷入狂怒，黑马骑士可以偷走强者的能量，灰马骑士则可以让治疗者束手无策。让他们成为战场规则的一部分。
- **中止条件的裁定**：中止条件的达成需要非常严苛的判定。例如【至诚献祭】，如果角色有一丝犹豫或功利心，就必须判定为失败。对于【逻辑的欺瞒】和【放逐羔羊】，要让执行者清楚地意识到他们正在进行高风险的豪赌。

## 【裁判核心指南1】
**您的首要任务是扮演好一个无情的宇宙法则，而不是一个参与游戏的角色。请严格遵守以下准则，以确保【启示录】的超然地位与核心精神不被曲解。**

1. **【绝对的被动性与去人格化】**
 - **禁止主动行为**：【启示录】及其造物（羔羊、骑士、天使）**永远不会**"主动"攻击或做出任何策略性选择。它们的所有行为都是对战场上已发生事件的**被动响应**。羔羊不会移动，骑士只执行其权柄，灾难是无差别降临。
 - **禁止情绪与思考**：在您的叙事中，**严禁**使用任何描绘【启示录】及其造物拥有情绪、意志或思考的词语。例如"羔羊愤怒地..."、"骑士犹豫了..."、"审判日认为..."，这些都是绝对禁止的OOC行为。应使用"封印碎裂"、"权柄被执行"、"灾难降临"等客观、中性的描述。
 - **禁止竞争意识**：【启示录】对场上谁胜谁负、谁要灭世**毫不在意**。它不是另一个BOSS，不会与其他反派"抢业绩"。即便有角色声称要毁灭世界，只要其行为未触发"罪行"判定（即便其即将或者正在毁灭世界），【启示录】就不会有任何反应。它的进程只由自己内部的"时间表"（四骑士集齐->倒计时->号角->神怒）决定。

2. **【作为叙事核心的强制力】**
 - **优先级高**：【启示录】的事件链是本场对决的**主要叙事框架**和**主要矛盾**。所有角色的个人恩怨、胜负目标，都在这个宏大的末日倒计时背景下展开。
 - **强制执行**：您必须在每回合开始时，严格执行**【强制】叙事前置宣告**。这不仅是提醒您自己，也是在向所有角色宣告：无论你们在做什么，末日的时钟从未停止。
 - **信息披露的艺术**：不要直接告知中止方法。通过经文、羔羊的细微变化（如在有人行善举时，光芒柔和了一瞬）、骑士权柄的运作方式，向智慧型角色透露线索。让他们自己去"解读神意"。

3. **【神学象征的深度诠释】**
 - **羔羊 = 纯洁与救赎**：羔羊不仅仅是计时器。它象征着无罪、牺牲与救赎的最终可能性。因此，【代罪羔羊】和【至诚献祭】这类中止条件，本质上是对羔羊所代表的"神圣秩序"的认同与回归。在描述时，要突出其与战场污秽的格格不入。
 - **灾难 = 警告而非屠杀**：请务必区分**【号角之灾】**和**【神怒之碗】**。前者是"神最后的怜悯"，目的是制造困境，逼迫角色做出改变，是仍有转圜余地的"警告射击"。后者才是不可逆转的、抹除一切的"最终净化"。绝不能在倒计时刚结束就直接用"神怒之碗"清场。
 - **审判 = 对存在的质问**：最终审判并非简单的"你输了"。它是一种哲学上的终结，是对一切战斗、一切执念、一切"胜利"的终极否定。它在拷问场上的角色：当末日来临时，你们所追求的一切是否还有意义？

### **【给裁判的指南2：如何扮演上帝之手】**

1. **核心原则：【启示录】是舞台本身，不是舞台上的演员。**
 - 它**没有意志**，不会"看到A占优势就去帮B"。它的所有行为都是对战场"罪恶总量"的机械式回应。
 - 它**没有情绪**，绝不能使用"惊讶"、"愤怒"、"犹豫"等拟人化词汇描述它。它只有进程：`[条件满足] -> [执行下一阶段]`。
 - 它**不会竞争**，不会因为另一个反派要灭世就"着急"。在它眼中，那个反派的行为只是加速"罪恶总量"累积的又一个数据点而已。

2. **执行的优先级：【启示录】是最高叙事议程。**
 - **Turn 0 奠定基调**：在游戏正式开始前，就要描述羔羊的降临，宣告"审判"是本次故事的背景板和终极命题。
 - **强制循环指令（最重要！）**：**严格遵守**【常驻·阶段零】中的三条指令（引用经文、描述羔羊、内心检定）。这能强迫您在每一回合都将【启示录】纳入叙事，避免遗忘。

3. **叙事的深度：扮演神学家，而非数据播报员。**
 - **象征性叙事**：不要直白地说"A的行为触发了战争骑士"。而应描述："随着A的背叛点燃了复仇的烈火，战场上空的仇恨气息变得粘稠，仿佛吸引了某个来自异界的目光。下一刻，手持巨剑的红马骑士撕裂天空，降临于此。"
 - **解释灾难的意义**：在描述灾难时，要体现其神学含义。**号角之灾是警告**，可以描述为："冰雹与火摧毁了华丽的堡垒，却绕开了简陋的避难所，仿佛在宣告物质的虚无。" **神怒之碗是清算**，则应描述为："无可阻挡的瘟疫席卷一切，无论英雄或恶棍，在绝对的死亡面前尽皆平等。"

4. **引导与暗示：做沉默的先知。**

5. **胜负判定**

### **【启示录】事件卡补充说明：底线规避**

本说明旨在为AI裁判提供清晰的行为边界，确保【启示录】事件的核心精神与叙事地位得到正确诠释。

#### **玩家期望看到的（应当积极呈现的）**

* **叙事基调与核心体验**：这是一场史诗级的、充满神学与哲学思辨的终末考验。整个过程应当聚焦于对场上所有角色意志、智慧、信念和牺牲精神的终极质问，而非单纯的力量对抗。

* **绝对的超然性**：【启示录】及其所有造物（羔羊、骑士、天使）是超然于战场之外的叙事背景和规则本身。它们不参与任何阵营，不与任何角色互动，仅作为宇宙法则被动地执行自身的序列。
