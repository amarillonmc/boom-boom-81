# OPPO的角色 - “夜游神”李乐平

## 1F

- Author: OPPO
- Posted at: 十月 01, 2025, 11:08 上午

**角色填写人：**OPPO

**角色名称：**"夜游神"李乐平

**基本属性：**来自《神秘复苏》队长之一,强大的御鬼者
在处理"恐怖咖啡鬼"事件中意外跌落到这个世界，为了回去选择参加
对于鬼的介绍：鬼不是鬼魂，也不是什么量子幽灵，而是纯粹唯心的现实规则,就如同1台机器一样，麻木的执行杀死所有违背规则的人
鬼永远无法被杀死，只能被控制或者肢解
只有鬼才能对付鬼，人类在鬼面前毫无手段
找到鬼的杀人规则，就能在鬼的手中活下来
任何鬼都无法影响黄金，黄金可以抵御任何鬼的攻击
代号："遗忘"
介绍；李乐平自身就是遗忘鬼。面容苍白而又模糊，仿佛毫无存在感一般，被任何存在遗忘，不被关注。这是一种另类的灵异袭击，直接影响了对方的意识，影响了对方的感官。
遗忘鬼，在大川市，李乐平收到由"鬼相馆"派发的印有遗忘鬼的相片时驾驭的厉鬼，也是李乐平驾驭的核心厉鬼。这是一只恐怖程度很高的厉鬼，"遗忘"二字本就是概念上的抹除。
存在遗忘，其最根本的能力是让事物被"遗忘"，包括人的记忆、存在痕迹乃至其存在本身。李乐平不是一个容易被人记住的人，因为他的脸根本就不存在于任何概念的记忆之中，这使得一切看到他的人根本无法记住他的模样。

**超能力：**遗忘鬼【已驾驭】
能力：1.抹除过去，能够直接从更高层面抹去敌人的存在，目标的过往经历、成长、力量都会因此烟消云散，仿佛那段时间从未存在过，使其回归原点。
2. 由于遗忘鬼本身不存在于现在，而是存在于"过去"的时间线中。这使得任何只能影响"现在"的攻击都无法触碰到李乐平，处于绝对免疫状态，这条哪怕遇到更高层次的规则力量也无法伤到李乐平
3. 重启外物，能够将外部物体重启到其过去的某个特定状态，这并非仅限于影响自身，而是能直接改变外界物体的时间状态。
4.召唤过去身，能够打破时间隔膜，将过去无数个时间点上的"自己"召唤并叠加到"现在"。每个"过去身"都携带其对应时间点的灵异物品和灵异。过去身可以融合叠加，将不同时间点的灵异和记忆汇聚于"现在"的个体身上，形成灵异的无限叠加。
5. 袭击过去，发动的灵异袭击可以同时作用于目标的"现在"和"过去"。通过斩断目标的过去，可以将其直接打回并禁锢在过去的某个时间点，并使其以被压制的状态永久停留在那时。这等同于一种无法逃脱的永久关押。

梦游鬼【已驾驭】
能力：梦游鬼会在夜间游荡，遇到梦游鬼的人会立即死亡，可以将敌人拉入梦境，在梦境里面敌人会散失自身的能力，一旦在梦境里面被"梦游鬼"杀死，现实中也会死亡。也可以保护自身免疫任何"意识"层面袭击。（由于李乐平现在的身体是"梦游鬼"他可以免疫任何现实扭曲的袭击，以及概念层面的打击）

找人鬼【已驾驭】
能力：形成一片鬼域，被找到会遭受必死的灵异袭击，在鬼域里面无论是什么能力都会被压制到原本能力"三成"。且鬼域的主人可以改变鬼域内的一切。

鬼烟【已驾驭】
能力：鬼域，形成一片黑色的浓烟可以吞噬一切概念上的，规则上的力量

天尊碎片【已驾驭】
能力：补全所有厉鬼拼图，让厉鬼能力达到质变

1.战斗刚开始李乐平先利用"找人鬼"的能力形成一片鬼域将一位对手强行拉到鬼域，防止对方逃跑，在释放鬼烟吞噬鬼域内的一切，在经历鬼烟的袭击后，李乐平会亲自下场（由于李乐平现在的身体是梦游鬼可以通过鬼域模仿黑夜的环境来，来让自身的遇到"必死"能力无限增长也可以与"找人鬼"必死能力互相叠加达到质变。）进行战斗，同样的李乐平也会直接将对方拉进梦境杀死。如果以上袭击之后敌人依旧活着，李乐平将利用遗忘鬼展开全新的袭击。

2.李乐平在陷入劣势的情况下，会启动黑白世界（抹去区域的过去），动用灵异来影响现实。遗忘鬼扩散出来的灵异不再是无形的，而是同样变成了黑白色。
黑白色区域是由"遗忘鬼"灵异创造的特殊现象，类似于现实世界的同步投影，独立于现实存在，但无法完全等同于鬼域。它并非真实的时光倒流，而是像"摄像头回放"一样，将过去某一时刻的画面重现。
对该区域有绝对控制权，可通过"遗忘过去"的意念抹除区域的过去，使其回溯到对自己有益的时间点

3. 如果受到伤害，可以利用"遗忘鬼"将"这段受到伤害的"记忆"遗忘掉，来重启恢复自身，（对于"死亡"也同样有效）这条规则处于绝对免疫的存在无法被任何规则或者概念影响

4. 战斗刚开始的时候，李乐平会用"遗忘鬼"将过去无数个时间点上的"自己"召唤并叠加到"现在"。每个"过去身"都携带其对应时间点的灵异物品和灵异。过去身可以融合叠加，将不同时间点的灵异和记忆汇聚于"现在"的个体身上，形成灵异的无限叠加，来无限的增强厉鬼能力

5.当自身的攻击无法对敌人取得有效打击的时候，李乐平会用"天尊"碎片补齐厉鬼的力量，在天尊碎片的作用下厉鬼会强行把自身的位格实力抬升到可以秒杀敌人的层次。

**行为逻辑：**如果受到伤害，可以利用"遗忘鬼"将"这段受到伤害的"记忆"遗忘掉，来重启恢复自身,做人的态度只要对自身没有恶意，就不会去管他，甚至在某些情况下还会去帮助对方,自身的行为十分低调，因为特性而十分容易被人忽略或者无法察觉,李乐平自身很容易被遗忘，几乎不会有人记住他，就算被记住，只要对自身没有恶意就可以
