# tdd的角色 - N12-伊洛洁菲

**角色填写人：**tdd

**基础信息：**

**角色名称：**N12-伊洛洁菲

**角色背景：**
在世界的最末，人造人的父亲带着莫大的觉悟将自身改造成了救世的怪物和黑色的穹顶，而"塔"的任务则全部交给了人造人。

N型12号人造人，也可以称为伊洛洁菲（Elkef）——她不喜欢别人用编号称呼自己。她是人造太阳的炉芯，将它点亮后也兼职人造太阳的维持监视。

但她不喜欢单调的工作，本就闲散的她将自己的监视工作推给了N13-奈喀纳伊，自己则经常跑出"塔"外到处玩。

她用这种方法逃离单一的生活，她甚至有自己的个人社交账户"尼莫@official"——这和其他那些古古板板的人造人有天壤之别。当然，她的零件磨损速率也是人造人中最快的（仅相对而言，实际上她的单个零件磨损时间仍然超过普通人类的寿命时间）。

大哥很闷但很温柔，姐姐话特别的多......

而......

巨大的堤坝上，少女踢着左左右右的脚步，哼着跑调的小曲，今天也从东走到西呀～

——————
外观描述：看起来是19岁的少女，粉色的散乱长发，经常穿着白色的睡衣。
毕竟"塔"作为承载人造太阳的结构，其位于大洋的另一边的"尼莫点"，不可能有人造人兄弟姐妹以外的人存在，所以她的穿衣风格相当随意。

性格描述：虽然很喜欢出门玩，也很活泼，但她其实不算外向——这并不矛盾，在堤坝以内，"出门"后其实还在她家里。
这代表她一旦见到人造人以外的生物（对的，堤坝以内没有生物能扛住人造太阳的温度），她都会经历害怕、害羞、好奇，然后才可能交流。她像一只被保护的太好的小兔子，对外界的真实一无所知（她对外界的认知全部来源于社交媒体）。
如果感到委屈或被否定，她可能会无意识地用【炉芯】的权能点燃身边的一些小物体（小花、小石子什么的），这是情绪的无意识外泄。事后她会慌张地踩灭，并反复道歉"对不起对不起我不是故意的......"。

**战斗能力：**

**核心能力·装备：**
【炉芯】伊洛洁菲拥有非常不讲道理的「点燃」权能。她能够将任何物体设置为"可燃的"或者"正在燃烧的"，无视其原本是否可燃，随后——点燃它。
因这个权能点燃的火焰不同于普通火焰，那是能够同时燃烧整个物体，连着物质的存在一同燃烧的"星焰"。这种火焰不会因助燃剂的阻断或温度的降低而熄灭，物质将依据质能转换公式被全部烧成无序的能量（当然，仅有她自己可以让这种燃烧中途停止）。
"你以为人造太阳为啥让我来点？"

**被动特性·常驻效果：**
【N型人造人】伊洛洁菲作为N型人造人的通用能力。这个能力使她的机体拥有对高温的极高抗性——至少人造太阳级别的高温无法奈何她。
并且作为人造人，其对物理打击的抗性也较高。

【"塔"的管理者】这是由父亲赋予的能力。这个能力让伊洛洁菲拥有极高的信息整合能力和信息获取能力，其注意力只要想集中就一定能够集中——虽然由于性格原因这很难。
这种信息能力让她同时有极高的危险预知能力。

**关键弱点/明确限制：**
【功能机】伊洛洁菲的存在意义稀薄——她主动推开了点燃太阳的后续责任。这使得她本就较高的精力无处释放，可能让她仅因"好奇"而误入险境。

**行为逻辑：**
**核心动机：**玩！
看看外面的世界，等我回家要跟那些板着脸的兄弟姐妹们好好说说——这次他们肯定不会说我是只从网上知道的。
骗你的，我要作为"我"而活着，而不是作为某个型号。

**角色定位：**侦查/潜行者　规则/概念干涉者
