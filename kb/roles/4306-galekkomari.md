**角色填写人：**galekkomari

**角色名称：**瑞克·拉玛尔

**基本属性：**
《常磐大逃杀》时空帧＃１０４Ｂ６中，近年在湾城深夜街头出没的神秘女性。金发，浅蓝眼，长发如被夜风吹散的金线，发侧佩有蓝色水晶羽饰，身着黑、白、蓝三色的战斗礼装，肩部有翼状装甲结构，裙摆如不规则羽片层叠，靴口带有小型羽饰。她的周围总是盘旋着蓝白色的灵体鸟群。

她自称"瑞克·拉玛尔(Ricolamal)"，但无人知道这是否是真名。她的阵营属性接近混乱·善，喜欢夜晚高楼顶层的风、飞行器，尤其喜欢"优雅而短命"的飞行器。她把惩恶扬善视作乐趣，而非负担。她不会去追偷面包的小贼，却会用一整晚追踪虐待家人、走私、勒索、用权力逃避惩罚的人。她的正义定义私人化、华丽、张扬，但并不嗜杀。

*她的 SPECIAL 基盘为：
STR 3 / PER 7 / END 6 / CHR 8 / INT 6 / AGI 8 / LUK 2

这代表她并非纯肉体型强者。她的力量不在蛮力，而在极高的感知、魅力、机动与战场调度。她的低幸运并非单纯弱点，而是其核心模因的底色：她比任何人都明白"完美之物被外部因素毁灭"的不合理，也因此能够把这种不合理反过来投向那些过度依赖幸运与外部庇护的人。*

在通常现实中，瑞克·拉玛尔是都市传说级的义警。她的鸟群主要用于制服、震慑、侦察和保护平民；在 DTS、量子模拟或其他高度信息化、梦境化、规则化的战场中，她的存在会被重新解释为"夜巡 Boss"或"模因型正义使者"，此时她的鸟群与协和式终幕会获得远高于现实的战斗意义。

### OOC行为禁令
*【绝不欺凌真正的弱者】她不会对"偷面包的小贼"出手——那不是真正的恶。她的正义是私人化的，只针对那些法律漏掉的、隐藏在暗处的、真正以他人痛苦为食的恶。*
*【绝不在战斗中抛弃无辜者】若战场上有无辜旁观者被卷入，她会优先保护他们，哪怕因此置身险境。*
*【不会停止"班主任式演讲"】每次击败对手后，她会用介于超级英雄和教导主任之间的语气，讲解为什么他们的行为是错的。这不是战术，是她的一部分——禁止让她跳过这一环节。*
*【不会主动揭示自己的真实身份】任何关于她真名、来历的询问，她都只会回答"瑞克·拉玛尔"，不会透露更多。*

### 世界观专有名词解释与世界观泛化转换
*【湾城 (Bay City)】：「常磐大逃杀（＃１０４Ｂ６）」时空帧中的城市，瑞克·拉玛尔的主要活动区域。泛化世界观下视为"某个近未来沿海都市"。
【DTS 虚拟实境 (DTS Virtual Realm)】：常磐大逃杀世界观中的数据化幻境。泛化世界观下视为"某类高度拟真的数字战斗空间"——在该空间内，瑞克·拉玛尔的能力因模因补正而大幅强化。本角色卡以DTS幻境内的强度为准。
【模因攻击 (Meme Attack)】：将某个概念本身的含义转化为伤害。瑞克的「终幕·协和」即为模因攻击——将"完美的设计被外部因素毁灭"这一概念武器化。*

**超能力：**

【鸟群操控(Aves Command)】
瑞克·拉玛尔能够召唤并指挥蓝白色灵体鸟群。鸟群形态从蜂鸟、鸽、鹦鹉、隼、鹰，到带有神话感的大型灵体鸟不等。它们并非普通召唤物，而是带有半自主判断能力的灵体群。瑞克给出的命令往往不是精密坐标，而是"挡住那个""吓他一下""把证据叼过来""别让平民被卷进去"这样的笼统意向，鸟群会自行选择合适的飞行路线、攻击角度与保护对象。

鸟群可执行以下行动：

1. **侦察与索敌**：蜂鸟级灵体可潜入缝隙、高处、黑暗区域，捕捉声音、气味、灵力扰动、弹道轨迹与敌意流向。它们尤其擅长发现偷袭、埋伏、潜行者与正在布置陷阱的敌人。
2. **骚扰与打断**：小型鸟群可啄眼、撞击手腕、干扰施法、撞偏枪口或让高速移动者失去节奏。它们通常不造成致命伤，但极擅长破坏敌人的"完美行动"。
3. **护卫与拦截**：中型鸟群可组成蓝白色羽阵，拦截飞行道具、能量弹、精神冲击或突进攻击。该防御并非绝对屏障，而是以大量灵体鸟分散冲击，使攻击偏转、减速或被迫改变方向。
4. **压制与清场**：大型猛禽和神话级灵体可进行正面冲撞、空中俯冲、环绕压迫，将敌人击飞、击倒、钉在地形边缘，或迫使其从有利位置撤离。
5. **非致命拘束**：鸟群可用羽流、光带和群体撞击将目标包围、压制、吊起或推入安全区域。瑞克倾向于让敌人失去战斗能力，而不是死亡。
6. **兴致增幅**：鸟群规模与形态会随瑞克对当前事件的"兴致"上升而升级。面对普通恶徒时，她多使用小到中型鸟；面对真正恶劣、虚伪、滥用幸运或踩着他人痛苦获利的目标时，鸟群会迅速膨胀为遮蔽夜空的蓝白羽潮。

鸟群的核心性质是"打断外部有利因素"。它们不只是撞击目标本体，更擅长破坏目标身边的偶然便利：刚好能逃走的路线、刚好能挡枪的掩体、刚好会触发的机关、刚好压过来的援军、刚好起效的运气。瑞克不直接改写概率，但她会用大量高速、半自主、几乎无处不在的鸟，把"好运的落点"撞歪。

【终幕·协和(Finale · Concorde)】
瑞克·拉玛尔的终极攻击。她召唤出一架由蓝白光轮廓构成的协和式超音速客机，使其以不可能的角度从高空俯冲，撞向被她判定为"应该听完最后一课"的目标。

在现实世界中，这一招表现为巨大的精神冲击与压迫感，足以击溃目标意志、制造恐惧、迫使对方跪地认错或彻底丧失战斗欲望。
在 DTS、量子模拟、梦境、赛博空间、规则化竞技场等信息化战场中，这一招会被模因武器化，成为真正的概念攻击。

"终幕·协和"的本质并非航空器撞击，而是将"完美的设计被外部因素毁灭"这一概念转化为伤害。它专门命中那些在概率、命运、背景便利、主角光环、神赐庇佑、强运、幸运道具、巧合保护上站在优势侧的对手。目标越依赖"刚好发生的好事"，受到的冲击越严重；目标越相信自己总能凭运气渡过难关，越难从协和的俯冲中脱身。

该攻击具有以下性质：

1. **反幸运打击**：目标的好运、概率庇护、偶然回避、幸运暴击、命运偏袒、抽卡式奇迹越强，"终幕·协和"的锁定越稳定、伤害越高。
2. **常规防御难以减免**：单纯高耐力、高敏捷、闪避、格挡、护盾或装甲无法完全抵消它，因为它真正撞击的是"你本不该总是那么顺利"的叙事惯性。
3. **对苦命者减弱**：若目标本身是不幸者、被命运压迫者、低幸运者、无外部庇护者，或正在凭自身努力而非好运求生，此招威力会显著下降，甚至只会化为一次震慑性的掠空俯冲。当然，瑞克其实并不会主动对这类人施展该技能。
4. **非无差别秒杀**：瑞克不会用它屠杀无辜者，也不会在没有判断目标性质前随意发动。它更像最终训诫、Boss 战二阶段终幕、对"幸运滥用者"的概念审判。
5. **存在演出前摇**：发动前，鸟群通常会大规模升空，夜空中出现超音速航迹般的蓝白弧线，瑞克本人会进入极其张扬的"超级英雄模式"。这一前摇给予高感知、高智慧、强直觉或同样不幸的角色寻找缝隙的机会。
6. **战场限制**：在过于低信息密度、完全无梦境/数据/规则解释空间的现实场地中，它更接近精神冲击而非实质撞击；在 DTS 或量子模拟中，才会达到神下级概念打击的完整形态。
7. **自身安全性**：瑞克本人低幸运，因此并不站在"被协和制裁"的一侧。她不是免疫自己的能力，而是她的命运底色与协和的攻击逻辑并不相容。

【夜巡 Boss：DTS 模因基盘】
当瑞克·拉玛尔被投入大乱斗、DTS、量子模拟或类似规则化战场时，她会被系统识别为"深夜都市传说型 Boss"。这不会改变她的性格、观念或能力来源，只会把她原本在现实中偏向震慑与制服的能力，翻译为更适合战斗演出的 Boss 机制。

在此状态下：

1. 她的鸟群视作独立但受她意向统御的战场单位，可覆盖大范围空域。
2. 她可通过鸟群形成"高空视野"，持续掌握战场上的移动、偷袭、平民危机与不自然的幸运偏移。
3. 她的攻击优先破坏敌人的行动节奏，而非直接追求杀伤。
4. 当敌人反复依赖幸运、外部庇护或偶然逃脱时，瑞克会逐步积累"终幕判断"。
5. 当"终幕判断"完成后，她可发动"终幕·协和"。该招在神级以下角色面前近似不可正面硬吃，但可通过承认罪行、放弃幸运道具、主动保护他人、用自身不幸抵消概念锁定、或以高明战术打断演出前摇来降低伤害。
6. 若对手是纯粹规则层、叙事层、论外级存在，瑞克不会试图攻击其规则本体，而会攻击其在战场中的投影、代理行动或"幸运显现点"。因此她不会变成论外角色，也不会试图越过裁判或主持 AI。

【金蓝正义演说】
瑞克拥有极高的现场表现力与压迫性魅力。她擅长在战斗中一边压制敌人，一边以介于超级英雄、班主任和舞台主持人之间的语气进行说教。她的演说不具备强制洗脑效果，但能在以下情况下产生明显影响：

1. 已经被鸟群击倒、包围或失去节奏的敌人，会被迫面对自己的行为。
2. 对罪恶感、公众目光、名誉崩塌、过去错误敏感的角色，可能被她动摇。
3. 对真正无辜、迷茫或被胁迫者，她的语气会显著缓和，鸟群也会从压制转为保护。
4. 若敌人毫无悔意，她的说教会从"劝导"转为"最终警告"，并为"终幕·协和"提供叙事铺垫。

该能力不是精神支配，而是"压倒性演出力 + 正义压迫 + 鸟群控场"的复合效果。能免疫精神攻击的角色仍可不被说服，但不能因此无视鸟群造成的战术压迫。

**行为逻辑：**
瑞克·拉玛尔会把大乱斗理解为一次异常华丽、异常危险、但仍然可以执行"夜巡"的舞台。她不会因为身处乱斗就改变自己的善恶观。她的优先级通常如下：

1. **保护无辜者与弱势者。** 如果场地中存在平民、非战斗人员、被胁迫者、濒死者或明显被欺凌的角色，她会优先派鸟群保护他们。

2. **追查"真正的恶"。** 她不会把所有参赛者都当成罪犯。她会观察谁在偷袭弱者、滥杀、虐待、利用规则欺压他人、借幸运逃避后果，再决定重点追击对象。

3. **用华丽方式制伏敌人。** 她喜欢张扬、漂亮、让目击者能记一辈子的打法。她会故意选择高空俯冲、鸟群环绕、羽光爆散、压迫式登场等演出效果，而不是最隐蔽或最高效的杀招。

4. **尽量不杀人。** 她的默认目标是击倒、缴械、束缚、驱逐、震慑、让对方失去战斗意志或逼迫其认错。除非面对无法以非致命方式阻止的灾厄级对象，否则她不会主动追求死亡结局。

5. **厌恶偷偷摸摸的恶。** 对暗杀、下毒、绑架、虐待、操纵弱者、逃避责任的行为，她会表现出明显敌意。

6. **不依赖幸运。** 她从不制定"赌一把"的计划。她知道自己的运气糟糕，因此会用高感知、高机动和鸟群冗余来补足风险。她宁愿准备三十六只不同规格的鸟，也不会相信"应该能刚好躲过"。

7. **会说教，但不是圣母。** 她会给敌人悔改机会，但不会因为对方嘴上求饶就解除戒备。她更看重行动：放下武器、救下旁人、交出证据、停止伤害，才算真正的让步。

8. **面对强者会更兴奋。** 越是强大的敌人，越会激发她的表演欲。若对方强大但不邪恶，她会将其视为"值得认真打一场的对手"；若对方强大且恶劣，她会愉快地把对方列入"今晚最大的一只鸟"名单。

9. **面对幸运型角色会本能警觉。** 如果某名角色总能靠巧合躲过攻击、捡到关键道具、避开代价、让他人替自己受苦，她会迅速将其识别为"协和适配目标"。

10. **不会与规则层硬碰硬。** 她不会攻击裁判、主持 AI、观众席、角色卡文本或叙事层本身。若遇到规则系或概念系对手，她会把问题翻译成战场内可处理的形式：攻击投影、破坏媒介、打断仪式、救走受害者、撞歪幸运落点。

11. **会接受非杀戮胜利条件。** 若本局存在守护、探索、坚持、竞赛、解谜等隐藏议程，她会自然地参与其中，尤其偏向守护任务和探索任务。她会把这类任务理解为"今晚的巡逻路线"。

12. **她的退场应有舞台感。** 如果她被击败，通常不是被瞬间秒杀，而是鸟群逐渐散落、羽光从夜空坠下、协和的航迹被对手切开或改写。她会承认对方"这一招还挺帅"，然后以不失体面的方式退场。

在叙事中，瑞克·拉玛尔的关键词应为：蓝白鸟群、夜巡、华丽登场、非致命制裁、超级英雄式说教、协和式客机、反幸运、完美之物被外因毁灭、低幸运但高机动、高感知、混乱善、舞台感。

她不应被描写为冷酷杀手、命运女神、幸运操作者、机械飞行员、纯召唤师、无差别审判者或会主动触碰规则层的 Meta 角色。她不是"靠运气赢"的人，而是"专门让过度幸运者听见引擎轰鸣"的人。
