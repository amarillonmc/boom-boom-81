「天轮圣王」的大轮部分。直径70000米，覆盖灵子世界的光轮。「事象选择树剪定机构」。
1.改写圣杯战争的规则，对Mooncell（《Fate/EXTRA》系列中的光子结晶超级计算机，主要行动场所，等效于小世界）进行入侵、控制、破坏。
2.催生出「死相」。「死相」是在大轮成立后，出现的极为罕见的强化现象。由于Mooncell遭到大轮入侵，灵子世界被剥离，各个阶层都出现了大量死者。而此时Mooncell处理死者残留情感的机能已被关闭，被称为「死相」的恶性情报由此诞生。在大轮的控制下，Mooncell逐渐崩坏，灵子世界开始沦为「死相」徘徊的场所。
3.干涉平行世界的运行，进行事象剪定。在TYPE-MOON的世界观中，地球文明、人类史在发展过程中会出现分歧，进而产生同时在这个宇宙中运行，但各方面都存在偏差的世界，即「平行世界」。由于宇宙本身的容量是有限的，它不会为了显而易见的结局使用能源，因此平行世界被划分为「编纂事象」与「剪定事象」。「编纂事象」是稳定且充满可能性，今后能发展出更多分歧的平行世界群；「剪定事象」则是由于过度偏离主要路线而变成特例，丧失大多数分歧的可能性，被判定为已经就此完成并完结，失去了继续观测其结局的必要性，即将被「采伐」的平行世界群。一旦某个平行世界被确认剪定，它的运行将会被停止，发展的可能性被断绝，世界中的一切存在都会被彻底消灭。大轮可以根据操纵者的指示，对平行世界执行事象剪定。在特维斯的「死相」控制下，经过近千年时间准备的大轮能够「停止全部平行世界的运行，断绝一切可能性」。
——小轮——
1.小轮是用于迎击敌人的宝具，搭载着大量炮门，如同全天候包围型的自动移动炮台那样，在覆盖范围内，发射能随意变形并且有追踪性能的密集光线牵制对手。小轮的炮门可以进行威力巨大的集中射击，这个破坏力需要高文真名解放并得到御主雷欧的膨大魔力及强化术式支援后，全力解放的宝具「转轮胜利之剑」才能与之抗衡。即使觉者已离去，留在炽天之槛上空的小轮仍能够进行一定程度的自律射击，但在这种模式下，射击精准度会大幅下降。
2.小轮的外部，装载着模仿「武之王·转轮圣王」所持的「七武具」而制成的巨大兵器。所谓转轮圣王，是指古印度吠陀时代传说的理想之王。由于觉者本人就是转轮圣王这个概念的源头，他拥有成为转轮圣王的可能性。这七件仿造品并非由魔力构成的光弹，而是纯粹的大质量兵器，如同「撕裂地表的导弹」、「印度式火箭」。一旦命中，还具有切断魔力以及将对手封印的特殊效果。
3.小轮由耐久度极高的不明物质构成，能够生成黑色的屏障阻隔访问Mooncell中枢的路径，并赋予「死相」近乎刀枪不入、连从者的宝具也能无视的不死性。只有施加强大的物理攻击，使小轮受到破坏，才能解除以上效果。
4.小轮的数量将随着战斗时间推进而不断增加，最终增加至七个。那个姿态会使人联想到彩虹，战斗时达到最高速，进而真名解放，发动最终攻击「归一转生」。光轮伴随「咔轰」的声音合为一体，高速回转，变化为「大日如来所背负的光晕」那样，以360度向敌人集结光柱，然后产生被认为是黑洞的从收束到大爆发的过程。

——「天轮宣告之时。一切众生、一切苦恼，皆回归于我。大悟之下，人类将在此成为一体。」
觉者的大宝具。
随着战斗时间（回合）的推进，「天轮圣王（小轮）」的数量增加到七个后，才能发动的宝具。
究极的「对个人宝具」。将匹敌人类创生之理的能量集中到一个敌人的身上，使之从痛苦中解放的最终解脱说法。
人类史越是延续，人类的版图越是扩大，威力就会继续提升。只要是有生之物就绝对无法违抗这个巨大的伤害。理论上，地球上没有人能够承受这个宝具的攻击。
若是根据印度的梵天思想，一切人类都是梵天的转世。也就是说，地球上所有的人类都是同一个人——在这样子的思想能够成立的场合，全人类成佛也是必然的，因为一切人类的究极都将会转生成为觉者（佛陀）。这就是将众生从苦恼中解救，「让一切人类皆回归于我」的轮回转生。
另一方面，这个宝具对非人者效果薄弱，也无法完全救赎存在规模超越人类版图之物。
「归一转生」的发动需要14回合的准备时间，与觉者的对战必须在（7+14）回合内结束。否则，「归一转生」一旦发动，就会直接判定败北并结束，是「绝对会导致GAME OVER的攻击」。
这个宝具的伤害值为五十六亿七千万。
至少归一1人后达成个人特殊胜利。

关键弱点/明确限制：

【特定的克星】："反·救世主（Nega-Savior）"是"兽（Beast）"所持有的特殊技能。这意味着在面对兽（Beast）时，觉者作为"救世主"职阶的天然优势可能会被逆转，甚至陷入不利境地。

【宝具的适用边界】：觉者的宝具"归一转生"虽然理论上能使地球上的任何人类强制解脱，但其效果也存在边界。对于不属于"人类"范畴的存在，例如游星来的外来者或某些规模超越人类史的"兽（Beast）"，其效果会变得薄弱。同时，如果目标本身已经没有生存的苦恼（例如已死的亡灵或某些特殊状态下的存在），这个宝具的强制解脱效果也可能无法完全发挥作用。

【超然的行动理念】：觉者早已超脱人类的烦恼，他的视角是宇宙尺度的。因此，他不会轻易直接介入人类的纷争。在他看来，人类的衰亡可能只是宇宙的一次"较大的波动"。他回应召唤，往往是出于对特定个体（如御主特维斯）的慈悲，而非认同其目标。这种近乎"旁观者"的立场，使得他很少会主动发挥全力去改变事态。

【防守倾向的战斗方式】：觉者所掌握的古印度武术"卡拉里帕亚特"，其精髓在于基于合理思想的防守，而非主动进攻。这反映了他的战斗风格可能更偏向于后发制人和绝对防御，而非侵略性的碾压，这在某些需要主动出击的战术局面下可能成为一种限制。

**行为逻辑：**1. 核心动机 / 胜利目标
觉者的终极目标并非通常意义上的"胜利"，而是 "完成救济" 或 "示现真理"。他回应召唤参与圣杯战争，目的不是为了实现御主的愿望或夺取圣杯，而是出于对持续受苦的特定个体（如其御主特维斯）的慈悲，以及为所有参与者提供一个领悟真理、从轮回之苦中解脱的机会。他的"胜利"是让众生理解"生老病死"之苦的本质，并自愿放下执念，获得解脱。如果无人能理解，他便会平静地执行"剪定"，将这个世界作为无解的错误答案予以消除。

2. 角色定位
主定位：

✅ 规则/概念干涉者

副定位 (可选)：

✅ 辅助/治疗者（这里的"治疗"并非治愈伤口，而是提供"精神上的解脱"，从根本上消除痛苦之源。）

定位分析：
觉者的能力核心完全超越了物理层面的攻防。其宝具"天轮圣王"能干涉乃至剪定平行世界，"归一转生"则是对"人类"这一概念的终极救济。他并不参与常规的战斗交换，而是直接作用于世界的规则和生命存在的意义。同时，他行使力量的出发点是一种极致的、非干涉性的慈悲，旨在为目标带来终极的"安宁"，因此带有强烈的"辅助/救济"色彩。

3. 行动倾向
被动反击型

分析：
觉者几乎永远不会主动发起攻击。他通常静观其变，如同一个固定的"机制"或"终点"，等待他人前来挑战。他的行动模式是基于他人的选择和行动而触发的"回应"。只有当对手执意攻击或无法被说服时，他才会出于"消除障碍"或"对其进行救济"的目的而动用力量。他的强大体现在绝对的防御（菩提树之悟）和后发先制的裁决上。

4. 其他补充逻辑
超然的视角：觉者视人类史的繁荣与衰败为宇宙中的"一次较大波动"，因此他不会主动介入拯救世界。这种宏观视角使他显得冷酷，但同时又对个体怀有深切的慈悲。

非杀生的胜利：他的最终手段"归一转生"并非毁灭对手，而是强制使其"悟道"解脱，肉体消散但灵魂进入轮回。这与其他从者以消灭对方为目的的战斗方式有本质区别。

唯一的"厌恶"：其设定中明确表示所恶之物是"毒蘑菇"，这源于释迦牟尼入灭的典故，是为数不多的能体现其"人性"残留的设定，但在战斗中几乎不构成弱点。

逻辑悖论：战胜觉者的唯一可能，在于成为其宝具"归一转生"的无效目标，即本身已非"人类"（如游星尖兵），或已超脱了生存之苦（状态特殊的存在），从而跳出其救济的逻辑链条。
