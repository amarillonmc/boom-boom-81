# 普通人的无聊生活的角色 - 无名剑神

## 1F

**角色填写人：**普通人的无聊生活

**基础信息：**

**角色名称：**无名剑神

**角色背景：**
一位来自低魔武侠世界的至高武者，其"剑道"已升华至触及宇宙规则本源。他并非依赖蛮力或法力，而是通过极致的"技"与"心"达到了"道"的境界。他游历万界并非为了征服，而是为了寻求一个能让他真正"败北"的对手，以验证道之无穷。在DND世界中，他是一位凌驾于大多数神祇和恶魔领主之上的、代表"武道终极"的叙事存在。然而，这份孤高与求败之心，是他作为"人"的最后枷锁。若有朝一日，出现了能真正威胁到"剑之道"本身的存在，或是某种概念从根本上否定了他所坚持的一切，这道枷锁便会粉碎。届时，无名剑神将不再是"人"，而是"道"本身。
**战斗能力：**
**核心能力·装备：**【一剑破万法 (One Sword Breaks All Laws)】：每日3次，他可以作为一个传奇动作，声明"破除"一个正在生效的、他能够理解的魔法效果、超自然能力或类法术能力。该效果立即终止，且24小时内无法被同一来源再次启动。
【止戈 (Cease War)】：每日1次，作为一个动作，他可以释放出他的"剑意领域"。范围内所有生物必须进行一次DC 30的感知豁免，失败则战意被彻底压制，无法进行任何攻击性动作，持续1分钟。此效果无视魔法免疫。
传奇动作 (Legendary Actions) (每回合5次)：
洞察 (Discernment)：自动感知一个生物的战斗模式与核心弱点。
剑气 (Sword Qi)：远程攻击，距离无限，造成力场伤害。
破招 (Break Technique)：中断一个正在准备或发动的动作/技能。

**被动特性·常驻效果：** 道之巅 (Pinnacle of the Dao)：无名剑神不受任何低于传奇级别的魔法效果影响。任何针对他的攻击检定需承受劣势。他可以对任何他理解的"技艺"或"法则"进行"破除"，视为自动成功。
【无剑胜有剑 (No Sword Surpasses the Sword)】：他不需要武器。他的徒手攻击被视为+5传奇魔法武器，具有"重击范围 15-20"和"瓦解"特性。他可以将任何环境元素（风、水、光、影）临时转化为拥有任意武器特性的"概念之剑"。
领域：环境变化到一座剑冢之中，无数残剑于剑冢中，无数把剑有无数种道，无数种道有无数把剑。所以规则如剑，剑如规则，那无名剑神就是剑之规则，剑之规则就是无名剑神。
【终末·万法归剑 (Finale · All Laws Return to the Sword)】 (特殊触发能力)：
触发条件：当无名剑神判定对手的"道"足以与自己的"道"进行最终对决时，或当他的"求败"之心被彻底满足（即，他真的在"技"的层面开始落败）时，此能力自动激活。
效果：他的人格意识将暂时消散，化身为纯粹的、绝对的**"剑之规则奇点"**。在此状态下：
他的一切行为不再有任何"限制"，包括"不伤人"的信条。
他将强制将整个世界观拉入一场以"剑"为唯一法则的终极叙事中。所有其他规则（魔法、科技、神性）要么被同化为"剑"的下位概念，要么被彻底从存在中抹除。
这场对决的败者，将不仅仅是"死亡"，而是其所代表的整个"道"与"叙事"被获胜方彻底吞噬和覆写。
**关键弱点/明确限制：**验证道之无穷，不屑于伤人性命。一旦【万法归剑】被触发，结果将是绝对的、不可逆转的。他自己也无法中止这个过程，直到其中一方的"道"被彻底抹消为止。这使得他在常态下会极力避免将战斗推向那个临界点。

**行为逻辑：**求战，每战全力以赴。求败，无败之孤高，无败之寂寞。
**核心动机：**验证道之无穷，验证剑之巅峰，求一败而不可得。触发【万法归剑】后作为"剑之道"的终极体现，吞噬或抹除一切对立的终极法则。
**角色定位：**正面战斗员　　
**行动倾向：**被动反击型
