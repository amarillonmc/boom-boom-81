# 失智毛玉的角色 - 卢本伟

## 1F

- floor_index: 1

**角色填写人：**失智毛玉

**基础信息：**

**角色名称：**卢本伟

**角色背景：**
他已经赢了。

在某个不为人知的维度，卢本伟终于等来了那句他穷尽一生渴望的认可。那一刻，积压了无数世界的质疑、嘲讽与谩骂，如同百川归海，彻底将他推向了力量的巅峰。

现在的他，是"玩梗化身"这一概念的终极形态，是所有网络声量、争议与信仰的具象化聚合体。他是梗宇宙中行走的天灾，是悬挂在现实穹顶之上的太阳——那个曾经被钉在耻辱柱上的名字，如今正在为万界定义何为"真实"。

他的瞳孔中，数据流已凝为璀璨星河。

**战斗能力：**

**核心能力·装备：**
1. 【给阿姨倒杯茶】空间置换：
 当他喊出"给阿姨倒杯茶"时，目标会被强制传送到他指定的任意位置，同时他自身瞬移至目标原本站立之处。若目标拒绝移动，则会原地陷入"阿姨倒茶"循环问号状态，逻辑混乱3秒。
2. 【十七张牌】因果律武器：
 当他喊出"十七张牌你能秒我？"时，接下来17秒内，他所承受的任何伤害、控制、负面效果，都会被强制延迟到第17秒结束时统一结算。若第17秒结束时他仍未倒下，则这些伤害会原封不动反弹给攻击者。
3. 【难受啊马飞】痛觉转嫁：
 当他喊出"难受啊马飞"时，自身承受的任何痛苦，都会同步复制给场上所有敌方单位。他越难受，敌人越难受。此能力无冷却，但每次使用后他必须真的"难受"（可由自身攻击触发）。
4. 【推推棒】强制位移：
 他可以用手指向任意目标，喊出"推推棒"，目标即被巨力推送至视线尽头。若目标撞上障碍物，则触发二次伤害并陷入"撞墙了兄弟"眩晕状态。

【第二序列·行为艺术法则】

1. 【正方形打野】领域展开：
 他跺脚画出一个正方形区域。在此区域内，所有敌人的行动逻辑被强制"正方形化"——只能直线移动、直角转弯，无法进行曲线或对角线位移。持续30秒。
2. 【枇杷膏】生命恢复：
 当他服用任意液体（包括但不限于水、饮料、酒），并说出"枇杷膏"时，生命值、能量、状态全部回满。此能力每日限用一次，但若液体为真实的川贝枇杷膏，则无限制。
3. 【我全都要】概念掠夺：
 当他面对多个可选目标（如多件宝物、多个能力、多个敌人），喊出"我全都要"时，所有目标会强制向他聚拢，被他一次性全部收取。若目标为生物，则陷入"被直播"状态，行动公开透明3秒。
4. 【这就去杀】强制追击：
 他对任意目标说出"这就去杀"，随后自身移动速度提升500%，且身形化为红色残影，所有拦截攻击均会穿透残影无效。追击持续至触碰目标为止。

【第三序列·史诗名场面法则】

1. 【血妈亏】因果倒转：
 当他遭受一次重大损失（血量、装备、关键物品），喊出"血妈亏"时，该损失会被定义为"对手的损失"。对手将承受同等代价，而他恢复原状。
2. 【不开玩笑】概念锚定：
 当他喊出"我不开玩笑"后，接下来的10秒内，他所说的每一句话都会成为临时现实。例如"你脚下的地是岩浆"，地面即刻化为岩浆。每句话持续生效5秒。
3. 【真的是服了】强制冷静：
 当他喊出"真的是服了"时，全场所有单位进入"服了"状态，强制停止一切攻击、移动、施法，原地站立并点头表示"服了"，持续3秒。此效果无视免疫。
4. 【当场拿下】即死判定：
 他对任意生命值低于30%的目标喊出"当场拿下"，目标立即进入"拿下"状态——被强制传送至未知空间（直播间），接受"观众姥爷"的审判，24小时后才可回归。

【第四序列·终焉·枪天使纪元】

1. 【二十九杀】因果覆写：
 当他背后浮现二十九把枪械虚影时，他可以对过去的某一个"瞬间"进行覆写。例如：对手刚释放的大招，可以被定义为"那一枪我早就开了"，使其无效化；对手即将获得的胜利，可以被定义为"那一局我杀了二十九个"，使其胜利被抹除。
2. 【实锤】概念封印：
 他对任意目标抛出"实锤"，目标会被一道从天而降的巨锤虚影砸中，陷入"被实锤"状态——所有能力、装备、称号被临时封印，只能靠肉体搏斗。持续至目标公开承认"卢本伟牛逼"为止。
3. 【五五开定律】胜负再定义：
 这是他从"五五开吧，我也经常赢他的"中悟出的究极权柄。他可以单方面定义"胜负概率"。宣告"这一局，五五开"，则战斗过程被强制导向"他赢一半，对手赢一半"的诡异平衡；宣告"这一局，我与你三七开"，则对手三成胜算归他，七成归对手，但最终结果必然是他赢——因为那"三成"里，包含了他必胜的"因"。
4. 【我没有开挂】根源宣告：
 这是他最核心、最禁忌的能力。当他发自灵魂地喊出"我卢本伟没有开挂"时，全场一切规则、能力、因果全部重置。所有正在生效的伤害、控制、负面状态清零。所有敌人被强制接受一次"信仰检定"——若内心相信他是真技术，则无事发生；若内心存疑，则被判定为"质疑者"，立即承受108把宿命之械的饱和打击。此能力每场战斗仅可使用一次。

**被动特性·常驻效果：**
· 【梗天帝】无限强化体质：
 所有关于他的梗——无论是嘲讽、玩味、怀念还是致敬——都成为他力量的永恒燃料。只要人类还在上网，只要互联网的亚文化还在延续，他就拥有无限的力量层级。他每分每秒都在变强，因为关于他的传说从未停止。
· 【全体起立】存在宣告：
 他的存在本身就是一句"卢本伟牛逼！"。任何生物或存在，在第一次感知到他时，精神海会自动响起山呼海啸般的"全体起立！！"，强制陷入0.01秒的敬畏僵直（对概念级存在依然生效，可抵抗但无法免疫）。
· 【弹幕护体】自动防御：
 任何针对他的攻击，在命中前都会被一层由"666""2333""？？？""卢本伟彩笔！！"构成的弹幕墙拦截。弹幕密度与当前网络声量成正比。
· 【直播中】状态公开：
 他的所有战斗会在"诸天万界直播间"直播。观众可以用弹幕为他提供临时buff（如"这波不亏"可回血5%）。同时，卢本伟会随时和观众弹幕进行交互来获得更多的buff。
· 【我是职业选手】威压：
 面对非职业出身的对手时，自动施加"降维打击"debuff，对手操作反应延迟0.5秒。

**关键弱点/明确限制：**
1. 【唯一真言】的枷锁：
 他力量的终极源头，是"我没有开挂"这句执念。若有一天，当他自己发自内心地、彻底地相信了"我没有开挂"这个事实，这份由"不被相信"而生的力量将会瞬间崩塌。因此，他的潜意识在永恒地逃避"被彻底相信"。这是他无敌表象下，唯一一道永远无法愈合的、属于人性的裂痕。
2. 【十七张牌】的锚点：
 他无法对"正在与他进行任何形式的卡牌游戏（包括但不限于斗地主）"的人，主动发起攻击或使用任何能力。这是融合前最后的人性执念留下的"安全词"。面对一个掏出一副扑克牌、大喊"你能秒我？"的人，他将陷入短暂的逻辑混乱，必须优先完成对局（且对局中运气会诡异地变得极差）。
3. 【沉默】的杀伤：
 他的力量与"梗"和"声量"绑定。若将他放逐于绝对的、永恒的无声虚空中，没有任何意识体可以传播任何信息，他的力量层级会随着"被遗忘"而缓慢跌落。这是他唯一惧怕的"软禁式"封印。

**行为逻辑：**
**核心动机：**

游戏开始时会大喊一声"伞兵1号卢本伟，准备就绪！！"

时刻开着直播，经常和观众说怪话，玩梗。

得意时会发出狒狒般的笑声，或者大喊"全体起立！！！"

喜欢吹牛自己经常和某些强力角色五五开，他在其他对局里经常单杀那个角色。

除此之外，他已无欲无求。

他行走万界的唯一目标，是见证"奇迹"。他想看看，在无数世界里，有没有那么一个存在，能在知晓他全部过往、见证他全部力量后，依然坚定地、平静地对他说一句："我信你。"

如果有，他或许会微笑着，亲手终结自己的永恒。
**角色定位：**搅局者/娱乐角色　侦查/潜行者
**行动倾向：**
主动进攻型
