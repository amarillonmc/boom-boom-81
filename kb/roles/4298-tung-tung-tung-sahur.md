# 失智毛玉的角色 - 木棍人（Tung Tung Tung Sahur）

**角色填写人：**失智毛玉

**基础信息：**

**角色名称：**木棍人（Tung Tung Tung Sahur）

**角色背景：**
2025年初从TikTok横空出世的AI脑腐顶流，外网播放量破740万的抽象派开山鼻祖。一根长了人脸的木棍，手里拎着棒球棍，嘴里永远循环着那句魔性印尼语："Tung Tung Tung Sahur～"名字源自斋月唤醒封斋饭的鼓声，但这跟严肃传统没半毛钱关系——木棍人的日常是跟耐克鲨鱼约会、在街头斗殴、被警察逮捕、被做成茶杯、甚至参加相亲节目，生活比所有人加起来还要丰富多彩。传入国内后被收编进"AI山海经"宇宙，从此开启了一段把所有人脑子都看坏的魔性旅程。

**战斗能力：**

**核心能力·装备：**
1. 三连魔性敲击（Tung Tung Tung！）： 用棒球棍敲击地面三下，发出灵魂三连音。被击中的敌人强制抖腿一秒，打断当前动作——这节奏上头到停不下来。第三次敲击命中时触发恐惧，对面会不由自主地想你接下来又要整什么活。
2. 终极叫醒服务（Sahur Time！）： 发出响彻全图的魔性呼唤。所有隐身、潜伏、装死、挂机的敌人全部强制揪出来，附加聚光灯特效和尴尬音效，让他们成为全场的笑话焦点。没人能在这个闹钟面前赖床。
3. 万物卡点战鼓（Drum Beat）： 敲击自己的木棍身体，产生的共振从精神层面造成伤害——对，被一根有自我意识的木头敲了的感觉就是这样的。范围内所有敌人的技能音效会被强换为"Tung Tung Tung"的魔性卡点，持续到他们开始怀疑自己到底在玩什么游戏。
4. 灵魂拷问回声（Echoing Call）： 敲击自己中空的木质身躯，发出超自然的回荡声波，所有隐身/潜行/藏在草丛里的敌人都被声呐定位并强制显形，同时施加恐惧——被一根木头用回声找到并吓到，确实是种很特别的体验。
5. 橡胶橡胶大木棍（Stretchy Arms）： 肢体可自由伸缩重组，手臂能像路飞一样甩出去老远，把人抽飞后瞬间弹回来，打完之后他自己还会嘀咕一句"我手刚才去哪了？"。
6. 耐克鲨鱼召唤（Bro, Assemble！）： 从迷因裂缝里召唤穿着耐克鞋的鲨鱼好兄弟。鲨鱼踩着太空步撞飞敌人，在原地留下旋转耐克鞋盒，踩到的单位当场滑倒并丢失一只鞋，移动速度降低——别问鞋去哪了，问就是被鲨鱼穿走了。
7. AI变形金刚模式（Mechanical Form）： 战斗白热化时可以化身巨型斑驳机械体，全身变成锈迹斑斑的工业机器形态，体型暴涨，攻击变成液压重锤，防御力拉满，跟茶壶人、仙人掌大象等AI山海经怪物展开荒诞的巨型对决。变身音效是一段极其劣质的机械运转声，让人不禁想问：他明明是木头变的为什么听起来像生锈的铁？

**被动特性·常驻效果：**
1. 抽象存在·物理免疫： 身为AI生成物，木棍人的画风跟所有人都不一样，各种物理规则对他只能算是"参考意见"。任何指向性技能打在他身上都有概率直接穿模无效化，同时他本人会发出类似卡带的声音。
2. 欢乐传播·全场特效替换： 只要木棍人在场，周围所有单位的严肃技能特效都会被替换成饱和度拉满的土味特效，死亡音效变为搞怪卡通音。在最酷炫的大招对决里听到一声"boing～"，整场战斗的气质就彻底木棍化了。
3. 耳虫病毒式传播： 在木棍人周围持续停留的敌人，脑中会被植入"Tung Tung Tung Sahur"的音频循环，自动释放技能时有概率喊出这句话代替原版语音，防不胜防的精神污染。
4. 碎片化重生： 木棍人被击杀后，身体炸成一地碎片。但只要有任意一块碎片被敌方踩到（基本都会踩到），他就会从鞋底钻出来重组复活——毕竟网络热梗是不会死的，梗在人在。
5. 午夜巡逻： 夜间获得额外的力量和潜行能力，木质身躯完美融入阴影，虽然他那张脸在黑暗中更吓人了，但这不是bug而是feature。
6. 脑腐光环： 木棍人在场时，敌我双方所有人物的智商和逻辑能力都会以肉眼可见的速度下降，严肃的战术讨论逐渐变成"哈哈哈你看他在敲地面"，战术博弈退化为纯粹的欢乐互殴。

**关键弱点/明确限制：**
1. 版权警告： 召唤耐克鲨鱼有极低概率触发"版权怪兽"的律师函空袭，导致鲨鱼反过来把他自己撞飞——AI合成物的版权灰色地带就是这么刺激。
2. AI传统艺能·画面Bug： 偶尔会出现严重画面BUG——手指变成六根、脸长到后脑勺、手从腿上长出来等。这些时候他会陷入"加载中..."的硬直状态，顺便把看到这一幕的敌人也看愣了。

**行为逻辑：**
**核心动机：**木棍人不是来战斗的，是来开派对的。他迈着完全不协调的步伐蹦进战场中央，先敲一遍"Tung Tung Tung"打断所有人的起手式，顺手用回声把藏在草里的刺客揪出来在聚光灯下公开处刑。然后手臂一甩——"橡胶橡胶大木棍！"，远程抽飞一个倒霉蛋。打嗨了直接开启变形金刚模式，全身零件咔咔一顿乱响，变成三米高的破铜烂铁机器人，跟对面的茶壶人和仙人掌大象激情互殴，场面荒诞到所有人都忘了自己本来在打什么。打到白热化，摇来耐克鲨鱼踩着鞋盒把对面阵型撞得七荤八素，全场都是滑倒的人和丢飞的鞋。他自己在混乱中疯狂卡Bug穿模躲伤害，偶尔因为画面崩坏原地加载挨一顿打，被击杀后碎片炸一地，等对面踩上去再从鞋底钻出来："想不到吧，我又回来了"。他存在的意义，就是把对面的电竞比赛，掰成一出谁也看不懂但谁也停不下来的抽象艺术大戏。
**角色定位：**搅局者/娱乐角色　正面战斗员
**行动倾向：**
主动进攻型
