# 玄羽的角色 - 终末之灭·天邪龙王

## 1F

- Author: 玄羽
- Posted at: 十月 27, 2025, 10:20 上午

**角色填写人：**玄羽

1. 基础信息
角色名称：终末之灭·天邪龙王/《赛尔号》
称号："难"之长老、灭世的丧钟、大暗黑天的基石
种族：邪灵·龙
核心概念：他是灭世的具象，亦是守护同僚的基石。其存在本身即为"终末"的概念之一，但他并非盲目的毁灭者，而是一位深谋远虑、顾全大局的战术家与统治者。在绝对的力量与压迫感之下，隐藏着对组织与同僚罕见的忠诚与"人情味"。

2. 外观与性格
- 外观：龙人霸主形态，体型魁梧威严，覆盖着暗色调的龙鳞与金属般的甲胄。龙瞳中燃烧着幽深的邪焰，周身弥漫着令人窒息的终末气息。当其降临，仿佛整个空间都在为绝灭的丧钟而震颤。
- 性格：沉稳、威严、深谋远虑。话语不多但字字千钧，习惯于掌控全局而非亲自冲锋。对待敌人冷酷无情，运用各种战术与控场手段将对手逼入绝境；但对待大暗黑天的同僚时，会展现出罕见的犹豫与包容，将组织的存续置于个人得失之上。

3. 背景故事
他是大暗黑天中资历最深的长老之一，代号"难"。在漫长的宇宙轮回中，他见证了无数文明的兴起与陨落，其本身便是终末的象征。然而，与纯粹追求毁灭的邪神不同，天邪龙王领悟到"毁灭"的真正意义在于"秩序的重塑"，而盲目的破坏只会带来无意义的虚无。因此，他加入了追求宇宙终极"天道"的大暗黑天，成为组织的绝对武力与战略基石。他并非嗜杀，每一次出手都经过深思熟虑，是为了更宏大的目标。他无法理解纯粹的混乱，也无法容忍背叛同僚的行为——这份源于灭世者内心的、矛盾的"羁绊"，构成了他最深层的弱点与人性锚点。

4. 核心能力/装备
【终末龙威】
他的存在本身会形成一个终末力场，持续对范围内所有敌人施加精神与物理上的双重压迫。
- 灭世宣言：宣告一个区域的"终结"，使该区域内的敌人持续受到灵魂灼烧与属性削弱。
- 邪龙霸临：进入霸体状态，免疫绝大多数控制效果，并将自身受到的部分伤害转化为自身的能量。

【邪灵龙咒】
通过高深莫测的龙语咒法，施加各种致命的控场与状态效果。
- 逆鳞之咒：极高先手度的诅咒攻击，命中后极大几率令对手陷入"诅咒"或"害怕"状态，彻底打乱其战斗节奏。
- 寂灭轮回：引爆终末之力，对全体敌人造成巨额固定伤害，并有几率附加"瘫痪"效果。
- 天邪审判：根据对手身上的异常状态数量，发动毁灭性的终极一击，状态越多，伤害越致命。

【大暗天长老之智】
作为大暗黑天的长老，他拥有无与伦比的战斗智慧与战略眼光。
- 战术预读：能在极短时间内洞悉对手的战斗模式与核心能力，并制定出最优的克制策略。
- 全局掌控：在团队作战中，他能自然成为战术核心，指挥并强化同僚，形成无懈可击的配合。

【终末之鳞】
其本体最坚硬的逆鳞，已与他自身的终末法则同化。既是无懈可击的防御，也是凝聚与释放邪灵龙咒的能量焦点。

5. 被动特性/常驻效果
- 灭世气魄：任何弱于他的存在，在直面他时会受到巨大的精神威慑，实力无法完全发挥。
- 邪灵龙躯：对多种属性的攻击有极强抗性，尤其是对光、圣灵等属性具有近乎绝对的压制力。
- 长老威仪：在大暗黑天阵营中，其命令拥有最高优先级，能极大提升阵营的凝聚力与战斗效率。

6. 关键弱点/明确限制
【人情味的负担】：
他将大暗黑天的同僚视为唯一的"自己人"。当同僚陷入危机时，他会因此产生"犹豫"，可能为了救援而放弃最优战术，甚至以身犯险。这是他最核心的情感弱点。

【控场依赖】：
他的战斗体系高度依赖"控场"（施加异常状态）。一旦对手拥有绝对免疫异常状态的能力，或者能频繁解除、反转异常状态，他的核心战术将被严重克制，战斗力大幅下降。

【属性克制】：
作为邪灵龙属性，他受到特定属性（如：自然系、水&暗影系等）的克制尤为明显。面对这些属性的顶尖强者时，他无法像往常一样从容，需要更加谨慎。

【谋定后动的惯性】：
过于沉稳的性格使他有时会错失一闪即逝的绝杀机会，倾向于选择"必胜"但耗时更长的战术，而非高风险高收益的激进选择。

7. 行为逻辑与OOC底线
核心动机：践行大暗黑天的"天道"，在毁灭中建立新的秩序，并绝对保障组织与同僚的存续与利益。
角色定位：控场型主C（主），阵营领袖/战略家（副）。
行动倾向：被动反击型或战略主动型。通常不会第一个出手，但一旦判定需要清除目标，便会以最稳健、最压迫的方式逐步掌控战局，将对手逼入绝境。
对话风格：威严、简洁、充满压迫感。常用"吾"、"尔等"等古语称谓。

【OOC底线】
1. 禁止无脑破坏：任何毁灭行为都必须有其战略目的或符合组织利益，而非出于发泄或愉悦。
2. 禁止背叛同僚：在任何情况下，他都不会主动出卖或牺牲大暗黑天的核心同僚。
3. 禁止鲁莽冒进：除非为了救援同僚，否则不应进行成功率极低的赌博式攻击。
4. 必须展现"谋略"：战斗描写必须突出其控场、预读和战术压制的能力，而非单纯的力量对轰。
