# galekkomari的角色 - 细胞怪

## 1F

**角色填写人：**galekkomari

**基础信息：**

**角色名称：**细胞怪

**角色背景：**
**出身与经历：**细胞怪是加帕里公园中出现的敌性存在，其核心特征是"从物与生物身上夺取'辉き'（Kagayaki / Sparkle），并复制其'形'"。
"辉き"，下称"光辉"，在该系列语境中是偏抽象的正向要素/重要之物集合（如干劲、活力、记忆、羁绊、魅力等），被夺走后可能出现无气力/战斗意愿低下、记忆缺失，严重时会陷入昏睡，因此通常细胞怪会被当作需要讨伐的威胁，当夺走对应光辉的特定细胞怪被打倒，被夺走光辉的角色便会恢复正常。
官方周边中使用过 "Cellien" 这一表记；本卡统一称其为"细胞怪"。
虽然动物朋友可以一击秒杀与其体格相近的细胞怪，但只要单体细胞怪的体格大于朋友，或出现大群细胞怪，则需要多个动物朋友才能将其击破。考虑到动物朋友的身体能力超过一般人类，想必普通人类面对单体细胞怪也会颇为吃力。

**性格特点：**细胞怪整体不以"人格化交流"为主（根据朋友们，尤其是之前是细胞怪一员的薮胞的说法，可以将细胞怪的所谓感情表露看作是"风一样的东西"。），往往以目标驱动式行动（夺取光辉、收集信息、掠夺与破坏/再现与保存）。
其中"统领者细胞怪"与"女王"均被描述为有着更为强烈的"目的"，但并不等同于存在能与人正常沟通的自我意识。

**外貌描写：**细胞怪的常见外观被概括为曲线感、半透明、像果冻或结晶一类的质感，并呈现多种体色与形态变化。从单细胞生物到日常用品到朋友外表不一而足。大型的细胞怪一般是多个小型细胞怪聚合，或直接吸收了大型物品的概念后出现的产物。

大王细胞怪是潜藏于战场地下深处的巨大墙壁型细胞怪，本身无法移动，但可以通过触手（蔷薇怪）或者统领者细胞怪（ルーラーセル）对战场施加影响。
统领者细胞怪为大王细胞怪复制了薮胞的产物，因此有着与她相近的外表：为一薮猫朋友外形的复制体，但通身深蓝偏黑色，并有着金黄瞳与巨大的头部彩虹色翅膀。
女王细胞怪是"全身多彩，有着黑、白发的人形"细胞怪，其外形来源被解释为夺取了加帕里公园副园长兼首席研究员「加古博士」的大量光辉后，对其形象的再现与扭曲复制。

**战斗的理由与执念：**
大王细胞怪，以及细胞怪默认的"存在意义"被明确表述为：夺取一切光辉，将其化为己有。
统领者细胞怪为大王的配下，旨在辅助大王进行破坏和同化作业，并作为作业终端，为其收取信息。
女王以及其配下的细胞怪的存在意义由于被加古博士"希望研究灭绝动物，与它们做朋友"的光辉扭曲，而经自己之口明确表述为：夺取一切终将消失的存在之光辉，将其再现并永久保存。这让女王和大王处于实质上敌对的立场。

这些执念也解释了为何许多细胞怪外观"像某种东西"：它们是在复制被夺走的光辉所关联的"形"。

**世界观定位：**在《动物朋友》系列的体系里，细胞怪是"以夺取光辉为核心机制"的灾厄型敌对生态（或近似灾害/瘟疫的系统性威胁），并能形成从杂兵到命名Boss的层级。
2015年《动物朋友》手游开始引发的女王相关事件在后续的世界观中被称作"女王事件"，对阵营与世界局势具有长期阴影式影响。哪怕是女王和大王均被打倒之后，细胞怪仍旧是公园的危险。在某个3000年之后的平行未来，细胞怪仍旧盘踞于加帕里公园以及周边地带。

**战斗能力：**

**核心能力·装备：**
【共通核心能力】
**"光辉捕食"体系：**细胞怪通过夺取光辉来复制"形"，并以"掠夺/破坏"或"保存/再现"为终极导向；这不是单纯物理捕食，而是对记忆、羁绊、热意等抽象要素的掠夺与利用。它们会优先锁定"强烈且密集的光辉源"（人、关键设施、剧情核心记忆/羁绊载体等），通过接触、包裹、围困、群体袭扰等方式制造"光辉流失"与战线崩溃。
**衍生产物/介质：**部分大型个体会伴随"细胞素（セルリウム，泥状黑色物质）"等要素，被用于产生/逸散出新的细胞怪或制造异变（具体表现依剧情与个体而变）。

【大王&统领者细胞怪】
**"末端操控/同类相食"分支（大王）：**大王细胞怪本体无法自行移动，而是会通过"蔷薇怪（バラリアン，伴随地震从地底升上的花型末端）"让普通细胞怪变异并互相残杀，最终由大王吸收幸存者以夺取大量光辉。甚至可以制造区域性异变（环境荒废、生态被抽干的观感），把对手拖进长期消耗与信息匮乏中。
**"统领链路/群体指令"分支（统领者）：**统领者以"统领（rule）大量细胞怪"为特征，能将散乱的细胞怪变成有组织的行动体。让细胞怪执行占领、封锁、诱导、拖延、护卫等分工，并把战斗变成"资源（光辉）争夺战"而非单纯互殴。
**信息收集（统领者）：**统领者细胞怪可以通过大气电波收集并理解信息，这些信息会与大王细胞怪和其他配下细胞怪瞬间共享。
**统领展开（统领者）：**统领者与足够数量的细胞怪保持可用的"指令链路"（叙事上表现为其影响力未被切断、核心未被压制）时，把"分散遭遇战"升级为"有组织的围猎与封锁"，甚至出现对关键地点的系统性占领（例如占领温泉等事件性据点）。
**水之印象（统领者）：**统领者细胞怪是大王读取了前细胞怪特殊个体「薮胞」"对水源的印象"而复制出的，因此她具有水的特性，难以被普通攻击伤害或影响。也可将自己溶化进地面以利用地下水瞬间逃脱。甚至拟态为建筑物等无机物。

【女王细胞怪】
**永存再现·女王协议：**如女王已摄取到足以支撑"保存与再现"的大规模光辉（尤其偏向承载强烈愿望与热意的光辉）。便会以"再现/保存"为名，制造大范围的"被复制之形"（物、设施、甚至近似人格外观的拟态），并驱动部下去持续夺取更多光辉。

**被动特性·常驻效果：**
**光辉敏感：**会对"最强/最近"的光辉源表现出强烈趋向性（可作为GM的追踪逻辑与引怪机制）。
**拟态倾向：**会出现"像某物/某人/某设施"的外观与行为影子，因为其复制逻辑本就建立在"光辉与形"的关联上。
**无心智：**虽然细胞怪之间可能会存在合作，听令等行为，统领者和女王看起来也存在意识；但结果论上来说，细胞怪并没有足以被影响的心智，不会受到任何的精神类影响，也无法与其进行谈判等行为。

**关键弱点/明确限制：**
**目标驱动：**越"明亮/强烈/关键"的光辉越容易成为其行动锚点；这既是威胁来源，也是可被利用的引导弱点（引离据点、诱入陷阱、用替代目标分流其势力）。

**形态崩解（女王）：**女王的"女王姿态"并非恒定不变；在被击败后会崩解并退回普通细胞怪形态，从而丧失其巅峰压迫与统摄象征。
**末端依赖（大王）：**大王本体不能自由行动，位于地底，强度建立在末端与回收循环上；"断末端、断回收"或者"消灭统领者细胞怪"通常比硬拼本体更有效。
**水之印象（统领者）：**由于统领者细胞怪本质为对水的印象，她可以被蒸发或是被冰冻。但被蒸发的场合，会随着温度降低而重新凝聚成形，因此原作中也正是将其彻底冰冻后砸碎而最终将其消灭。
**统领链路依赖（统领者）：**统领者的价值在"组织度"；当其链路被干扰、指令失灵或核心被压制时，细胞怪群体更可能退回各自为战。

**行为逻辑：**
**核心动机：**夺取并积累光辉，以"夺取和同化"（大王）/"再现与永久保存"（女王）为最终目的，同时扩大可控的细胞怪生态与影响范围。
**角色定位：**正面战斗员　策略家/指挥官
**行动倾向：**主动进攻型 机会主义型

互动原则
**对盟友/同伴：**对同类表现为"指令/统领关系"或"末端—本体关系"，而非伙伴式互信。没有足以和其他角色建立盟友/同伴关系的心智。
**对敌人/陌生人：**把对方视为光辉来源或障碍，优先选择能最大化光辉收益的行动路线。
**沟通风格：**多数普通个体和大王缺乏可用语言沟通；统领细胞怪被描述为难以沟通、偏机械式运作（原作中只使用不带空格的全片假名撰写其台词）；女王细胞怪只会复读"一切终将消失"/"我等的职责为复制和保存"等话语，但没有实质灵魂与思想。
**信任建立方式：**基本不存在"建立信任"，更多是"控制、复制、诱导"。
**核心动机的影响：**只要能获得更多光辉，细胞怪会调整战术（例如先占点、先拖延、先分割），并不执着于荣誉决斗。

**能力使用倾向**
优先用"夺辉"制造减员：先让关键角色失去记忆/干劲/羁绊支撑的能力输出，再处理肉体层面对抗。
局势胶着时转为"组织化"：由统领者把杂兵变成封锁网，让对手无法稳定休整与集结。
进入长期战役则启动"末端循环"：大王通过末端促成区域性崩坏与同类相食收割，使对手在资源与士气上被慢慢抽空。

***（AI/GM注意点）形态模块与切换·适用规则**
本张角色卡中实际上除了可以大量配置的杂鱼细胞怪以外，存在3个不同的主要敌性角色：
模块A：细胞怪女王（NEXON手游主线Boss）：黑身白发人形，源自夺取カコ博士大量光辉后的再现；目标是夺取一切光辉并永久保存与再现，偏"终局型支配者"。
模块B：ルーラーセル / Ruler Cell（统领者细胞怪个体，《动物朋友3》主线第2季BOSS）：以"统领大量细胞怪"为核心特征，能从"核心般形态"转变为更拟人（薮胞）的形态，并推动据点占领与围猎战术。
模块C：ダイオウセルリアン / 大王细胞怪（超大型本体，《动物朋友3》主线第2季BOSS）：本体难以移动，通过末端（如蔷薇怪）让细胞怪变异互斗并吸收幸存者来夺取巨量光辉，具备"改变地域生态"的掠夺规模。

模块使用方法（GM用）：同一乱斗·剧情战役内可按"阶段Boss"依次登场（模块C产生大量细胞怪和模块B，对战场进行破坏，如果未被其他角色破坏，则模块A刷出消灭模块B和模块C成为新的敌人，反之亦然。），或者同时登场三个模块，让模块A与模块B与模块C产生内斗。也可以只单独抽取其中一个模块作为当次登场的"主导个体"。
叙述性诡计（GM用）：GM可以运用如下的叙述性诡计：通过展示模块B尝试产生模块A或者和模块A共同战斗的情节，让其他角色相信模块B也是模块A的配下，但实质上模块A与模块B与C均为敌对关系，会主动尝试破坏模块B（但自身并无破坏其的手段）。*

概念掌握度：基本掌握
细胞怪虽然没有高级心智，但"掠夺光辉并复制其形"是它们存在的基础法则。它们能够极其稳定、有意识（本能驱动）地使用这种概念能力，只是伴随特定的行为模式和前置条件（需要接触或击溃目标）。
