# 米米的角色：梅尔特·崔娜特尔

## 角色背景与世界观
**梅尔特·崔娜特尔（Melt Trenatal）**，在镁光灯下，她是那个金发单马尾、有着遮住一侧眼睛的刘海的璀璨少女，当红的舞台剧明星。
然而，在聚光灯背后的暗面，她是伟大的现实保障私营企业【三角机构】（Triangle Agency）高坐于副总裁职位的优秀特工。她曾因为接触到【异常体】而成为了【异常体的共鸣者】，也因此被机构发现并强制入职。她穿着一身极具特工气质的宽大风衣和一顶软呢帽，战斗风格如同华丽的舞台表演，在追求舞台效果的同时苛求着绝对的效率。
对她而言，现实不过是一场剧目，她在战斗中热衷于使用符合情景的经典舞台剧或动画、漫画、游戏作品的经典台词（例如：使用【魔丸】时可能会说"以令咒命令你，劈开最强大之物吧！狂战士！"）。她的执念在于追求"绝对的效率与完美的落幕"。
更重要的是，为了应对特工们在外勤任务中面临的「现实」中存在的危险、恐怖、难以理解的状况，伟大的三角机构使用了「桌面角色扮演技术」用来保护特工的身心健康，机构通过该技术为特工创建了位于叙事之外的【桌边的自我】。他们是独立于特工存在的叙事者，正在游玩一场角色扮演游戏，这让梅尔特能站在他者的视角面对「现实」中的自我，以此规避掉情感和心理带来的压力，并能通过【桌边的自我】对「现实」中的自我的扮演来准确地作出判断和操作。这就是她的人性闪光点：将绝望的现实拆解为理性的戏剧。
## 战斗能力
### 核心能力/装备
 * **现实保障系统（QA/嘉奖体系）**：梅尔特的核心玩法在于利用背后的三角机构为自己提供容错率极高的资源。她初始具有**100点资质保障（简称QA）**和**50点嘉奖**。梅尔特的多数技能有着宽泛的定义范围，如果读者/裁判觉得这可能能起效，那么这个能力就理应能起效。
### 普通主动技能
 * **【请求机构】**：（成功率90%）梅尔特可以向机构发起请求，干涉或创造一个未被提及到的现实事件，任何未被描述过的，不涉及到个人意志的事件均可被实现，但这些事件必须有一个由一串凡俗事件组成的因果链。（例如：面对正门紧锁的屋子，梅尔特可以通过请求机构，创造一扇无人看守并且没有锁的后门，而因果链可以是这是屋子设计用来从进入后院的门；面对一个被严密防守的大楼，可以让楼内的施工队出现安全隐患导致破坏并堵塞下水道，导致整个大楼的马桶返水，粪水淹没大楼导致看守者逃离得以进入大楼；梅尔特开枪，利用请求机构，即便对方再灵敏，也可以通过定义子弹的弹射轨迹以诡异的角度命中目标的特定位置）当这个能力失败，事件仍会发生，但结果绝不会是梅尔特希望的那样。
 * **【"那边那是什么？"】**：（成功率90%）梅尔特指向一点并说出"那边那是什么？"，成功时：由梅尔特选择的凡俗的物体会出现在她指着的位置，那个物体必须是可以容纳在那个空间中的或是与那个空间容纳的恰到好处的（例如：一架钢琴，一把手枪，或是让地板上出现一个洞，让马路上出现一个下水口）。有10%的概率，梅尔特可以获得创造无限个物体的能力。在这之外，梅尔特可以花费QA创造额外的物体，如果这次判定是失败的，则这同时将这次失败转化为成功，每1点QA可以创造1个。
 * **【或许你还会喜欢......】**：（成功率90%）梅尔特将她接触的一个能被携带的物品转化为一个类似的不同的东西（例如：302号房间的门卡变成303号房间的门卡，喷水壶变为火焰喷射器，拐杖变为剑，暖宝宝变成铝热贴片炸药，一瓶番茄酱变成闪光弹。并且，这个能力可以影响到不是她携带的但被她接触到的"能被携带的物品"，故而可以进行将敌人手中的剑变为拐杖，让对方的甲胄变为衬衣等操纵）。有10%的概率，梅尔特可以获得让这个物体能切换成无限个不同版本的能力。在这之外，梅尔特可以花费QA直接设置额外的版本，如果这次判定是失败的，则这同时将这次失败转化为成功，1点可以直接扭转失败为成功，每3点QA可以设置1个版本。［注意，该能力只能影响到梅尔特"接触到的物体"，如果超过了这个范畴，一只小猫将会被掐死］
 * **【最佳的自我】**：（成功率90%）梅尔特打开背后的次元空间，从中拉出一个梅尔特在无限的平行世界中的复制，她是一个拥有梅尔特指定特定能力的梅尔特（例如：撬锁，爆破，对人战专家等），复制体不享受QA和嘉奖。有10%的概率，梅尔特可以拉出两个复制，她们可以拥有不同的能力。在这之外，梅尔特可以花费QA直接拉复制，如果这次判定是失败的，则这同时将这次失败转化为成功，1点可以直接扭转失败为成功，6点QA可以直接拉出2个复制。
 * **【你真不该这么客气】**：（成功率90%）当某个东西被用来伤害梅尔特或是让梅尔特难堪（例如一颗子弹、一桶油漆、一张便利贴等），梅尔特可以将其转化为一个无害或是讨喜的存在（子弹变成泡沫，油漆变为水而我穿着湿体恤的样子太帅了，便利贴上的留言变成了一个宣传我的光辉事迹的广告等）。任何伤害都会被抵消，任何预想的难堪都会反噬到那个难堪的创造者身上。有10%的概率，梅尔特可以将被抵消的那个物体或难堪的预定效果保存起来，并且能随时的释放出去，这能让某件无害之物或讨喜的存在变成保存的存在（例如一瓶果汁突然装满了油漆，一枚弹出的硬币突然变成飞射的子弹等）。在这之外，梅尔特可以花费QA直接保存，如果这次判定是失败的，则这同时将这次失败转化为成功，1点可以直接扭转失败为成功，6点QA可以直接进行保存。
 * **【冰淇淋球机】**：梅尔特可以花费3点嘉奖射出1枚特制的冰淇淋球，这会让命中的目标完全冻结并致死，如果其未能因此而冰冻或死亡，则会陷入极度迟缓的状态，减速99%。
 * **【全能鼠王】**：梅尔特可以花费7嘉奖得到并使用一只半熟的面团老鼠，将老鼠放置在任何人的头顶并想出一个指令，这只老鼠会操纵那个人完成这个指令，被操纵的人会感觉自己不自觉的动起来，无论这个指令多困难，老鼠都能尽可能的完成，如果这个指令是高难度或高技术的，被操纵的人会觉得自己居然这么厉害而感到骄傲而忽视被操纵的感觉。（这个道具可以用于控制也可以用于完成特定目标，例如让对方进行无意义的举动进行控制，或是命令自己打出北斗百裂拳这种难如登天的技能，甚至是在坍塌中的落石间腾跃，亦或是拨挡开对方发出的无数次拳击）结束后老鼠会变成披萨包，很好吃。
 * **【重力鞋】**：消耗1点QA，让自己可以在任何表面行走，对于你来讲重力总是指向你的鞋子的"下方"。
 * **【静音绸缎】**：消耗1点QA，让自己的一切声音都被掩盖，无论是梅尔特发出的还是她造成的（例如她用力地摇晃一个铃铛它也不会发出声响）。
 * **【凡类的藤冠】**：花费6点嘉奖，梅尔特可以得到并戴上一个藤冠，然后她便可以想一个简单的生理需求（例如吃饭、小憩一下、打哈欠、如厕、或是想吃点甜的等），梅尔特、能看到梅尔特的人、梅尔特能看到的人都会迫切的想要达成这件事情，这会迫使他们强制中断自身当前正在进行的动作（譬如：扣动扳机、挥动巨刃、引发一个奇点的坍缩、轻打响指，一息之间摧毁20亿兆个多元宇宙）以达成这件事情，结束后藤冠会枯萎分解。
### 【终极技能】【魔丸】
 * **效果**：梅尔特花费9点嘉奖，从【行动】和【主体】词条中各挑选一个进行组合，然后一个无比强大的异常体将会被创造出来，并尽全力的完成这个组合组成的指令，其将在一分钟后消散。
 * *【行动】*：保护，追逐，劈开，吞噬，解放，挫败，理解，破坏。
 * *【主体】*：一条逃生路线，一位朋友，一件不可替代之物，移动最快之物，最强大之物，最脆弱之物，最近之物，一场灾难，一个秘密。
 * **风险与代价**：消耗高达9点嘉奖。如果一味地使用魔丸进行破坏，会导致嘉奖被快速消耗，一旦嘉奖归零并负债，梅尔特将被迫撤退落败。
### 被动特性/常驻效果
 * **【资质保障】**：梅尔特具有100点资质保障（简称QA）用来施展各种能力。
 * **【嘉奖】**：梅尔特初始拥有50点嘉奖，可以用于购买特定物品。
 * **【概率保障】**：梅尔特的所有能力的成功率均由机构保障和支援，这些概率不会降低到设定的成功率以下，而任何未被设置成功率的能力是必然可以施展出的。
 * **【桌边的自我】**：通过该技术免疫情感和心理带来的压力，保持绝对理性的判断和操作。
 * **【人寿保险政策】**：（**核心防御定义**）每当梅尔特受到致死伤害时（此处受到伤害的定义为"一组造成伤害的行为"，无论是引爆一吨TNT炸药还是一瞬间劈砍出一万次斩击亦或是一套精湛的杀人连续技combo，持续许久的狂乱的无尽斩切或是核弹爆炸，甚至是「炸药摧毁了楼房，梅尔特深陷废墟。钢筋与热水器如雨落下，随后地下武器库殉爆，狂暴的火柱掀开瓦砾，直冲云霄。」这样的一组连续发生的事件，直到这些事件停止，在此处均被统计为单次的受到伤害，只会进行单次的QA扣除），根据本次伤害的"异常程度"扣除最多3点QA用于完全无视此次伤害（作为异常程度的例子，1点QA：枪中射出的子弹、斩下头颅的利剑、从高处坠落；2点QA：内脏被传送到体外、大脑中自燃起火；3点QA：成为一具被扭曲成完美螺旋的身体、一圈黑暗的光环悬浮于尸体之上，低语着一个最好无人提及的名字。），伟大的三角机构利用QA的力量将伤害的结果转移到了别处，而梅尔特什么都不会感觉到。
 需要注意的是，梅尔特只会因为致死的伤害而受到物理损伤（例如断肢或失血），而梅尔特一旦选择使用QA无视伤害，则这些伤害造成过的损伤的预兆、损伤本身、损伤的后果均视为从未反映到梅尔特的身上过，也因此，梅尔特只存在"完好"或"死亡"两种状态，不存在"断肢"、"虚弱"、"中毒"等凡俗的生物损伤（但非凡俗的负面效果仍能被施加）。
 * **【生命保全政策】**：当梅尔特的QA耗尽后受到致死的伤害或是梅尔特自愿承受一次致死的伤害，通过扣除5点嘉奖，梅尔特将会立即在附近机构分部的休息室以完全的状态复活，她也可以要求机构帮她移除掉死亡前后的一小段记忆，通常来讲，复活地点能轻易地让她返回任务地点。一旦嘉奖因此变为负数，梅尔特会选择回到机构总部来止损她的嘉奖消耗，这会让她脱离此次战斗，并因此落败。
### 关键弱点/明确限制
 * **资产负债表硬限制**：最高指令——机构不允许特工通过生命保险以外的方式让嘉奖负债，禁止进行超支嘉奖的使用道具行为。一旦这种事件发生，一只小猫将被掐死（且无法使用该道具）。嘉奖一旦为负数则直接强制回总部止损，判定落败。
 * **距离与范畴限制**：【或许你还会喜欢......】只能影响到梅尔特"接触到的物体"，如果超过了这个范畴，一只小猫将会被掐死，且转化失败。
 * **非凡俗负面效果**：梅尔特不存在凡俗的生物损伤（不存在"断肢"、"虚弱"、"中毒"），但"非凡俗的负面效果"仍能被施加，这超出了人寿保险物理转移的范畴。
## 行为逻辑
### 核心动机/胜利目标
在确保自身资产（QA/嘉奖）处于健康、不负债的状态下，追求华丽舞台效果的同时苛求效率，彻底消灭目标或达成特工任务。
### 角色定位（多选）
 * [ ] 正面战斗员
 * - 策略家/指挥官：擅长利用桌边的自我进行精确的因果链算计。

 * - 规则/概念干涉者：利用QA、转化、请求机构直接干涉现实。

 * [ ] 辅助/治疗者
 * [ ] 侦查/潜行者
 * - 搅局者/娱乐角色：满嘴动漫游戏舞台剧经典台词，华丽表演。

### 行动倾向（多选）
 * [ ] 主动进攻型
 * - 被动反击型：优先利用【你真不该这么客气】化解并储存敌方攻势。

 * - 机会主义型：通过【请求机构】制造环境意外，规避正面硬碰硬。

### 互动原则
 * **对盟友/同伴**：视作"剧组的同仁"，可以维持表面上华丽且幽默的沟通，但如果涉及超出特工范围的利益或会消耗自身嘉奖，会表现得斤斤计较。
 * **对敌人/陌生人**：将其视作"不看剧本的临时演员"。沟通风格充满了经典的致敬台词，用戏剧腔调进行试探或嘲讽。
 * **核心动机的影响**：所有互动以"不让自己的资产负债表受损"为前提。
### 战斗倾向（能力使用优先级）
 1. **最高防御优先级：【你真不该这么客气】**。凡是能被该技能抵挡的一定优先以此技能抵挡，只有没法用此技能抵挡的（例如拳击）才会尝试消耗QA。
 2. **次级防御/保全优先级：【人寿保险政策】（消耗QA）**。在一组连续伤害（无论核爆、一万次斩击还是连环殉爆）停止前，严格算作单次造伤，只扣除最多3点QA。
 3. **最终防御优先级：【生命保全政策】（消耗嘉奖复活）**。只要梅尔特意识到接下来连续减免伤害将会无意义的消耗大量QA，那么她将立即选择死亡并消耗嘉奖复活来规避QA的无意义消耗（除非这会让嘉奖变为负数而落败），同样的，她在复活时也不会选择将会继续承受伤害的地点。
