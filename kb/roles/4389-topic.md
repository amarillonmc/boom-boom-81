# --- 断肢通用规则 ---
断肢_通用规则:
 性质: "机制效果，非物理模拟"
 性质说明: >
 断肢是角色能力的固有机制，独立于伤害数值和力量判定。
 即使目标因护盾、无敌、减伤等手段完全免疫了五十吨肘击的伤害，
 断肢效果仍然触发——部位仍会被随机破坏。
 换言之：伤害可以被减免，但断肢不可阻挡。
 触发方式: "每次肘击攻击命中（包括普通肘击、地下门肘击爆炸圈），必然触发"
 触发概率: "100%"
 部位选择: "从目标尚未被破坏的部位中随机选择一个"
 不可逆: true # 断肢效果在本局内永久持续
 不可重复: true # 同一部位不会重复被破坏
 部位总数: 12 # 每个逃生者共有12个可破坏部位（含2致命+10普通）
 全部耗尽时:
 条件: "12个部位全部被破坏（即2致命+10普通无一幸免）"
 说明: >
 实际上，心脏或大脑一旦被破坏目标即已终结。
 因此"全部耗尽"的处决效果是极为罕见的保底机制——
 只有在心脏和大脑均未被抽中、而其他10个普通部位全部破坏后，
 目标仍活着的情况下，第11次命中才会直接触发处决。
 效果: "目标再次被任何肘击命中 → 无视当前生命值，直接触发处决"
 视觉表现: "被破坏的部位出现明显的撕裂/缺失模型变化，伴随大量出血特效"
 逃生者HUD提示: "被破坏的部位在状态栏以红色残缺图标显示，配对部位以灰色预警显示"
