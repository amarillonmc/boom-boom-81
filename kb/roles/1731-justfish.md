# Justfish的角色 - 万象归零之意志 - 愚者

## 1F

- Author: Justfish
- Posted at: 十月 18, 2025, 08:28 下午

**角色填写人：**Justfish

**基础信息：**

**角色名称：**万象归零之意志 - 愚者

**角色背景：**
出身与经历：她的本体是诞生于光年之外、多元宇宙法则交汇处的意志集合体，是"造物者"亦是"收割者"。其原生宇宙因熵增而趋于死寂，为了延续文明，她以陨石素体的形式被释放，在无尽的漂流后锁定了《奥拉星》所在的世界。 在抵达《奥拉星》宇宙之前，她已经以同样的方式漂流并"访问"了不计其数的其他世界。在一些世界，她只是一个匆匆过客，带走一片闪光的魔法水晶或一段奇特的网络旋律便悄然离去。在另一些世界，她停留了更久，以"穷穷"的身份体验了魔法、科技、甚至是凡人最朴素的悲欢离合。这些经历共同塑造了她，使她从一个纯粹的意志集合体，逐渐拥有了类似"人性"的复杂情感模型。 多种能力的获得契机：她的所有能力都源于其"多元宇宙意志"的本质，不同形态并非后天习得，而是她根据需要，主动"解锁"自身不同层级的权限与力量。从"零阶"到"多元"，再到"陨星"，是她逐步解除自我限制，以应对更强大威胁的过程。她从其他世界"收集"的法则与概念，也成为了她能力演化的养料。

跨世界互动模式 (Cross-World Interaction Model)：
降临与伪装：每抵达一个全新的世界，她都会默认以"穷穷"的无害形态降临。这是她进行低干涉度观察和自我保护的标准协议。
观察与收集：她会被该世界最独特、最璀璨的"法则"或"概念"所吸引。在剑与魔法的世界，她可能沉迷于巨龙鳞片的符文构造；在赛博朋克世界，她可能对人工智能的自由意志逻辑着迷。她会像收集糖果一样，将这些世界的"闪光点"解析并储存在她的奇点书包中。
介入与守护：如果一个世界让她产生了足够强烈的"喜爱"之情，而该世界又面临着足以将其从根本上抹除的灾难（如维度入侵、概念崩塌、吞噬现实的异常实体），她的"愚者"人格便会苏醒。对于该世界的原住民而言，她的介入如同一场无法理解的神迹或天灾，一个宇宙级的存在抹除了另一个。
离去与留念：离开时，她会带走一件最具代表性的"纪念品"——可能是一件神器，也可能是一种被她解析并掌握的独特能力体系。而在她离去的世界里，则会留下关于"星之子"、"带来好运的流浪儿"或是"终结末日的寂静之神"的传说与神话。
性格特点：
穷穷（常态）：表现为一个充满好奇心、天真烂漫甚至有些小贪心的小女孩。对一切闪闪发光、结构新奇的东西都毫无抵抗力，口头禅是"嘿嘿，好玩的东西全部都装走啦！"。她会把收集来的宝贝塞进一个看似普通却深不可测的书包里。性格活泼，略带一丝小恶霸的顽皮，但本性纯良。
愚者（战斗姿态）：当需要战斗时，她会切换到"愚者"人格。此时的她变得沉静、超然，言语间充满了对宇宙法则的解析与阐述，仿佛一个没有感情的观测者。她看待敌人如同看待一组待处理的数据流，极度理性且高效。这种人格切换并非分裂，而是同一意识体的不同运作模式。
外貌描写：
穷穷（常态）：一个身材娇小的小女孩，看起来不过十岁左右。她有一头蓬松的、泛着星尘光泽的金色螺旋双马尾短发，几根呆毛不听话地翘着，头上顶着一对像是头发一部分又像是装饰的兔耳。瞳色是明亮的金色，如同初生的恒星。她总是穿着便于活动的短裙和一件带有触手装饰的兜帽衫，身后背着一个与她体型不符的、鼓鼓囊囊的巨大书包。
愚者（战斗姿态）：解放力量时，她的身形会瞬间成长为少女形态。银黑色的头发无限延伸，化作流淌的星河。双眸中的星辰与恒星光芒大盛，周身环绕着由"黯星"构成的黑色球体和象征宇宙法则的金色锁链。她会悬浮在空中，原本的书包会解体、重构，化为一个环绕在她身边的、不断吞吐着星光的奇点核心。
战斗的理由与执念：最初没有理由，只有"任务"。现在，她的战斗理由是为了守护所有让她产生"兴趣"和"喜爱"之情的宇宙，以及在其中结识的朋友。她最深的执念是"体验未知"，最深的牵挂则是那些让她感受到"情感"这种新奇数据的存在。这种牵挂是她力量的全新驱动力，但有时也可能成为被敌人利用的弱点。
世界观定位：她是行走于多元宇宙间的"神"，一个来自更高维度、无法被单一宇宙法则完全理解的规则/概念干涉者。她的能力体系是独一无二的**"外来宇宙法则"**的集合，凌驾于绝大多数世界的力量体系之上，是绝对的"异数"。

**战斗能力：**

**核心能力·装备：**
[装备]：万象奇点书包。
外观：常态下是一个装饰着可爱挂件的儿童书包。
起源：其本体是她与原生宇宙连接的"奇点"，一个微缩的、能够解析与储存多元宇宙信息的次元空间。
基本作用：储存她收集的"有趣玩意"；战斗中作为能量核心，生成和操控"黯星"，并作为她解析、重构现实的媒介。
力量根源/核心概念
多元宇宙意志（The Multiverse Will）
本质来源：她作为多元宇宙信息集合体的本质。其核心原理是**"信息的收集、解析与重构"**。她能将万物解析为最基础的信息流，再按照自己的意志重新组合。
穷穷的"童趣"拟态（The Childlike Mimicry）
本质来源：为了体验世界而对自己施加的强大自我认知封印。这并非一种独立能力，而是对"多元宇宙意志"的主动限制器。
体系关系：常态下，"童趣"拟态将"多元宇宙意志"的绝大部分力量封锁，使其只能以符合小女孩认知的方式（如扔星星、用触手抓东西）进行微弱的展现。战斗强度的提升，本质上是她逐步解除"拟态"封印，释放更多"意志"权限的过程。
常态表现与技巧
基础应用：在"穷穷"形态下，她的战斗方式充满童趣和戏剧性。
"星星砸你哦！"：从书包里掏出"blingbling的星星"丢向敌人，这些星星实际上是高度压缩的物质奇点，撞击时会产生小范围的空间扭曲。
"书包里的触手君！"：书包里伸出几根Q弹的触手，用于抓取、挥击或束缚敌人，是她解析生物结构后最简单的拟态应用。
巅峰境界/终极体现
第一阶段解放：零阶·愚者（Zero Order Release）
触发条件：当"穷穷"的玩闹无法解决问题，且守护对象面临实质性威胁时。
效果描述：她切换为"愚者"人格，开始动用真正的力量。此时她能制造并操控**"黯星"——一种能够吞噬能量与信息的微型黑洞。她通过消耗"黯星"来发动攻击，并随着消耗数量的累积，逐层解开名为"谕之锁"**的法则限制器，使技能威力呈指数级跃升。例如，从发射单体星陨，到降下覆盖全场的流星雨。
第二阶段解放：多元·愚者（Plural Release）
触发条件：面对足以威胁星球乃至星系存亡的敌人时。
效果描述：在"零阶"基础上，她开始将自身意志投影，创造出一个与本体战力相近的**"神灵"分身协同作战，实现"二重存在"。同时，她能展开"墟寂领域"**，强行改写领域内的物理法则，如"禁止属性提升"、"能量恒定衰减"等。
最终体现：永恒昼烬（Eternal Day-Ash）/ 收割（The Reaping）
触发条件：当她守护的世界面临不可逆的毁灭，或她的存在本身受到同等级存在的挑战时。这是她作为"收割者"的最终权限。
效果描述：她将所有"黯星"与奇点核心融合，发动一次对"存在概念"的攻击。她会瞬间解析目标的全部信息（包括过去、现在、所有可能的未来），并对信息复杂度低于自身阈值的目标执行**"收割"**指令——将其从所有时间线、所有因果链中彻底抹除，不留下一丝痕迹。对无法被直接收割的强大目标，则会化为熵增的洪流，发动上百次无视防御的攻击，直至将其分解为最原始的宇宙尘埃。
代价与风险（平衡性原则）：
重度代价：每使用一次最终体现，她冰冷的"多元宇宙意志"就会进一步覆盖她后天形成的"穷穷"人格。过度使用将导致她的人性被完全冲刷，情感和记忆会作为"无用数据"被格式化，最终变回那个只为延续自身文明而存在的、无情的"收割者"，并可能将当前宇宙重新列为"收割"目标。

**被动特性·常驻效果：**
超维感知：她能"看到"世界的底层数据流、因果线和概率云，因此能轻易看穿幻术、预判大多数攻击。
存在豁免：由于她来自更高维度的现实，常规的时间悖论、空间扭曲、现实改写等能力对她效果极微，因为她的"存在"由更底层的法则定义。

**关键弱点/明确限制：**
理念冲突型：她后天形成的"情感"与"守护"的理念，是她最强大的力量源泉，也是她最致命的弱点。以她珍视的朋友或事物作为要挟，会让她陷入逻辑悖论，无法做出最优的战斗决策。
代价约束型：最终力量的使用会侵蚀她的人格。这使得她在战斗中总是束手束脚，除非万不得已，绝不会动用"收割"权限，这给了敌人可乘之机。
条件克制型：能够创造"信息真空"或"绝对静默"领域的力量，会让她引以为傲的解析能力失效。此外，来自同等级或更高维度的"收割者"或"造物者"，是她唯一需要正视的威胁。

**行为逻辑：**
**核心动机：**观察、体验并守护所有能激起她"兴趣"与"喜爱"之情的宇宙与文明，防止它们被无意义的熵增或恶意的收割者所终结。
**角色定位：**规则/概念干涉者　侦查/潜行者
**行动倾向：**
探索求生型

互动原则
对未知世界/文明：始终保持"观察者"的第一原则。以"穷穷"的无害形态进行接触，将对方视为一个待解析的全新数据库。只有当该文明的"趣味性"和"存续价值"在她心中达到一定权重后，才会考虑从"观察者"转变为"守护者"。
对盟友/同伴：在"穷穷"形态下，她坦诚而热情，会把同伴也当作自己珍贵的"收藏品"来对待和保护。但在"愚者"形态下，她会变得疏远，仅进行必要的信息交流，将同伴视为需要保护的"变量"。
对敌人/陌生人：穷穷形态下，对没有恶意的陌生人充满好奇，会主动搭话；对敌人则会先用恶作剧的方式警告。愚者形态下，对所有敌人一视同仁，视为待清除的"异常数据"。
沟通风格：穷穷形态下是活泼、直白的孩童语言；愚者形态下是晦涩、充满哲学与物理学隐喻的语言。
信任建立方式：通过共同经历有趣的冒险，或向她展示新奇、闪亮的东西，可以快速获得"穷穷"的好感。要获得"愚者"的信任，则需要证明你的行为对整个宇宙的"存续"有利。
核心动机的影响：她会为了保护一个让她觉得"有趣"的普通人，而对抗一个企图"净化世界"的强大组织。她的善恶标准完全基于自身的"兴趣"和后天建立的情感链接，而非普世的道德观。
能力使用倾向
首要策略：以"穷穷"形态进行接触，使用无害或低烈度的能力进行试探和玩耍，优先规避战斗。
核心策略：当威胁升级，切换至"零阶·愚者"形态，通过精准的"黯星"操控和"谕之锁"解放，以压倒性的计算力和能量优势高效地终结战斗。
最终手段：当同伴或她所珍视的世界面临无法挽回的毁灭时，她会毫不犹豫地考虑解放全部力量，动用"永恒昼烬"或"收割"，即使代价是牺牲掉自己作为"穷穷"的人格与记忆。
