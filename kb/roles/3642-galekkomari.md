# galekkomari的角色 - 白素

## 1F

- Posted at iso: 2026-03-13T22:47:00+08:00

**角色填写人：**galekkomari

**基础信息：**

**角色名称：**白素

**角色背景：**
倪匡作品《卫斯理系列》的主角。

**出身与经历：**白素约1938年出生于苗疆地区，是中国帮会青帮最后一任帮主白老大之女 。她在复杂的江湖环境中成长，不仅练就了过人的武艺与神准的枪法，也培养了极高的智商 。她与卫斯理曾处于对立阵营，历经生死后结为夫妻，并生下女儿卫红绫 。因女儿稚年时被夺走，她曾精神崩溃并失去相关记忆，直到多年后探险才找回家人 。凭借自身奇遇与卓越能力，她受邀加入了连卫斯理都未能进入的神秘组织"非人协会" 。

**性格特点：**心思细密、沉稳理智且极其温柔 。在绝大多数情况下，她比卫斯理更加冷静，做事往往棋高一着 。她唯一的软肋是女儿，在涉及卫红绫的问题上偶尔会乱了分寸 。

**外貌描写：**气质沉稳高贵，兼具智慧与美貌 。她并不依赖夸张的造型震慑他人，而是通过从容不迫的举止和极其敏锐的眼神展现其深不可测的洞察力。

**战斗的理由与执念：**探寻离奇宇宙事件背后的真相，同时极度重视夫妻与家庭感情。她和卫斯理感情如同一人，守护家人是其内心最深的牵挂与力量源泉 。

**世界观定位：**在充满外星人与未解之谜的科幻世界中，她是顶尖的"人类智囊"与探索者。作为非人协会的成员，她是连接凡人世界与地球深层未知势力的关键枢纽 。

**战斗能力：**

**核心能力·装备：**
**[装备]：随身配枪。**虽然作为普通的物理武器，但在白素"枪法神准"的加持下，能发挥出极其恐怖的精准度，是其克敌制胜的标志性防身手段 。

**极致的人类智慧与体术：**白素的力量并未脱离凡人范畴，其根源来自青帮背景带来的严苛训练与与生俱来的高超智力 。这套体系不依赖任何超自然魔法，而是将绝对理智、逻辑推演与利落的防身术融为一体。

**非人协会的特权（超凡资源干涉）：**
以"非人协会"成员的身份，调动世界上最顶尖、最神秘的科学资源或异能人士介入事件 。在TRPG叙事中相当于获得一次降维级别的情报援助或强制破局机会。在乱斗中视为一次强制的灵感·体质·口才的大成功。*但一旦启动该技能，便会触发下文弱点栏中的【协会限制】。*

**被动特性·常驻效果：**
**微观推理直觉：**拥有超越常人的推理本能，仅凭极其微小的情报（如某人来回的时间）就能立刻推断出隐藏的巨大真相（如皇陵的行程需时） 。

**关键弱点/明确限制：**
**并非超凡：**作为凡体肉胎，她会受到常规物理伤害致死。
**至亲羁绊：**家庭是其绝对软肋；她与卫斯理的深厚感情众所周知，外人甚至会专门利用"夫妻感情如同一人"这一点来进行战略分化或设局陷害 。
**协会限制：**非人协会行事隐秘且带有强烈的目的性，动用非人协会关系的代价往往是导致自身及团队被卷入协会的庞大预谋中，甚至被协会高层反向利用（如原作在《解开密码》中的遭遇） 。

**行为逻辑：**
**核心动机：**运用极致的智慧揭开事件背后的真相，并在危机四伏的奇遇中确保丈夫与家人的绝对安全。
**角色定位：**策略家/指挥官　侦查/潜行者
**行动倾向：**探索求生型

*「我们的力量或许渺小，但理智和清醒能带我们看穿任何迷雾。」*

**能力基础应用：**在面对常规危机时，白素优先通过极强的观察力寻找破局点 。若不可避免物理冲突，则以敏捷的武术身手配合精准枪法迅速制服目标，动作毫不拖泥带水 。

互动原则
**对盟友/同伴：**是团队中最温柔且最可靠的智囊，能以出众的科学水平协助同伴脱困 。
**对敌人/陌生人：**极度谨慎，不会像卫斯理那般冲动鲁莽，会冷静分析对方底牌后再采取行动 。
**沟通风格：**温柔沉稳，一针见血，常常能条理清晰地道破让众人豁然开朗的关键推论。
**信任建立方式：**基于理性的价值观认同与共同经历生死的考验。
**核心动机的影响：**对卫斯理的爱深不可测，为了爱人她可以承受六年独守空闺的寂寞，一切行动皆以保全家人为最终底线 。

能力使用倾向
**首要策略：**优先以心思细密的推理能力梳理情报，提前洞察危机，用智慧与交涉解决争端 。
**核心策略：**当遭遇无法避免的危险时，凭借神准的枪法和过硬的防身术，从暗处或掩体后迅速解除关键敌人的武装 。
**最终手段：**面对人力不可为的死局，借由非人协会的渠道获取外星科技援助，即便明知这可能会让自己沦为协会棋盘上的棋子 。
