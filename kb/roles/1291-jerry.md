# Jerry的角色 - 人头气球

## 1F

- Author: Jerry
- Posted at: 十月 05, 2025, 06:40 上午

**角色填写人：**Jerry

**角色名称：**人头气球

**基本属性：**- **出身与经历**：没有人知道"人头气球"现象从何而来。它并非魔法，也非科技，更像是一种来自未知维度的、充满恶意的宇宙法则的具象化。这一现象的当前锚点，是前偶像**藤野辉美**。某日，她的上吊自杀的狂热粉丝的尸体被发现后，一个与她一模一样的"人头气球"便凭空出现，并开始对她展开无休止的追杀。从此，辉美从万众瞩目的舞台坠入无尽的逃亡深渊。
- **性格特点**：作为规则锚点的藤野辉美，其精神已在长期的追杀和恐惧中被消磨殆尽。她极度神经质、敏感、脆弱且自私，为了活下去可以毫不犹豫地将灾祸引向他人。她会本能地向强者求助，但又因恐惧而无法完全信任任何人。
- **外貌描写**：藤野辉美是一个外表憔悴的年轻女性，曾经的光彩荡然无存，取而代之的是深深的黑眼圈和紧张的神情。她总是穿着便于逃跑的朴素衣物。时刻跟随着她的**零号气球**，则是她作为偶像时期最光鲜亮丽的头颅的放大版，面带诡异而僵硬的微笑，下方垂着一根粗长的绳圈，如同索命的绞索。
- **行动的理由与执念**：藤野辉美的唯一执念就是**活下去**。而"人头气球"这一现象的内在驱动力，则是无法被理解的、纯粹的**增殖与毁灭**的欲望。
- **世界观定位**：一个显现于物质世界的、**现实结构中的一个不可名状的底层BUG**。它不属于任何已知的力量体系，是一种蛮横地将自身规则强加于现实世界的禁忌现象。它并非一个可以被'驱散'或'抹除'的附加诅咒，而是 **'存在'这一概念本身固有的、被激活的缺陷**。对于场上所有角色而言，藤野辉美的存在本身，就是一个随时会引爆的、威胁所有人的定时炸弹。

**超能力：**
## 核心机制：绝望悬空之日
[这是该角色的灵魂，是其一切行动的中心。]

- **机制本质与最终目标**：
 - **核心概念**：一种具有传染性的、强制性的"镜像猎杀"规则。
 - **最终目标**：当机制完成时，场景中所有符合条件的"本体"都将被其对应的"人头气球"吊死，最终形成一幅万千头颅在空中静默飘浮，下方悬挂着无数尸体的末日景象。

- **机制进程与阶段性效果（事件链）**：
 - **[阶段一：原初追猎（启动）]**
 - **触发条件**：游戏开局即启动。藤野辉美与零号气球同时在场。
 - **阶段效果**：零号气球会以杀死藤野辉美为唯一目标，无视其他一切角色和环境。这会形成一个移动的、极不稳定的"事件中心"，迫使其他角色不得不对这对诡异的追猎者与猎物做出反应。整个场景都笼罩在一种山雨欲来的紧张氛围之下。当零号气球被摧毁后，藤野辉美也会一并死亡。
 - **[阶段二：全面爆发（蔓延）]**
 - **触发条件**：**藤野辉美死亡**，或**零号气球被彻底摧毁**（爆裂或完全漏气）。
 - **机制描述**：
 1. **气球生成**：所有在场的、满足以下两个条件的角色，都会在自己头顶上方的空中生成一个与自己头颅一模一样的人头气球：
 - 拥有物理实体。
 - 拥有通常生物学意义上的"头颅"构造。
 2. **生命链接（规则级）**：气球与本体之间建立绝对的伤害共享链接。
 - 气球受到的任何损伤，都会**100%**同步反映在对应角色**本人**的头颅上。
 - 当气球被彻底破坏（如爆裂、完全漏气），对应角色的头颅也会瞬间爆裂。此效果被视为**真实伤害**，无视大部分防御和护盾，直接作用于角色的生存根基。
 - 如果有角色选择动用规则或者概念攻击抹杀气球，则对应的本体也会一并被同样的规则打击直接抹除。
 3. **猎杀本能**：
 - 每个气球都拥有一个与生俱来的、唯一的行动目标：**找到自己的"本体"，并用气球下悬挂的绳圈将其吊死。**
 - 气球会无视任何非本体的其他角色，不会对其进行任何形式的攻击或互动。
 - 气球下方的绳圈拥有异常的韧性和力量，一旦套住目标，极难挣脱。
 4. **自毁倾向**：
 - 当气球因障碍、束缚等原因长时间无法接近其本体时，它会主动尝试自我毁灭（如撞向尖锐物），以通过生命链接来杀死本体。

- **机制推进方式（角色的"主动技能"）**：
 - 藤野辉美的**逃亡与求助**本身就是机制的推进器。她的移动会带动零号气球，将混乱和选择难题带到战场的每一个角落。
 - 当被逼入绝境时，辉美可能会主动冲向人群或坚固的掩体，试图利用他人或环境来对抗零号气球，这会极大地增加零号气球被摧毁的风险，从而加速"全面爆发"的到来。

- **机制干涉与逆转手段（角色的"弱点"）**：
 - **根源性弱点**：只要**藤野辉美存活**且**零号气球完好无损**，第二阶段"全面爆发"就永远不会被触发。保护这个看似无用的女孩，是阻止灾难降临的唯一方法。
 - **规则豁免**：该机制对**无实体**（如幽灵、能量体）或**无常规头颅**（如机器人、元素生物）的角色完全无效。这些角色是天然的"免疫者"。
 - **非绝对致死**：气球的淘汰方式（窒息或头颅爆裂）并非绝对的规则抹杀。拥有超强再生能力、不死之身、或能摆脱物理伤害致死判定的角色，即使头颅爆裂也可能存活。
 - **物理限制**：气球本身是物理实体，可以被**困住、封印、转移、放逐**。只要能阻止气球接触到本体，就能暂时保证安全。

### 自身能力
- **藤野辉美**：无任何战斗能力，仅拥有普通人的体能和求生本能。
- **人头气球（所有）**：拥有基本的飞行能力和识别"本体"的诡异感知力。

### 关键弱点
- **锚点脆弱性**：藤野辉美是一个没有任何自保能力的普通人，极易在交火余波中意外死亡，从而导致灾难失控。

**行为逻辑：****核心动机/胜利目标**
 - **藤野辉美**：不惜一切代价活下去。
 - **人头气球（现象）**：触发"全面爆发"，并淘汰所有符合条件的角色。
**行为逻辑**：
 - **藤野辉美**会歇斯底里地向任何看起来强大或友善的角色求救，并用关于气球的可怕情报作为交换。她会极力夸大威胁，试图将所有人都绑上自己的战车。在被追杀时，她会毫不犹豫地将其他角色当作"肉盾"。
 - **人头气球**是沉默的执行者，不与任何单位进行除猎杀之外的互动。它们是纯粹的规则化身。作为一种现象，它没有胜利目标。其存在的唯一意义就是执行"猎杀并吊死本体"的程序，直到所有气球或所有对应的本体从战场上消失。
**能力使用倾向/行动优先级（人头气球）**
1. **首要策略**：不惜一切代价接近本体，尝试用绳圈进行绞杀。
2. **次要策略**：若被障碍物阻挡，会尝试绕过或寻找路径突破。
3. **最终手段**：若长时间被困或无法接触到本体，将立刻转向执行自杀程序，优先选择能最快摧毁自身的方式。
