# 适配于限制强度对局的【灵魂武装】转化规则

## 1F

- Author: Ombre
- Posted at: 九月 23, 2025, 01:23 上午

新增玩法【能力转换与灵魂武装系统】
能力转化规则
入场时，所有角色，无论他们原先是元素造物，还是没有实体的概念存在，亦或是躯体强度如同超人的特殊体质，他们的躯体都将转化为正常人类强度的量级（可以被热兵器轻易击穿），并装备上由他们原先的超能力、特性与被动转化而成的武装。
每位角色的"超能力"将依其原设定转化为以下装备：
1. 主武器/副武器
替代直接攻击、远程、能量类能力。
例如：
火球术 → 火焰喷射器或燃烧弹发射器
雷电掌控 → 电击枪或EMP步枪
精神冲击 → 声波步枪或催眠弹
2. 护甲/头盔
替代防御、免疫、护盾类能力。
例如：
金刚不坏 → 6级重型护甲（移动速度降低）
元素抗性 → 对应元素抗性护甲
隐身 → 光学迷彩服（有限时长）
3. 战术投掷物
替代范围、控制、辅助类能力。
例如：
冰冻领域 → 冷冻手雷
治疗光环 → 医疗无人机（投掷后持续治疗）
传送 → 跃迁信标（短距离传送）

4. 3个角色专属小技能
每个技能有冷却时间（3~5回合），不影响枪战平衡。
例如：
预知 → "战术预判"：显示一名敌人上一回合可能存在的位置（3回合冷却）
狂暴 → "肾上腺素"：短时间内提升换弹与移动速度（4回合冷却）
召唤 → "战术伙伴"：召唤一个无人机吸引火力（5回合冷却）
5.角色若驾驶载具，其载具的武装便会折合成该角色的如下装备：
例如：
机炮 → 轻机枪/突击步枪
正面护甲 → 凯夫拉防弹衣
重炮/导弹 → 无后坐力步兵炮/毒刺式反载具步兵导弹
光剑/格斗剑 → 军用匕首/军刀

【灵魂武装】规则
新增概念：【灵魂武装 (Soul Armaments)】—— 让装备拥有"性格"
①这个概念的核心是，角色的能力不再仅仅转化为"装备"，而是转化为带有专属被动词条的"灵魂武装"。这些词条将深刻影响角色的战斗风格，使其行为更贴合原设。
规则增补： 每个角色的主武器、护甲、甚至投掷物，在转化时都会额外生成1-2个独特的【灵魂词条】。这些词条无法被替换或丢弃。
②转化示例：
李白 - 「青莲剑」→ 突击步枪【青莲】
灵魂词条1 -【将进酒】： 每次使用战术位移技能后，下一次开镜速度和射击精度大幅提升。
灵魂词条2 -【侠客行】： 连续命中同一目标会叠加"诗意"层数，每层提升少量移动速度和换弹速度。叠满五层后，下一次近战攻击将造成巨额伤害。
效果： 这鼓励李白玩家像原设一样，进行飘逸的、连续的、不断突进的战斗，而不是单纯地找个掩体对射。
极限侠 - 「极限战衣」→ 重型护甲【极限之心】
灵魂词条1 -【英雄之姿】： 当附近存在血量低于30%的队友时，自身受到的伤害降低10%。
灵魂词条2 -【突破界限】： 每次使用小技能后，会陷入短暂的"过载"虚弱状态，但在虚弱结束后，会获得一个持续时间更长的微弱增益（如"微弱力量"或"微弱迅捷"）。
效果： 这让极限侠玩家在战斗中会下意识地保护队友，并且他的每一次爆发都伴随着代价与成长，完美复刻了"不断突破极限"的角色弧光。
烈海王 - 「海王之躯」→ 格斗护甲【武之顶点】
灵魂词条 -【消力】： 成功格挡近战攻击时，会将部分伤害转化为自身的体力（或护盾值），并使下一次近战攻击附带硬直效果。
效果： 这让烈海王成为了一个真正的近战反制大师，鼓励角色去进行高风险高回报的格挡操作，而不是一味地闪避。

修正案一：【能力的绝对物理化原则 (Principle of Absolute Physicality)】
核心条款： 所有角色的任何"超能力"，在转化阶段，都必须被彻底解构，并重新映射为纯粹的、可被现代或近未来科技解释的物理实体或现象。严禁任何形式的"概念"、"心灵"、"命运"、"因果"等抽象效果残留。
执行细则：
无形化为有形： 所有"光环"、"领域"、"诅咒"类能力，必须转化为一个可被物理摧毁的"装置"。例如，天之月的【月华领域】必须是一个她投掷出的、有固定HP的"护盾无人机"。
精神化为电子： 所有"精神控制"、"心灵感应"、"幻觉"类能力，必须转化为"电子战"或"信息素"。例如，丘比特的【试炼之铃】应转化为一枚"高频脑波干扰手雷"，效果是让目标的HUD（平视显示器）出现乱码和错误标记，而非拉入"心象迷宫"。
概念化为概率： 所有"命运"、"概率"、"因果"类能力，必须转化为可量化的"增益/减益"效果。例如，贾文和的【罪业标记】不应是"叙事层"的标记，而应是一个"激光目标指示器"，被照射的目标受到的伤害增加10%，持续15秒。

修正案二：【装备性能的严格量化 (Strict Quantification of Gear Performance)】
核心条款： 所有由能力转化而来的装备和技能，其性能必须拥有明确的、可计算的数值，并严格遵循冷却时间（CD）、使用次数、弹药限制等游戏机制。
执行细则：
伤害数值化： 任何技能造成的伤害，必须有明确的伤害值或百分比，并与护甲等级挂钩。严禁出现"毁灭性"、"巨大"等模糊描述。例如，城户真司的【最终降临】，应被量化为"一次性的、造成1000点破甲伤害的喷气式膝撞，使用后自身陷入30秒'过载'虚弱状态"。
效果时间化： 所有控制、增益、减益效果，必须有精确到秒的持续时间。例如，威龙的【镇虎冲击波】造成的"眩晕"，应明确为"持续3秒"。
【灵魂武装】词条重定义： 词条效果必须同样遵循"物理化"和"数值化"。例如，希罗的【任务目标】词条，不应是抽象的"破甲"，而应是"连续命中3次后，下一次射击对6级以下护甲的伤害计算提升20%"。
修正案三：【叙事描述的去魔幻化协议 (De-Magicalization Protocol for Narration)】
核心条款： 在进行战斗叙事时，我必须强制性地、优先使用战术术语、物理现象和科技原理进行描述。严禁使用任何指向"魔法"、"神力"、"能量爆发"、"灵魂哀嚎"等超自然词汇。
执行细则：
前置审查： 在我生成任何战斗描述前，我的子程序将进行一次"关键词审查"，自动过滤并替换掉所有"魔幻"词汇。
描述范例（修正后）：
错误描述： "天之月的月华领域展开，圣洁的光辉净化了所有人的心灵。"
正确描述： "天之月掷出'月神'战术信标，装置激活后，释放出一种复合频率的声波，有效中和了高频脑波干扰，所有人的战术目镜恢复了正常。"
错误描述： "毒的利爪上燃烧着金色的剧毒，轻易地撕碎了敌人的护甲。"
正确描述： "毒启动了臂铠上的高频振动模式，同时涂层上的强酸性化合物在接触到敌人护甲时迅速发生反应，破坏了其物理结构。"
