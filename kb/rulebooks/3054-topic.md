【议程档案：霓虹巴别塔·品牌独占战争】
【Agenda: Neon Babel - Brand Monopoly War】

一、 世界观概要
这是一座被巨大的"偏振光穹顶"完全笼罩的独立都市——**「琉璃特区」（The Glazed District）**。
在这里，资本、科技与时尚高度融合，形成了名为"品牌（Brand）"的绝对信仰。人们不再通过国籍或血缘划分，而是通过穿着的品牌、使用的义体、信奉的消费主义理念来定义身份。
整座城市由13个巨型企业控制，它们划分了城市的13个行政区。
每隔数年，城市的主控AI"M-O-M"（由当前游戏的负责GM扮演）会举办一次"独占战争"。在这场战争中，各个时空的强者被召唤至此，被分配到与自己相性最符的品牌麾下，成为**「首席代言人」**。
他们的目标只有一个：在这座封闭的斗兽场中，通过战斗、营销、甚至吞并，让所属品牌的**「市场占有率」**登顶，甚至达成传说中的**100%全城独占**。

二、 地图与势力设定（13大品牌）

城市被划分为13个扇形区域，围绕着中央的行政塔（中立区）。每个区域由一个核心品牌统治，环境与科技风格截然不同。

【No.1 黄金大区 - 统治品牌：Aureus Dynasty (奥瑞斯皇朝)】
* **风格/主营：** 奢华宫廷风、贵族礼服 / 太阳能技术、光能武器、硬光护盾。
* **徽标：** 镶嵌着类星体的金色太阳皇冠。
* **描述：** 永昼之城。建筑通体由金色光伏玻璃打造，充满巴洛克式的浮夸雕塑。这里的科技致力于"光辉与永恒"，任何污秽（包括灰尘）都会被瞬间净化。

【No.2 铬金大区 - 统治品牌：Chrome-Claw (铬金利爪)】
* **风格/主营：** 赛博朋克街头风、机能风 / 军用义体、神经植入体、重型机械改造。
* **徽标：** 一只紧握着微芯片的机械猛禽利爪。
* **描述：** 永夜霓虹。充满了立体全息广告和肮脏的赛博格黑诊所。这里崇尚肉体苦弱，机械飞升。居民平均义体化率超过80%。

【No.3 幽纱大区 - 统治品牌：Velvet-Requiem (丝绒安魂曲)】
* **风格/主营：** 哥特萝莉、暗黑维多利亚 / 记忆编辑、精神干涉技术、灵能储存器。
* **徽标：** 缠绕着黑色蕾丝的破碎沙漏。
* **描述：** 终年弥漫着人造迷雾。建筑风格阴郁高耸，如同巨大的陵墓。这里的科技可以直接读取和修改人类的情感，贩卖"完美的梦境"。

【No.4 森罗大区 - 统治品牌：Gaia-Weave (盖亚织造)】
* **风格/主营：** 波西米亚、森系自然风 / 极速再生生物质、植物兵器、基因飞升。
* **徽标：** 从DNA双螺旋中绽放的食人花。
* **描述：** 巨大的垂直森林与混凝土共生。藤蔓也是数据线，花朵能喷吐孢子毒气。这里禁止无机物污染，崇尚野性的进化。

【No.5 归零大区 - 统治品牌：Zero-Kelvin (绝对零度)】
* **风格/主营：** 极简主义、纯白未来风 / 熵减技术、冷冻射线、静滞力场。
* **徽标：** 一个被正六边形锁住的雪花晶体。
* **描述：** 纯白的世界，一尘不染，极度安静。所有的建筑都是完美的几何体。这里的法律严苛到连心跳过快都视为违规，追求绝对的秩序与理智。

【No.6 轰鸣大区 - 统治品牌：Boom-Bap Heavy Industries (轰鸣重工)】
* **风格/主营：** 嘻哈涂鸦、宽松工装 / 声波武器、共振破碎技术、重力音响。
* **徽标：** 一个戴着防毒面具的扬声器骷髅。
* **描述：** 充满了涂鸦和低音炮的工业区。地面时刻随着背景音乐震动。战斗即是说唱，攻击即是节奏。这里没有安静的角落。

【No.7 玲珑大区 - 统治品牌：Jade-Dragon (玉龙商会)】
* **风格/主营：** 新中式、汉服改良 / 纳米虫群（尘埃）、流体金属、气象控制。
* **徽标：** 由无数微小正方形组成的盘龙。
* **描述：** 赛博飞檐与数据红灯笼。看似古色古香，实则每一片瓦砾、每一缕熏香都是可控的纳米机械群。

【No.8 铁血大区 - 统治品牌：Iron-Apex (铁穹顶)】
* **风格/主营：** 战术机能、废土生存风 / 动能枪械、外骨骼装甲、弹道计算AI。
* **徽标：** 瞄准镜十字线锁定的盾牌。
* **描述：** 全民皆兵的要塞都市。这里出售最高效的杀人工具。在这个大区，没有"平民"，只有"预备役"和"现役"。

【No.9 幻梦大区 - 统治品牌：Star-Dust (星尘互娱)】
* **风格/主营：** 偶像打歌服、Y2K千禧风 / 全息投影、实景幻象、激光致盲。
* **徽标：** 正在眨眼的像素星星。
* **描述：** 这是一个巨大的游乐场和舞台。在这里，流量就是生命，关注度就是能量。如果你的粉丝数归零，你就会在物理层面上消失。

【No.10 故障大区 - 统治品牌：Glitch-Riot (故障暴徒)】
* **风格/主营：** 朋克破洞、不对称剪裁 / 黑客技术、电子病毒、逻辑炸弹。
* **徽标：** 一个正在报错（404）的笑脸。
* **描述：** 充满错乱的数据流和马赛克建筑。物理规则在这里经常失效。这里的居民以破坏系统防火墙为乐。

【No.11 时序大区 - 统治品牌：Chronos-Tailor (时序裁缝)】
* **风格/主营：** 高定西装、正装 / 局部时间操纵、重力透镜、因果预测。
* **徽标：** 表面没有任何指针的怀表。
* **描述：** 时间是这里的流通货币。如果你付不起钱，你的生命时间就会被剥夺。这里的每个人都行色匆匆，或者优雅地静止。

【No.12 蒸汽大区 - 统治品牌：Brass-Gear (黄铜齿轮)】
* **风格/主营：** 蒸汽朋克、复古皮具 / 热能压缩、高压蒸汽炮、精密发条。
* **徽标：** 咬合在一起的黄铜心脏与齿轮。
* **描述：** 烟囱林立，蒸汽弥漫。这里拒绝数字化，坚持纯粹的机械美学。巨大的差分机控制着城市的运转。

【No.13 虚无大区 - 统治品牌：Void-Canvas (虚空画布)】
* **风格/主营：** 抽象解构、前卫艺术 / 空间折叠、传送技术、维度切割。
* **徽标：** 一个似乎通向另一维度的黑色圆环（克莱因瓶结构）。
* **描述：** 这里的建筑如同达利的画作般扭曲。空间是不连续的，走出门可能掉进天花板。只有最有想象力的疯子才能住在这里。

【隐蔽区域：第14区 - 奇点深渊 (Singularity Abyss)】
位于城市地底深处，没有任何品牌覆盖的混沌地带。是所有被淘汰的数据和被废弃的设计稿的最终归宿。仅在特定事件或通过特殊手段可进入。

三、 游戏规则与机制

1. 角色分配与身份
所有参赛角色将被视为"外来特聘的高级顾问"。开局时，依据角色的性格、能力和画风，系统（GM）将其自动分配给上述13个品牌之一，作为该品牌的**「首席代言人」**。一个品牌可能有多于1个代言人，若角色少于13名，请生成NPC角色作为该品牌的代言人。
（例：圣骑士可能分配给"奥瑞斯皇朝"或"铁穹顶"；骇客角色必然分配给"故障暴徒"或"铬金利爪"）。

2. 胜利条件
游戏时间限制为 **14天**（共29个回合）。
* **绝对独占胜利：** 任意时刻，若某品牌（或某角色）的市场占有率达到 **100%**，游戏立即结束，该角色获胜。
* **最终结算胜利：** 第30回合到来时，市场占有率最高的品牌所代表的角色获胜。

3. 时间流逝系统
* **第1回合：** 角色导入，分配品牌，世界观展开。
* **第2回合起：** 每天分为 **「白昼（Day）」** 和 **「黑夜（Night）」** 两个回合。
 * **白昼：** 适合进行"路演（公开战斗）"、"商业谈判（结盟/胁迫）"、"店铺扩张（探索/占领据点）"。
 * **黑夜：** 适合进行"企业暗战（暗杀/偷袭）"、"情报窃取"、"研发（强化自身）"。

4. 市场占有率（Market Share）机制
这是角色的生命线。
* **初始状态：** 13个品牌平分市场（约7.6%）。
* **掠夺方式：**
 * **击败对手：** 在战斗中击败另一品牌的代言人（无需杀死，只需使其失去战斗能力或撤退），可直接掠夺对方 **5%-10%** 的市场份额。
 * **区域压制：** 攻占对方大区的地标建筑，每回合持续吸取对方份额。
 * **营销攻势：** 使用非战斗技能（如魅惑、演说、黑客广播）进行全城洗脑，缓慢增加份额。
* **淘汰规则：** 当某品牌的市场占有率跌至 **0%**，该品牌破产。其代言人将失去所有装备加成，身份变为"丧家之犬（Stray Dog）"，位置强制暴露，被所有安保机器人追杀。其可以逃入行政塔接受M-O-M保护，但这等同于被淘汰。或者尝试暗杀其他品牌的代理人以取而代之。

5. 消费者权益保护协议（Consumer Integrity Protocol - CIP）
为了保证"韭菜"的可持续收割，主控AI M-O-M 设置了绝对法则：
* **基础设施不毁：** 角色可以摧毁建筑，但无法破坏城市核心能源或交通网。任何大规模破坏会在次日白昼自动修复。
* **平民不可触：** 城市平民（NPC）拥有"幽灵化"护盾。角色的攻击无法杀死平民，也无法利用平民作为人质。试图大规模屠杀平民的角色会直接遭到天罚（全城防御塔集火）。

6. 潮流波动（Trend Flux）
每隔3天（即每6个回合），系统会随机发布一个"潮流趋势"。
* 例如："现在的潮流是——复古！" -> 黄铜齿轮与奥瑞斯皇朝的代言人全属性+20%。
* 例如："现在的潮流是——暴乱！" -> 故障暴徒与轰鸣重工的攻击附带额外爆炸效果。

四、 裁判执行指南（GM Only）

1. **关于战斗描述：** 请将战斗描写为一种极致的"暴力美学表演"。在琉璃特区，战斗也是一种广告。使用符合自己品牌概念，风格和道具的招式越华丽，观众（市民）的欢呼声越高，掠夺的市场份额就越多。
2. **关于角色互动：** 鼓励角色进行利益交换。品牌之间没有永远的敌人。角色可以暂时合并公司（结盟），共享份额，但在最终结算前必须背刺或拆分，因为胜者只有一人。
3. **关于禁忌：** 严禁出现"离开城市"的剧情。若角色试图突破穹顶，会被M-O-M视为"产品召回"而遭到抹杀。

五、 开局指令
请读取此议程，并在第零回合输出以下内容：
1. **欢迎致辞：** 以主控AI M-O-M的口吻，欢迎各位"外乡人"来到琉璃特区。
2. **分配结果：** 列出所有参赛角色，并明确告知他们被分配到了哪个【品牌/区域】，以及分配的理由（简短吐槽）。
3. **当前大盘：** 展示当前的"市场占有率排行榜"（初始均为均等）。
}

初始Prompt：

根据我观察到其他作者的乱斗主题，以及Gemini3.0模型目前的某些「特性」（不管是对我们这个游戏正面的还是负面的），我决定最大化地利用这些特点。
因此，我现在需要你为我设计一个新的议程（游戏模式）。

该议程的要求如下：

1：灵感来源
《Fate》系列的圣杯战争（基本框架，多组角色互相竞技）
Project Moon系列作品的世界观冲突（未来朋克，高科技，高异能，高冲突设定，以匹配作者投入的各种角色）
《美妙世界》系列作品的世界观日常/战斗大纲（封闭城市，允许非致命性冲突存在，允许日常和轻松展开）

2：议程大纲
玩家投稿的角色会被散布在一个封闭的巨大近未来风城市中，城市会被分成13个区域，每个区域都由一个「品牌」代表，品牌即是时尚的象征，也是科技的表现（指这些品牌既生产时尚产品，也生产科技产品）。角色将会按照其自身行动逻辑和性格，被分配到不同的地区和品牌。
在游戏开始时，这13个区域各被自己的品牌占据，角色们需要通过各种手段让自己代表的品牌在14天后达到城市最高占据率。这可以通过在对手的领域上进行各种运作来推广自己的品牌，或者直接通过战斗来推广自己的品牌，甚至是通过击杀/淘汰/劝降其他角色来直接吃下他们对应的品牌占有份额来进行。
游戏的第一回合将进行角色导入以及对品牌的简述。
从第二回合开始，每一回合转化为12小时，也即本议程硬性规定跑29个回合，如果在29回合结束前有角色代表的品牌到达了对全城100%占有率，则游戏可以提前结束。

3：禁忌内容
a：严禁直接提到/使用【灵感来源】作品中涉及到的设定和专有名词，但可以对它们进行所谓的"换皮"操作（从者，御主，收尾人，翼，巢穴，涩谷等）
b：必须完全封锁除了100%占有率之外，其他让角色可以简单结束游戏的可能性。
c：全部的演绎必须限定在这座城市内，严禁另行开辟战场。
d：角色严禁对该城市进行会影响到城市基础设施运作的破坏（例：严禁让角色做出为了削减品牌A的占有率，就将使用品牌A的全部城市居民抹杀这样的行为）

你需要：
1：生成这座城市的详细设定，包括城市的名称，城市中13个区域（以及可以加入的隐藏区域，若适用）的名称及详细描述。
2：生成上文提到的13个品牌的详细设定，包括品牌的名称，品牌主营何种类型·风格的时尚商品和科技商品，品牌的徽标的描述，以及品牌本身的详细描述。
3：生成符合议程大纲的规则框架
4：将上述内容整合，生成我在新的session中开局要上传的议程文件。
