# 临时规则-能力载具化

## 1F

- Author: Ombre
- Posted at: 十月 02, 2025, 02:52 上午

我们将引入一个核心概念："共感神经座舱" (Sympathetic Neural Cockpit - SNC)。

背景故事设定：
在对抗联邦的最终决战中，卡斯卡迪亚独立军面临着王牌飞行员短缺的困境。为了最大化利用一切可用战力，顶尖的科学家和工程师开发了实验性的"共感神经座舱"系统。这套系统能读取驾驶员的内在特质——无论是超能力、强大的意志力还是独特的专业技能——并将其转化为一架高度定制化的战斗机的性能参数和武器系统。这些战斗机被称为**"化身" (Avatar)**，它们是驾驶员内心力量在天空中的具现。

通过这种方式，即使是没有飞行经验的普通人或超能力者，也能凭借他们独特的天赋，成为战场上不可或缺的王牌。

能力/被动/背景故事 载具转化规则
这个规则系统分为三个核心部分：1. 职业原型转化，2. 核心能力转化，和 3. 背景故事转化。

第一部分：职业原型转化 (决定基础机体框架)
角色的基本设定会决定其"化身"的基础机型类别。

力量/耐力型角色 (The Brute):

特征: 拥有超强力量、物理防御、或极强忍耐力的角色。
转化机型: 重型攻击机 (Heavy Attacker)。
机体特性: 拥有最高的装甲值和最多的武器挂点，但机动性和速度较低。适合作为团队中的"坦克"和重火力平台。
例子: 拥有"超级力量"或"钢铁皮肤"的超能力者。
敏捷/速度型角色 (The Stinger):

特征: 以速度、反应力、或灵巧身法见长的角色。
转化机型: 轻型拦截机 (Light Interceptor)。
机体特性: 拥有最高的机动性和加速度，装甲和火力相对较弱。擅长狗斗、躲避弹幕和追击高机动目标。
例子: 拥有"超级速度"的超能力者，或身手矫健的武术家、刺客。
智力/感知型角色 (The Mastermind):

特征: 拥有超凡智慧、强大精神力、黑客技术或战场洞察力的角色。
转化机型: 电子战/支援机 (EW/Support Craft)。
机体特性: 直接火力最弱，但拥有强大的电子战套件和团队增益/敌方减益能力。是团队的战术核心。
例子: 拥有"心灵感应"、"技术亲和"的超能力者，或顶级的战略家、工程师。
能量/元素操控型角色 (The Warlock):

特征: 能够操控火焰、冰霜、闪电、光线等能量形式的超能力者。
转化机型: 多用途战斗机 (Multi-Role Fighter)。
机体特性: 性能均衡，其真正的力量体现在由其超能力转化的特殊武器上。
例子: 火焰操控者、雷电法师等。
意志/信念驱动的普通人 (The Everyman Hero):

特征: 没有超能力，但凭借坚定不移的意志、专业技能或纯粹的勇气而战的普通人。
转化机型: 标准战斗机 (Standard Fighter)。
机体特性: 各项性能最为均衡、可靠，没有明显短板。他们的被动和背景故事将赋予其独特的优势。
例子: 勇敢的医生、坚毅的士兵、经验丰富的工程师。
第二部分：核心能力转化 (决定特殊武器与主动技能)
角色的主动能力或最突出的被动将转化为飞机的特殊武器 (SP Weapon) 和一个主动技能。

火焰操控:

SP Weapon: 燃烧弹药舱 (BML-U): 发射凝固汽油弹，在小范围内造成持续性伤害，非常适合清理无人机群。
主动技能: "过载熔炉": 短时间内，所有武器附带灼烧效果。
心灵感应/精神控制:

SP Weapon: 精神干扰导弹 (Psionic Missile): 命中后不仅造成伤害，还会短暂（2-3秒）扰乱目标的飞控系统，使其机动性大幅下降。
主动技能: "战术预知": 向全队通报BOSS即将使用的技能，提供更长的预警时间。
超级力量/物理强化:

SP Weapon: 动能冲击炮 (Kinetic Impactor / EML): 一种电磁炮，将角色的力量转化为纯粹的物理毁灭。
主动技能: "蛮力冲撞": 消耗大量能量进行一次短距离高速冲锋，对路径上的一切造成伤害并免疫伤害（CD长）。
治疗/生命力赋予:

SP Weapon: 修复无人机 (Repair Drones): 发射可以锁定友军的无人机，为其恢复少量装甲值。
主动技能: "生命绿洲": 在指定空域生成一个持续10秒的绿色能量场，进入的友军会缓慢恢复装甲。
技术专精 (普通人 - 工程师):

SP Weapon: 多目标空对空导弹 (MLAA): 精密的计算能力转化为高效的多目标锁定系统。
主动技能: "战场抢修": 立刻恢复自身一部分特殊武器弹药，并缩短所有技能的冷却时间。
第三部分：背景故事转化 (决定被动加成与涂装)
角色的个人经历和背景将提供独特的被动增益和个性化的飞机涂装。

背景：前线医生

被动增益: "Triage Instinct" (伤检分类本能): 对装甲值低于50%的友军，治疗技能效果提升20%。
涂装: 白色机身，带有红十字标志和简洁的线条。
背景：街头混混/生存专家

被动增益: "Street Smarts" (街头智慧): 导弹来袭警告的反应时间更长，热诱弹的有效范围更大。
涂装: 充满涂鸦、划痕和不羁风格的自定义涂装。
背景：沉着冷静的狙击手

被动增益: "Eagle Eye" (鹰眼): 使用单发、高伤害武器（如EML）时，锁定或充能速度提升15%。
涂装: 迷彩或深色系的低可视度涂装。
背景：学院派的学者/研究员

被动增益: "Calculated Move" (精密计算): 自身飞机的能量恢复速度（用于加力燃烧）提升10%。
涂装: 带有几何图案或母校徽章的典雅涂装。
综合示例
角色A：莱德，一个能硬化皮肤的前拳击手

原型转化: 力量/耐力型 -> 重型攻击机。高装甲、低机动。
能力转化: 硬化皮肤 -> 主动技能: "石肤壁垒" (短时间大幅减伤)。核心是拳击 -> SP Weapon: "震荡拳"火箭弹 (HPAA) (高单体伤害)。
背景转化: 前拳击手 -> 被动: "连击" (标准导弹连续命中同一目标时，伤害会递增)。涂装: 类似拳击手套的深红色和皮革色，机头有冠军腰带的图案。
角色B：莉娜，一位普通但意志坚定的战地记者

原型转化: 意志驱动的普通人 -> 标准战斗机。性能均衡。
能力转化: 记者的"揭露真相" -> SP Weapon: 目标指示吊舱 (Target Pod) (锁定敌人后，全队对其造成的伤害增加5%)。
背景转化: 战地记者 -> 被动: "身处险境" (当周围有3个以上敌人时，机动性会小幅提升)。涂装: 简洁的灰色机身，印有她所在新闻机构的Logo。
通过这套系统，每一位玩家，无论其角色背景如何，都能在"深红黎明"团本中找到自己独一无二的位置，其"化身"战机不仅是一台战争机器，更是其角色灵魂和故事在天空中的延伸。
