# 能够将角色卡转换为东方project程度的风格的协议

## 1F

- Author: Ombre
- Posted at: 十一月 10, 2025, 12:29 上午

### 第一部分：东方Project人物档案核心要素总结

东方角色的档案之所以独特，在于其模糊、诗意而又自成体系的设定方式。总结下来，主要包含以下几个关键部分：

1. **基本身份信息 (Basic Identity)**
 * **种族 (Species):** 这是最基础的设定。例如：人类、妖怪、神明、亡灵、吸血鬼、天狗、河童、仙人、月之民等。"妖怪"是一个非常宽泛的分类，可以容纳各种原创想法。
 * **能力 (Ability):** 东方角色能力的核心格式是 **"○○○程度の能力" (...teido no nouryoku)**，即"操纵/实行......程度的能力"。这个"程度"非常关键，它为能力划定了模糊的边界，充满了想象空间。例如，蕾米莉亚是"操纵命运程度的能力"，但实际表现多为物理层面的强大；八云紫是"操纵境界程度的能力"，几乎无所不能但又受到限制。
 * **二つ名 (Futanana - 称号/Title):** 每个角色在作为Boss登场时，都会有一个极具诗意和概括性的称号。例如："永远与须臾的罪人"（蓬莱山辉夜）、"不动的大图书馆"（帕秋莉·诺蕾姬）。一个角色可以有多个称号。

2. **外在表现 (External Profile)**
 * **外貌 (Appearance):** ZUN的风格通常是日式与西洋风格的奇妙结合，并且角色经常佩戴帽子或巨大的蝴蝶结。服装设计往往与其能力或主题相关。
 * **主题曲 (Theme Music):** 这是东方角色的灵魂！每个角色至少都有一首代表其个性的主题曲。曲名同样富有诗意，例如："亡き王女の為のセプテット"（为已逝王女的七重奏）。
 * **主要活动场所 (Location):** 角色通常有固定的居所或活动范围，如博丽神社、红魔馆、白玉楼、永远亭等。这让角色与幻想乡这个大舞台联系起来。

3. **游戏内具体表现 (In-Game Representation)**
 * **登场作品 (Appearances):** 标明角色在哪个作品中作为道中Boss、关底Boss或自机登场。
 * **符卡 (Spell Card):** 东方角色战斗的核心。符卡有固定的命名规则：**`X符「符卡名称」`** (例如：`境符「四重結界」`)。符卡的名称和弹幕样式都与其角色的能力、故事背景和性格紧密相关。通常一个Boss会拥有若干非符卡、普通符卡、以及一到两个最终符卡（Last Spell / Last Word）。

4. **稗田阿求视角的补充设定 (From "Perfect Memento in Strict Sense")**
 * **危险度 (Threat Level):** 阿求从人类的视角为妖怪划分的危险等级，如：低、中、高、极高、不明。
 * **人类友好度 (Human Friendship Level):** 角色对人类的态度，如：差、普通、良好、绝佳。
 * **应对方法 (Counter-strategy):** 如果在野外遇到这个角色，人类该如何应对。这往往是基于角色性格弱点或能力限制的有趣建议。

### 第二部分：角色卡"东方化"转化协议与模板

请按照以下模板和指南，将你角色的信息逐项填入并进行"东方化"改造。

#### **东方Project风格角色卡模板**

**【第一部分：核心档案】**

| 字段 | 日文参考 | 填写指南与"东方化"要点 |
| :--- | :--- | :--- |
| **全名** | フルネーム | 建议使用使用该角色原名，并附上假名注音和罗马音。名字可以蕴含其能力、种族或主题的线索。例如："魂魄 妖梦 (こんぱく ようむ, Konpaku Youmu)"。 |
| **种族** | 種族 | 从东方世界观中选择一个合适的种族。如果是原创种族，可以归为"妖怪"大类，并加以说明（例如：书写妖怪、星图妖怪）。 |
| **能力** | 能力 | **【核心步骤】** 这是最重要的转化。将你角色的能力改写为 **"〇〇〇程度の能力"** 的格式。**要点：**<br>1. **抽象化**：不要写"控制火焰"，而是"操纵热量程度的能力"。<br>2. **诗意化**：不要写"快速移动"，而是"超越视线所及速度程度的能力"。<br>3. **保留模糊性**："程度"二字意味着能力并非绝对，有其极限和应用范围。 |
| **称号** | 二つ名 | 至少想出2-3个。可以是一个身份描述，一个场景描述，或是一个性格侧写。<br>**格式参考**：<br>• XX的YY (例如：隙間の妖怪 - 隙间的妖怪)<br>• 形容词+名词 (例如：動かない大図書館 - 不动的大图书馆)<br>• 诗意的短句 (例如：永遠に紅い幼き月 - 永远鲜红的幼月) |
| **主题曲** | テーマ曲 | 发挥想象力，为你的角色起一个帅气的主题曲名字。可以参考现有曲名格式，如"〇〇なXX ～ Adjective Noun."或纯日文/英文。<br>例如：《忘れられた古書のノクターン ～ Forgotten Grimoire》 |
| **登场作品** | 登場作品 | 虚构一个登场作品和关卡。这能极大地增加角色的"实感"。<br>例如：《东方星莲船》 ～ Stage 4 Boss 或 《东方心绮楼》 ～ 可用角色 |

**【第二部分：详细资料】**

| 字段 | 日文参考 | 填写指南与"东方化"要点 |
| :--- | :--- | :--- |
| **居住地/活动场所** | 棲家/主な活動場所 | 将角色安置在幻想乡的某个地方。可以是现有地点（如图书馆、妖怪之山），也可以是自己创造的新地点（如"无名之丘的时计塔"）。 |
| **外貌与服饰** | 容姿 | 描述角色的外形。**要点：**<br>1. **标志性头饰**：帽子？巨大的蝴蝶结？兽耳？<br>2. **风格混搭**：将和风与洋装元素结合起来。<br>3. **与能力/主题相关**：例如，操纵时间的角色可能会佩戴怀表。 |
| **性格** | 性格 | 东方角色性格大多不坏，但有各种各样的"怪癖"。可能是懒散、好奇心旺盛、好战、爱捉弄人等。强大的角色往往有些孩子气或天真的一面。 |
| **人际关系** | 人間関係 | 设想你的角色会和哪些现有东方角色产生互动。是蕾米莉亚的牌友？还是魔理沙的魔法研究竞争对手？是紫的棋子还是朋友？这能让你的角色更好地融入世界。 |

**【第三部分：《求闻史纪》风格档案】**

| 字段 | 日文参考 | 填写指南与"东方化"要点 |
| :--- | :--- | :--- |
| **危险度** | 危険度 | 根据角色的能力和性格，从人类视角评定危险等级（低、中、高、极高、不明）。 |
| **人类友好度** | 人間友好度 | 角色对普通人类的态度如何？（极差、差、普通、良好、绝佳）。注意：友好度高不代表危险度低（例如芙兰朵露）。 |
| **主要活动** | 主な活動 | 角色平时都在做什么？（例如：读书、修行、睡觉、恶作剧、收集物品）。 |
| **应对策略** | 対応策 | 这是一个非常有趣的部分。设想一下，如果一个普通人遇到了你的角色，该怎么做才能安全脱身？<br>**要点：**<br>• 利用其性格弱点（如用新奇的故事吸引其注意力）。<br>• 遵守某个特定的规则（如不要触碰她的帽子）。<br>• 纯粹看运气（"遇到她的时候，祈祷她心情好吧"）。 |

**【第四部分：符卡（Spell Card）设定】**

这是展现角色战斗风格的核心。请为你的角色设计几张符卡。

* **命名规则:** `类别「符卡名」` (例如：`星符「Stardust Reverie」`)
* **类别参考:** 通常与角色的象征物、能力或种族相关（星符、时符、境符、罪符等）。
* **符卡名:** 可以是英文、日文或两者结合，要富有诗意和想象力，并能让人联想到弹幕的形态。

**符卡设计示例：**

* **通常弹幕:** 描述角色在不使用符卡时的攻击方式。
* **符卡1 (Easy/Normal):**
 * **名称:** `墨符「万卷流」`
 * **描述:** 从屏幕上方降下大量墨点状的子弹，轨迹如同瀑布。
* **符卡2 (Hard/Lunatic):**
 * **名称:** `書罪「ページ・オブ・ギルティ」` (Shōzai "Page of Guilty")
 * **描述:** 释放出数个巨大的、会缓慢旋转并全方位发射激光的魔法书。
* **最终符卡 (Last Word):**
 * **名称:** `「幻想図書館のアカシックレコード」` ("Akashic Record of the Fantasy Library")
 * **描述:** 历史生存卡。弹幕极其华丽且复杂，描绘出星空、文字和几何图案，几乎没有躲避的空隙，考验玩家的微操和耐心。
