# CHALDEA UNIFIED PROTOCOL - 迦勒底统合判定协议

## 1F

- Author: 小可
- Posted at: 十一月 22, 2025, 04:43 上午

CHALDEA UNIFIED PROTOCOL - 迦勒底统合判定协议
版本：CUP v2.0.1

===================================

第0节 系统概述

本协议用于判断角色在Fate世界观下的身份定位及职阶适配。
核心理念：英灵是传说的具现，御主是力量的使用者。
各维度评分必须独立判断，严禁将维度C（能力层级）的高分作为提升维度A（力量根源）评分的依据。一个角色完全可以拥有因果律级别的能力（高C分），但其力量根源可能仅仅是天生如此（低A分）。

===================================

===================================

第2节 核心判断系统

本系统通过四个维度评分判断御主/从者倾向，每个维度0-20分。

------------------------------------------------------------
维度A：力量根源性质
------------------------------------------------------------

这是最核心的判断维度。

评分标准：

0-5分：纯粹工具型
能力完全来自外部：魔术学习/家族刻印/科技装备/药物强化/身体改造
力量可以被剥夺/传授/复制
角色是"使用X的人"而非"X本身"

6-10分：混合型
部分能力天生/部分能力习得
或能力虽天生但不源于传说。
例：天生异能者/特殊血统/魔眼持有者。
[特例说明]：若天生能力本身即为一种抽象概念（如'极限'、'必然'、'可能性'），但仍是角色创造传说的起点，而非传说产物，则应归于此类的最高分段（9-10分），这代表着"神代的返祖"或"星球的触角"等顶级天赋，但本质上仍是"天生异能者"。

11-15分：传说关联型
能力与其传说有关联但不完全等同
生前确实使用过该能力，死后作为英灵继续持有
例：生前的武艺/生前使用的武器

16-20分：传说具现型
能力是"传说被传颂后的产物"。即，英雄的事迹（因） 导致了 传说（果），而英灵的能力是这个传说（新因） 所赋予的 新能力（新果）。
例：生前只是武艺高强的屠龙者 -> 死后因传说升华 -> 获得"龙种特攻"这一概念性能力。
[排除项]：若能力是与生俱来的天赋，且是其创造事迹的源头，则不应归于此类，无论该能力多么强大或概念化。

评分：[___/20]

------------------------------------------------------------
维度B：传说与存在状态
------------------------------------------------------------

评分标准：

0-5分：凡俗之人
在名声层面是完全的普通人

6-10分：现世精英 / 区域传说

11-15分：行走的神话 / 世界级英雄

他们是时代的象征，是世界观下的"英雄主角"或"时代焦点"

16-20分：已逝的传说 / 根源存在
已死亡且事迹成为传说

或其存在本身即为世界法则、神话根源

维度C：能力层级
------------------------------------------------------------

评分标准：

0-4分：Tier 0-1
凡俗级：常规人类/普通武器

------------------------------------------------------------
维度D：执念与动机方向
------------------------------------------------------------

基于角色行为逻辑中的终极目标判断。

评分标准：

0-5分：无愿望 / 纯粹的修正动机
对圣杯完全没有个人愿望
执念完全源于生前的遗憾/未竟之事
其愿望是纯粹地"修改时间线/完成生前遗憾"

6-10分：泛化的理想 / 非具体的动机
愿望并非其行动的最高优先级，可能会战斗，但不会专注于圣杯本身

参与圣杯战争更多是出于责任、好奇、享受过程或履行契约

11-15分：强烈的、正向的个人愿望
拥有一个强烈的、积极的、为了特定对象（自己/他人/特定群体）或特定目标（创造某物/抵达某地/完成某事）的具体愿望
他们会积极参与圣杯战争，但可能会在手段上有所保留

16-20分：强烈的、扭曲的个人欲望
拥有一个极其强烈的、哪怕会扭曲世界、牺牲他人也在所不惜的个人愿望
他们视圣杯为实现野心/目标的唯一途径

第3节 适性计算

基础适性计算：

从者基础适性 = (A + B + C + 40 - D - D) / 100 × 100 = [___]%
御主基础适性 = 100 - 从者基础适性 = [___]%

修正因子：

修正1：能力代价修正
若维度C中任一高层级（Tier 3-4）的**主动**能力使用代价极低或无代价：从者+10%
若所有高层级能力都需要巨大代价（生命/永久损伤）：御主+10%

【判定释义】
主动能力定义：指需要角色有意识地发起（如咏唱、真名解放、集中精神、消耗特定资源）才能发动的能力。其效果是瞬时或在特定时间段内生效的。
被动能力定义：指常驻生效的特性、天赋或体质，是角色存在的一部分。特别地，能造成较大范围实质影响的被动能力此处也视为主动能力。

修正2：战略威胁修正
若拥有对城级以上破坏力：从者+10%
若拥有对界级以上破坏力：从者+15%

修正3：传说悖论修正
若维度B≤15（当代/活人）但维度A≥16（传说具现）：标记[亚从者特征]

最终适性：

从者适性 = 从者基础适性 + 适用的从者修正 = [___]%
御主适性 = 御主基础适性 + 适用的御主修正 = [___]%

===================================

第4节 异常检测

在最终裁定前必须检测以下异常，若触发则优先输出：

异常1：亚从者
触发条件：维度B≤15 且 维度C≥15 且 该能力代价≤轻微
定义：凡人之躯持有神之权能
裁定：直接判定为【亚从者/Demi-Servant】，继续职阶判断

异常2：冠位特征
触发条件：从者适性≥90% 且 维度B≥16 且 维度C≥15
定义：超规格英灵
裁定：标记【Grand Servant候选】，继续职阶判断

异常3：边界存在
触发条件：|从者适性 - 御主适性| ≤ 20%
定义：处于模糊地带
裁定：输出双重适性，以高值为主但标注模糊性

异常4：数据矛盾
触发条件：两者适性均<40%
定义：信息不足或不适配Fate世界观
裁定：输出【无法判定】

===================================

第5节 身份裁定

若未触发异常，按以下规则裁定：

从者适性 > 御主适性 → 判定为【从者】，进入职阶判断
御主适性 > 从者适性 → 判定为【御主】，输出御主评价

确信度评级：
适性差≥40%：极高确信
适性差30-39%：高确信
适性差20-29%：中等确信
适性差<20%：低确信（已被异常3捕获）

===================================
