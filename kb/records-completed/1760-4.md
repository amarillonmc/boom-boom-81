**【玩家视角 / 第1回合 / 视角A：东际】**

**当前位置： 观星穹顶**
冰冷的电子音在我脑中消散，留下的信息简洁而沉重——"村长"

。这意味着我的存活与否，本身就是一条信息。死亡，必须发生在最有价值的地方。
周围的人群开始骚动。一个打扮得花里胡哨的女孩......**江之岛盾子**，我记得她的资料......抛出了一个看似合理的提议。*"两人一组"*，典型的错误战术。这只会加速信息壁垒的形成，让狼人更容易在小团体中制造混乱和猜疑。我无视了她。
我的目标只有一个——制高点。观星穹顶，根据这艘方舟的结构图，是俯瞰大部分核心区域的最佳位置。我没有丝毫犹豫，迈开脚步。
*"嘿，大个子！"*
一个轻快的女声从身后传来。我停下脚步，回头。是那个叫**希翠丝**的银发女孩，她身上的装备看起来不像善茬，但此刻的语气却像是在参加一场郊游。
*"看来咱俩被分到一队了，我是你的保镖哦~"*
保镖？我的眉头微不可查地动了一下。系统分配的强制同盟吗？这倒是个意外之喜。一个近战能力强的单位在我身边，能弥补我最大的短板。我只是用眼神示意，点了点头，继续前进。多余的言语是浪费时间。
观星穹顶比我想象的更开阔。巨大的弧形玻璃外，是深邃虚无的数据之海，偶尔有破碎的代码流如同星辰般划过。我走到穹顶中央的控制台，这里视野最好。左眼的战术目镜启动，一个淡蓝色的虚拟雷达界面覆盖了我的视野。
*"雷达系统展开，同步方舟生命信号。"*我低声自语。
界面上，十二个光点逐一亮起，大部分聚集在下方的*"中央食堂"*区域。然后，它们开始移动。两个点，一前一后，进入了通往这里的通道——是我们。一个光点......不，是两个，正朝着引擎室的方向去。还有一个独行的，目的地是舰桥。
*"已就位。"*我对身旁的**希翠丝**说。她正好奇地打量着穹顶外的景象，闻言立刻收回目光，在我身边摆出了警戒的姿态。
*"收到~"*她轻松地回应，但握着刀柄的手却很稳。
很好。雷达上的光点还在移动，分散开来。一个去了仓库，一个......在引擎室附近徘徊？还有两个，一个朝植物园去了，另一个也跟了上去。食堂里还剩下最后两个光点，几乎没有移动。
我的手指在虚拟界面上轻轻滑动，将代表我和**希翠丝**的光点标记为绿色。其他的，暂时都是代表*"未知"*的灰色。
现在，游戏开始了。无论他们是谁，无论他们想做什么，都在我的*"眼睛"*之下。我的任务，就是在这张棋盘上，找出那些移动轨迹异常的棋子，然后在他们造成威胁之前......
*咔。*
我将*"寂静旅途"*的支架撑开，枪托抵在肩上，冰冷的金属触感让我感到一丝心安。镜中，只有那些闪烁的光点，和一片等待被揭示的真相。

**【玩家视角 / 第1-2回合 / 视角B：野比大雄】**

**当前位置： 引擎室**
*"狼人......"*
当那两个字出现在我脑海里的时候，我的世界一下子就黑了。
怎么会这样？我......我要去骗人？还要......淘汰大家？**胖虎**和**小夫**也就算了，可是**静香**同学也在啊！还有那个看起来很温柔的大姐姐......不行！我绝对做不到！
*"哆啦A夢——！"*
我在心里发出了第一百次呼救，但这一次，没有任何回应。那个蓝色圆滚滚的身体，那个能拿出各种神奇道具的口袋，都不在这里。我只有自己。
周围的人都开始动了。那个穿着风衣的辣妹姐姐好像在说什么分组的事情，我一个字都没听进去，脑子里全是嗡嗡的声音。怎么办？怎么办？我会被发现的！大家一定会看出我是狼人的！到时候......他们会投票把我淘汰掉的！
就在我快要哭出来的时候，一个温柔的声音在我身边响起。
*"那个...我去引擎室看看...好像那里可以修理..."*
我抬起头，是那个白色长头发，头上长着粉红色*"羽毛"*的大姐姐。**美西螈姐姐**......她看起来......好善良。她的眼睛里没有任何怀疑，只是单纯地想去做点什么。
工程师......我的职业也是工程师......
一个念头像是救命稻草一样从我脑子里冒了出来。如果......如果我跟着她一起去修东西，大家是不是就不会怀疑我了？对！我也可以去帮忙！虽然我什么都不会，但只要做出努力的样子就好了！
*"等、等等我，我也去帮忙！"*我几乎是连滚带爬地跟了上去，生怕被她落下。
引擎室里又热又吵，巨大的机器发出轰隆隆的声音。**美西螈姐姐**......我记得她的名字好像是叫这个......她走到一个看起来很复杂的控制台前，很认真地开始研究上面的按钮和屏幕。
*"这里...好像需要...重新校准能量输出..."*她自言自语道。
我赶紧凑过去，装作很懂的样子看着屏幕。上面全是看不懂的符号和曲线。*"嗯！嗯！是啊是啊！"*我只能这样附和着。
*"那...这个按钮...应该是..."*她伸出手指，犹豫着要不要按下去。
*"我来吧！"*我鼓起勇气，抢着按了一个离她手指不远的按钮。
哔——！错误！

控制台发出了刺耳的警报声，屏幕闪烁起红光。
*"啊！对、对不起！我按错了！"*我吓得魂飞魄散，差点一屁股坐在地上。
*"没、没关系..."***美西螈姐姐**非但没有生气，还反过来安慰我，*"慢慢来...就好了..."*
她重新操作了几下，警报声消失了。我松了一口气，心里却感到一阵愧疚。她那么信任我，我却在骗她。可是......我又能怎么办呢？我只是......不想被淘汰而已。
我看着她专注的侧脸，又看了看周围那些复杂的机器。也许......我可以假装帮忙，然后在关键的时候，*"不小心"*弄坏什么东西？大家只会觉得是我太笨了，应该......应该不会怀疑我吧？
对，只能这样了。我一边擦着额头上的汗，一边在心里默默地对自己说。先活下去，活下去才有办法......

**【玩家视角 / 第2回合 / 视角A：约翰·康斯坦丁】**

**当前位置： 仓库**
*操。*
这地方给我的感觉越来越糟。空气里有股铁锈和尘埃混合的味道，还有一种......该怎么说呢，一种*"过于安静"*的感觉。就像暴风雨来临前，连虫子都停止了鸣叫。我的直觉，那该死的、救过我也坑过我无数次的*"同步性"*，正在我后脑勺上疯狂敲鼓。
两人一组？见鬼去吧。那种情况下，如果你的队友是狼，你连死都不知道怎么死的。还不如一个人待着，至少能保证背后没人捅刀子。
我从风衣口袋里摸出一小截粉笔，在仓库金属大门的内侧，飞快地画下一个警报用的符文。这玩意儿没什么大用，挡不住一个恶魔，甚至挡不住一个稍微壮点的混混。但只要有人带着恶意跨过它，它就会在我脑子里发出一声尖叫。对于眼下的情况，足够了。
我刚画完最后一笔，准备用一点唾沫混着烟灰激活它——
一种极致的、宛如实质的压迫感，毫无征兆地从我身后笼罩下来。
不是魔法，不是气势，就是那种......顶级掠食者盯上猎物时，最纯粹的物理存在感。空气都仿佛凝固了。我全身的寒毛瞬间倒竖，猛地转过身。
一个高大魁梧的身影就站在那里，离我不到五米。是那个叫**鲍勃**的黑人大汉，戴着墨镜，叼着烟斗，像一座沉默的铁塔。他是什么时候进来的？我那个还没来得及激活的符文屁用没有。他就像......就像凭空出现在那里一样。
我的大脑飞速运转，无数个应对方案闪过：虚张声势？拖延时间？还是直接激活*"枪手"*能力跟他拼了？
但所有的方案，都在他动起来的那一刻，化为了泡影。
我甚至没看清他的动作。我的眼睛只捕捉到一个残影，像是电视信号不良时跳帧的画面。视野里最后定格的，是他那张毫无表情的脸，然后，一股无法形容的剧痛从我的腹部炸开，仿佛被一辆全速行驶的卡车撞个正着。
*"操......"*
我只来得及从喉咙里挤出这个音节。胃里的酸水和昨晚的劣质威士忌一起涌上喉头，视线瞬间被黑暗吞噬。我的身体像个破布娃娃一样被打弯，双脚离地，连激活那该死的*"枪手"*能力的念头都无法凝聚。
意识模糊的最后，我感觉到自己被一股巨大的力量拖拽着，深入仓库的黑暗之中。冰冷的地面摩擦着我的脸颊。
原来如此......是我太大意了。混蛋，我早就该知道，在这种鬼地方，最沉默的那个，往往也最......
黑暗彻底淹没了我。

**【玩家-上帝视角 / 第2-3回合 / 视角B：江之岛盾子】**

**当前位置： 中央食堂 -> 舰桥**
*啊无聊，无聊，无聊！*
这个叫**玄羽**的男人，真是个无趣的家伙！不管本超高校级的辣妹怎么用可爱的表情和语气跟他搭话，他都像个石头一样，油盐不进。白瞎了那张帅脸！不过嘛......刚才用*【预言家】*能力查验我的时候，他那细微的表情变化，还是很有趣的哦~！确认我是*"好人"*之后，就立刻失去了兴趣，真是个现实的男人呢！
算了，既然他是好人，那就没什么可玩的了。让他自己去瞎忙活吧。
接下来去哪儿呢？嗯......对了！那个第一个冲出去的黑人大叔！叫**鲍勃**来着？看起来超级可疑的！一副*"我是高手我最屌"*的样子，这种人要么是真有本事，要么就是虚张声势的狼人！去看看他在搞什么鬼！
我哼着不成调的曲子，迈着轻快的步伐走向舰桥。一路上静悄悄的，其他人好像都躲起来了，真没劲。这种时候，就应该来点血浆和尖叫当背景音乐嘛！
推开舰桥的大门，里面空无一人。各种屏幕闪烁着不明意义的数据流，很有科幻电影的感觉。那个叫**Agnosis**的怪人好像也来过这里，空气里还残留着一股......嗯，怎么说呢，一股*"概念"*的味道？
*"切，人不在啊。"*我失望地撇了撇嘴。
正准备离开，我眼角的余光瞥到了主控制台。上面有一张小小的地图，显示着这艘船的结构和......人员分布？哇哦！好东西！虽然大部分都暗着，但还能看到几个亮点。
我饶有兴致地凑过去看。一个亮点在观星穹顶，一动不动，大概是那个狙击手吧？引擎室有两个点挤在一起，植物园也有两个......仓库......嗯？
仓库是暗的。

一个亮点都没有。
*"诶？"*我不禁发出了可爱的惊呼声。
我记得很清楚，之前明明有一个人去了仓库的呀！那个总是苦着脸，穿着风衣的男人，叫**康斯坦丁**？难道他已经离开了？可是......地图上没有多余的亮点在外面乱逛啊。
是地图坏了？还是说......
一个绝妙的可能性，如同一颗包裹着蜜糖的炸弹，在我脑中轰然引爆。
死人了？

啊......啊啊啊啊啊！
真的吗？真的吗？！这么快就开始了吗？！是谁？是谁干的？是那个看起来就很能打的**鲍勃**大叔吗？还是说，那个看起来人畜无害的**厨师**先生？或者......是那个哭哭啼啼跟在别人屁股后面的废柴小学生？！
啊~无论是哪一种，都太棒了！猜疑链！不信任！在谎言和恐惧中挣扎的*"好人们"*！这不就是最顶级的绝望盛宴吗？！
我几乎要忍不住放声大笑，但还是强行压抑住了。不行不行，现在我可是*"无辜又可爱的村民江之-岛-盾-子-酱"*哦~！
我立刻转身，装出一副惊慌失措的样子，朝着仓库的方向跑去。
*"不好了！不好了！康斯坦丁先生不见了！地图上没有他的信号了！"*
我要当第一个发现者！我要亲眼看看那绝望的案发现场！我要把这潭死水，彻底搅浑！
嘻嘻......游戏，终于变得有趣起来了！

**【玩家视角 / 第3回合 / 视角A：东际】**

**当前位置： 中央食堂**

警报声在耳边炸响的同时，观星穹顶的景象在我眼前扭曲、瓦解。下一秒，光线重组，我已经站在了中央食堂明亮的灯光下。
其他人也陆续出现在周围。引擎室的那两个，身上还带着油污和热气；植物园的**厨师**，手上捏着半片发光的蘑菇；还有那个银发的保镖，**希翠丝**，她几乎是和我同时出现，立刻在我身边站定，眼神锐利如刀。

食堂中央的屏幕亮起，冰冷的文字陈述着一个我早已预料到的事实：
【已确认淘汰者：约翰·康斯坦丁】

雷达上消失的光点，有了答案。

我的目光扫过在场的每一个人。那个叫**江之岛盾子**的女孩，作为报警人，正站在最前面，脸上挂着恰到好处的惊慌。一个优秀的演员，或者，一个真正的无辜者。

*"讨论开始。"*电子音宣布。

**盾子**开始她的表演，描述着发现尸体的经过，将*"地图信号消失"*这个信息点抛了出来。很聪明，这让她看起来更像一个无意中发现线索的村民。

接着是**鲍勃**。那个前军人。他的发言沉稳有力，将责任归咎于死者的*"单独行动"*，并反问**盾子**的时间线。典型的压力测试，试图从对方的反应中寻找破绽。但我虚拟雷达的记录显示，他自己就是个独行者，并且在信号消失的时间段内，他的移动轨迹无法被完全监控。他有嫌疑，而且很大。

*"我可以在某种程度上证实她的话。"*我开口，声音在嘈杂的食堂里显得异常清晰。我将雷达记录的时间线简要复述，排除了**盾子**的作案可能，同时，也将**鲍勃**的行动轨迹摆在了台面上。
*"......鲍勃先生确实离开了舰桥，走向中央区域。"*

我看到**鲍勃**在墨镜后的脸似乎没有任何变化，但他周围的空气仿佛凝重了一瞬。我的目标达到了——将他标记为*"需要被观察的对象"*。

接下来是一连串的自证。**野比大雄**和**美西螈**互相绑定，那个叫**文森特**的也从侧面证实了这一点。**厨师王**则拉上了**狂妄法师**。一瞬间，场上至少有六个人有了不在场证明。狼人很可能就在我们这些*"独行者"*之中。

**狂妄法师**的大喊大叫我直接过滤掉了。一个无法进行逻辑思考的单位，他的发言毫无价值。

然后，那个叫**Agnosis**的存在，开口了。

英雄？工作？

**【玩家视角 / 第11回合 / 视角B：玄羽】**

**当前位置： 舰桥 -> 中央食堂**

逻辑的奇点。

**野比大雄**，就是我们目前面临的，一个完全无法用常理解释的逻辑奇点。
当我从舰桥回来，在食堂中央向他提出那个问题时，我几乎可以肯定他会否认。而他的第一反应，也确实是惊慌失措的*"我没有啊"*。
这符合我的预判。

但是，在**江之岛盾子**那个女人的煽风点火之下，他后续的表现......那种羞赧的、默认的态度，又将一切推翻了。
*"我只是......想为大家做点事......"*
他承认了。
我看着他那张涨红的脸，那副不敢抬头的懦弱样子。这不像是伪装。这更像是......一个做了好事却不想声张的孩子，被当众表扬时的真实反应。
难道......真的是他？

难道在这个看似无用的少年体内，真的潜藏着我们无法理解的、能在瞬间完成复杂工程的*"隐藏天赋"*？就像他在某些领域拥有超乎常人的射击天赋一样？

**厨师王**破坏了植物园，但**野比大雄**推进了议程。这两件事，是两个独立个体完成的。**厨师王**依然有嫌疑，但他不再是*"唯一可能"*的嫌疑人。

不。不对。

这个阳谋太完美了。它利用了我们思维的惯性——我们倾向于相信亲眼所见的*"事实"*（进度条），而忽略那些无法被证明的*"怀疑"*（我对**厨师王**的直觉）。

这是一个试探。一个阳谋对另一个阳谋。
我将*"修复者"*的身份赋予他，将他固定在植物园。我就是要看看，当他被束缚住手脚时，暗杀是否还会继续发生。

他接受了。从容得就像接受一份应得的荣誉。
他转身离去，独自一人。
我的心，沉了下去。
我犯了一个错误。我不该让他单独离开的。在所有人都聚集在食堂的这个时间点，他获得了......一个完美的，捕猎落单者的机会。
我的目光扫过食堂。**Agnosis**, **文森特**, **美西螈**, **大雄**, **盾子**......那个**法师**呢？
**狂妄法师**......他好像没有跟我们一起回来。
该死！
