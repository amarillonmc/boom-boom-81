第 1 回合：【命运的四选一】

---

**📍 场景状态：嚎哭深渊·泉水**
游戏开始的瞬间，所有角色并未直接冲向中线，而是被强制停留在泉水中。
系统提示：【欢迎进入海克斯大乱斗。当前等级：4 级。请立即选择您的强化符文。】
每个人的面前都浮现金色、银色、棱彩三种颜色的符文光球，内容纯随机。
初始金币：1800（仅限购买基础组件）。
商店规则：下次购买装备需死亡回城。

🎲 全员第一次海克斯选择 (Level 4)

🔵 蓝方：秩序与希望
1. **红露**
 * 选项：[银色] 在一起更好 / [金色] 召唤军团 / [银色] 防护面纱
 * 选择：**【在一起更好】(Silver)**
 * 评价：*普通。* 身边有友军（黑兽）时增加双抗。适合前排，但不够爆发。
2. **瓦勒里乌斯**
 * 选项：[金色] 直击弱点 / [银色] 神射手 / [棱彩] 潘朵拉的盒子
 * 选择：**【直击弱点】(Gold)**
 * 评价：*优质。* 技能命中降低敌方双抗，完美契合他的精准射击。
3. **奇点霍金**
 * 选项：[银色] 蓝电池 / [金色] 溢流 / [银色] 急救用具
 * 选择：**【蓝电池】(Silver)**
 * 评价：*拉胯。* 只是回蓝加快，对于零点能反应堆来说略显多余。
4. **中务桐绪**
 * 选项：[金色] 正义之手 / [银色] 命运之舞 / [金色] 珠光护手
 * 选择：**【正义之手】(Gold)**
 * 评价：*优质。* 周期性必暴 + 治疗，增强生存与爆发。
5. **李奥瑞克**
 * 选项：[银色] 坚定意志 / [金色] 巨像的勇气 / [银色] 潘朵拉的装备
 * 选择：**【坚定意志】(Silver)**
 * 评价：*拉胯。* 仅增加韧性。对于需要抗伤的他来说，不如护盾或吸血。

🔴 红方：绝望与黑暗
1. **安培拉星人**
 * 选项：[金色] 超凡邪恶 / [银色] 亮出你的剑 / [棱彩] 飞升者
 * 选择：**【超凡邪恶】(Gold)**
 * 评价：*优质。* 随时间增加法强，符合他后期无敌的设定。
2. **希斯克利夫**
 * 选项：[金色] 死亡之环 / [银色] 杀戮时间 / [金色] 狂妄
 * 选择：**【狂妄】(Gold)**
 * 评价：*天胡！* 击杀永久增加伤害。虽然开局买不起成装，但这个海克斯让他有了滚雪球的资本。
3. **奥德修斯**
 * 选项：[棱彩] 潘朵拉的盒子 / [银色] 神射手 / [金色] 古代档案
 * 选择：**【潘朵拉的盒子】(Prismatic)**
 * 评价：*天胡！* 每回合随机获得临时道具。开局即巅峰。
4. **神·貉狸**
 * 选项：[银色] 小猫咪找妈妈 / [金色] 虚假前线 / [银色] 急救用具
 * 选择：**【小猫咪找妈妈】(Silver)**
 * 评价：*普通。* 附着队友获得加成。她决定附着在安培拉身上。
5. **水晶翼同调龙**
 * 选项：[金色] 珠光护手 / [银色] 速度恶魔 / [金色] 循环往复
 * 选择：**【珠光护手】(Gold)**
 * 评价：*优质。* 技能可暴击，大幅提升它的翼刃伤害。

🛒 初始装备购买 (1800 金币)
(注：无法购买成装，仅限组件)

* 🔵 蓝方：
 * 红露：[长剑] + [红水晶] + [生命药水] (偏战士开局)
 * 瓦勒里乌斯：[多兰之刃] + [生命药水] (射手续航)
 * 霍金：[增幅典籍] + [遗失的章节] 组件 (法强 + 回蓝)
 * 桐绪：[多兰之刃] + [生命药水] (普攻收益)
 * 李奥瑞克：[布甲] + [红水晶] + [生命药水] (纯坦开局)
* 🔴 红方：
 * 安培拉：[黑暗封印] + [增幅典籍] (赌狗开局，依赖海克斯成长)
 * 希斯克利夫：[长剑] x2 + [生命药水] (为后续【狂妄】叠层做准备)
 * 奥德修斯：[长剑] + [ boots 组件] (移速 + 攻击)
 * 貉狸：[圣物之盾] 组件 + [生命药水] (辅助开局)
 * 水晶翼：[长剑] + [布甲] (战士混伤)

⚔️ 战斗开始：【海克斯初显威】

**📍 场景：嚎哭深渊·中线**
选择完毕，泉水保护消失。十位英雄带着各自的随机命运冲向中线。

**"为了凯瑟琳！"**
**希斯克利夫** 最为激进。【狂妄】海克斯在他身上散发着微弱的红光。他骑着杜拉罕，利用【无首之马】的隐身效果，直接绕后切向蓝方后排。
他的目标：中务桐绪。
**"你的命运，到此为止！"** 魔王之剑挥下。
桐绪的【正义之手】被动触发，一次普攻暴击，配合【纬线之断】自动开枪，冰屑炸裂，勉强挡住了第一击。
但希斯克利夫的伤害太高了。【狂妄】虽未叠层，但基础伤害足以威胁脆皮。

**"守护我，戌。"**
**红露** 及时赶到，【在一起更好】海克斯生效，身边黑兽环绕，双抗提升。他挥动关刀，【雪落·斩诗】格挡了希斯克利夫的第二次攻击。
**"你的恨意，我收到了。但请冷静。"** 红露试图【劝降】，但希斯克利夫处于狂热状态。

**"检测到高威胁目标。"**
**奇点霍金** 悬浮高空，虽然【蓝电池】海克斯略显鸡肋，但它的【虚无龙息】依旧恐怖。激光直射安培拉星人。
安培拉星人【超凡邪恶】海克斯正在随时间叠加法强。他单手抬起，【反射披风】雏形开启。
**"黯淡。"** 激光被反弹了一半，霍金被迫走位躲避。

**"潘朵拉，会给我什么呢？"**
**奥德修斯** 笑着打开了【潘朵拉的盒子】。
**【系统提示：获得临时物品——【灭世者的死亡之帽】（持续 30 秒）！】**
**"哈哈！女神偏爱智者！"** 奥德修斯瞬间获得巨额法强加成。他立刻射出【致命一箭】，直奔蓝方残血的李奥瑞克。
李奥瑞克刚顶完一波伤害，血量岌岌可危。
**"呃啊！"** 箭矢贯穿了骷髅王的胸膛。

**"本官......不会......倒下......"**
桐绪试图救援，但水晶翼同调龙俯冲而下。【珠光护手】生效，它的【水晶翼刃】技能暴击了！
**"太弱了！"** 水晶翼一翼拍飞了桐绪和红露。它目前站在红方，因为它觉得安培拉的气场更强，值得挑战。

**"装死......装死......"**
**神·貉狸** 早就开启了【狸猫装死】，躺在战场边缘的草丛里，身上挂着【小猫咪找妈妈】的链接线连着安培拉。她在疯狂记录："那个弓箭手开了帽子......那个龙暴击了......"

💀 第一滴血与结算

**"咔嚓。"**
李奥瑞克 再次倒下。这次是被奥德修斯的临时帽子 + 水晶翼的暴击带走。
**First Blood!** 人头算给了奥德修斯（虽然貉狸也有贡献，但系统判定最后一击）。
奥德修斯获得 500 金币。希斯克利夫获得助攻。

蓝方陷入劣势：
* 李奥瑞克死亡，需回城才能买装备/选下一个海克斯。
* 桐绪残血，被迫后退。
* 红露被击退。
* 霍金被安培拉压制。
* 瓦勒里乌斯正在寻找机会点杀奥德修斯。

红方优势：
* 奥德修斯【潘朵拉】天胡，临时帽子还在持续。
* 希斯克利夫【狂妄】在手，急需人头。
* 水晶翼暴击流初显威力。
* 安培拉【超凡邪恶】随时间变强。

第 1 回合 结算

**📊 战场进展：**
- 第一滴血：奥德修斯 击杀 李奥瑞克。
- 海克斯运势：红方奥德修斯（棱彩）、希斯克利夫（金色狂妄）、水晶翼（金色珠光）均获优质海克斯。蓝方仅瓦勒里乌斯、桐绪获优质，李奥瑞克、霍金、红露海克斯较普通。
- 装备状态：全员基础组件，无成装。
- 特殊状态：李奥瑞克复活倒计时；奥德修斯【潘朵拉】临时帽子剩余 15 秒；希斯克利夫【棺】0 层（需击杀叠层，助攻不计）。

**💀 存活角色名单：**

第 2 回合：【复活与逝者的特权】

---

**📍 场景状态：嚎哭深渊·泉水 -> 中线**
李奥瑞克的骨骼在泉水中重新拼合。他是第一个死亡的人，因此也是第一个获得"购物权"的人。在这个残酷的竞技场，死亡成为了变强的捷径。
其他九人仍在战场上苦苦支撑，身上的初始组件在激烈的交锋中显得捉襟见肘。

*"活着的人被困在原地，死去的人却迎来了新生。"*

🔵 蓝方：李奥瑞克的回归

**李奥瑞克** 重生完毕。他拥有 500 金币（死亡掉落部分 + 自然增长）+ 初始剩余。
* 购买装备：**【燃烧宝石】** + **【锁子甲】** + **【生命药水】**。
 * 策略：既然【坚定意志】海克斯只加韧性，那就用装备弥补双抗和冷却。他要成为真正的不死壁垒。
* 状态：满血满蓝，带着新装备的寒意重返战场。

**"我回来了......为了未竟的赎罪。"**
李奥瑞克踏入中线，【绝望光环】再次笼罩战场。这次，他的骨锤上缠绕着更浓郁的黑色雾气。

🔴 红方：潘朵拉的倒计时

**奥德修斯** 正处于巅峰，但也处于危机之中。
**【系统提示：临时物品【灭世者的死亡之帽】效果剩余 5 秒。】**
*"必须在这之前再杀一个！"[/b] 奥德修斯眼中闪过狠厉。他瞄准了残血的中务桐绪。
**"命运......也能被计算！"** 他拉满弓弦，【致命一箭】带着剩余的法强加成呼啸而出。

**中务桐绪** 虽然残血，但【正义之手】海克斯提供的周期性护盾刚刚刷新。
**"本官的未来......还没结束！"** 她向脚下开了一枪【纬线之断】。
**"砰！"**
子弹击中了地面的一块松动冰块，冰屑飞溅，恰好挡住了箭矢的轨迹。箭矢偏转，擦着桐绪的肩膀飞过，击中了身后的奇点霍金。
霍金吃痛，龙躯晃动，但并未致命。

**"可惜了。"** 奥德修斯叹了口气。就在这时，【灭世者的死亡之帽】效果消失。他的法强瞬间跌落回初始水平。
**"女神的偏爱......真是短暂。"** 他迅速后撤，躲到安培拉星人身后。

⚔️ 团战爆发：【逝者 vs 生者】

**李奥瑞克** 赶到了。他直接开启了【白骨牢笼】，将试图追击的希斯克利夫困住。
**"你的仇恨，到此为止。"** 李奥瑞克挥锤重击。
希斯克利夫怒吼，【魔王之剑】砍在李奥瑞克的新护甲上，火花四溅，但伤害被【锁子甲】大幅削弱。
**"碍事！"** 希斯克利夫想要绕过坦克，但红露的关刀已至。

**红露** 开启了【雪拥·关城】。领域展开，时间内 slows 流速。
**"在这里，没有人能逃避审判。"** 红露试图再次【劝降】水晶翼同调龙。
**水晶翼同调龙** 在领域内感到不适。它的【珠光护手】海克斯让它渴望暴击的快感，但红露的领域让它无法冲锋。
**"卑鄙的束缚！"** 水晶翼强行发动【音爆冲撞】，试图冲破领域。
**"铛！"** 撞在了李奥瑞克的骨墙上。李奥瑞克纹丝不动，新装备让他硬如钢铁。

**奇点霍金** 抓住了机会。虽然【蓝电池】海克斯很鸡肋，但它的【龙骨导弹】冷却完毕。
**"目标：安培拉。优先级：最高。"**
导弹落下。安培拉星人并未躲避，而是开启了【反射披风】。
**"无用。"**
导弹伤害被反弹，霍金被迫开启护盾自保。但安培拉的【超凡邪恶】海克斯正在随时间叠加法强，他的存在感越来越强。

**神·貉狸** 依然在装死。她记录了：
1. 李奥瑞克的复活机制（死亡非终结）。
2. 奥德修斯的临时 Buff 机制（潘朵拉）。
